新标初_33 电车突然停了

（自）閉しまる〔音调：③〕｜动1　⇄　（他）閉しめる〔音调：②〕｜动2

义：关上；使…关上 自句：

  1. ドアが閉しまる。—— 门关上了。
  2. 窓まどが閉しまる。—— 窗户自己关上了。 他句：
  3. わたしはドアを閉しめる。—— 我把门关上。
  4. 鍵かぎをして窓まどを閉しめる。—— 上锁并把窗户关上。
（自）壊こわれる〔音调：◎〕｜动2　⇄　（他）壊こわす〔音调：②〕｜动1

义：坏掉；弄坏 自句：

  1. コップが壊こわれる。—— 杯子碎了。
  2. パソコンが壊こわれる。—— 电脑坏了。 他句：
  3. 子こ 供どもがおもちゃを壊こわす。—— 小孩把玩具弄坏了。
  4. ミスでデータを壊こわす。—— 因为失误把数据弄坏了。
（自）割われる〔音调：◎〕｜动2　⇄　（他）割わる〔音调：◎〕｜动1

义：裂、碎；打碎、分开 自句：

  1. ガラスが割われる。—— 玻璃裂了。
  2. たまごが割われる。—— 鸡蛋裂开了。 他句：
  3. チョコを割わる。—— 把巧克力掰开。
  4. りんごを割わる。—— 把苹果切开/劈开。
（自）付つく〔音调：①〕｜动1　⇄　（他）付つける〔音调：②〕｜动2

义：附着、带上；使附着、加上 自句：

  1. ほこりがシャツに付つく。—— 衣服上沾上灰尘。
  2. においが付つく。—— 沾上气味。 他句：
  3. ラベルを付つける。—— 贴上标签。
  4. しょうゆを付つける。—— 蘸上酱油。
（自）掛かかる〔音调：②〕｜动1　⇄　（他）掛かける〔音调：②〕｜动2

义：挂着；把…挂上／花费 自句：

  1. かさがフックに掛かかる。—— 伞挂在挂钩上。
  2. そのしごとはじかんが掛かかる。—— 那个工作要花时间。 他句：
  3. コートをいすに掛かける。—— 把大衣搭在椅子上。
  4. マフラーを首くびに掛かける。—— 把围巾围在脖子上。
（自）汚よごれる〔音调：◎〕｜动2　⇄　（他）汚よごす〔音调：◎〕｜动1

义：变脏；弄脏 自句：

  1. シャツが汚よごれる。—— 衬衫变脏了。
  2. くつが汚よごれる。—— 鞋子弄脏了。 他句：
  3. コーヒーでシャツを汚よごす。—— 咖啡把衬衫弄脏了。
  4. どろでゆかを汚よごす。—— 泥把地板弄脏了。
（自）止やむ〔音调：◎〕｜动1　⇄　（他）止やめる〔音调：◎〕｜动2

义：（雨/风等）停；停止、作罢 自句：

  1. 雨あめが止やむ。—— 雨停了。
  2. 風かぜが止やむ。—— 风停了。 他句：
  3. たばこを止やめる。—— 戒烟/不抽了。
  4. けんかを止やめる。—— 停止吵架/打架。
（自）建たつ〔音调：①〕｜动1　⇄　（他）建たてる〔音调：②〕｜动2

义：（楼等）建成；建造 自句：

  1. ここにアパートが建たつ。—— 这里建起了公寓。
  2. 南みなみのしまにホテルが建たつ。—— 南方小岛上建起了酒店。 他句：
  3. わたしはちいさいいえを建たてる。—— 我建一所小房子。
  4. きねんモニュメントを建たてる。—— 建纪念雕塑。
（自）並ならぶ〔音调：◎〕｜动1　⇄　（他）並ならべる〔音调：◎〕｜动2

义：排成行；把…排好 自句：

  1. ひとが並ならぶ。—— 人在排队。
  2. きっぷうりばに並ならぶ。—— 在售票处排队。 他句：
  3. 本ほんを並ならべる。—— 把书排好。
  4. 皿さらを並ならべる。—— 把盘子一一摆好。
（自）着つく〔音调：①〕｜动1（常无对应他动）

义：到达 自句：

  1. 電でん 車しゃが駅えきに着つく。—— 电车到站。
  2. 客きゃくがホテルに着つく。—— 客人到酒店。
（他）被かぶる〔音调：②〕｜动1　⇄　（他）被かぶせる〔音调：◎〕｜动2 ／（自）被かぶさる〔音调：◎〕｜动1

义：戴；给…戴／（自）盖在上面 他句（被る）：

  1. ぼうしを被かぶる。—— 戴帽子。
  2. バケツの水みずを被かぶる。—— 把桶里的水浇在身上。 他句（被せる）：
  3. こどもにぼうしを被かぶせる。—— 给孩子戴帽子。
  4. イスにカバーを被かぶせる。—— 给椅子套上罩子。 自句（被さる）：
  5. おちたはながテーブルに被かぶさる。—— 落下的花盖在桌上。
  6. くもが月つきに被かぶさる。—— 云遮住了月亮。
（他）履はく〔音调：①〕｜动1（无常用自动）

义：穿（鞋、袜、裤子） 他句：

  1. くつを履はく。—— 穿鞋。
  2. ズボンを履はく。—— 穿裤子。
（他）飼かう〔音调：①〕｜动1（无对应自动）

义：饲养 他句：

  1. 犬いぬを飼かう。—— 养狗。
  2. 魚さかなを飼かう。—— 养鱼。
（自）消きえる〔音调：◎〕｜动2　⇄　（他）消けす〔音调：◎〕｜动1

义：熄灭、消失；关掉、使…消失 自句：

  1. あかりが消きえる。—— 灯光熄灭了。
  2. 霧きりが消きえる。—— 雾散了。 他句：
  3. ライトを消けす。—— 把灯关掉。
  4. テレビを消けす。—— 关电视。
（自）落おちる〔音调：②〕｜动2　⇄　（他）落おとす〔音调：②〕｜动1

义：掉落；使…掉下、丢 自句：

  1. はなが落おちる。—— 花掉落。
  2. 鍵かぎが落おちる。—— 钥匙掉下来了。 他句：
  3. スマホを落おとす。—— 把手机掉了。
  4. さいふを落おとす。—— 把钱包弄丢了。
（自）負まける〔音调：◎〕｜动2　⇄　（他）負まかす〔音调：②〕｜动1

义：输；打败 自句：

  1. チームが負まける。—— 队伍输了。
  2. わたしはゲームで負まける。—— 我在游戏里输了。 他句：
  3. かれはライバルを負まかす。—— 他打败了对手。
  4. わたしはコンテストでみんなを負まかす。—— 我在比赛中战胜众人。
👒 被かぶる【③】

中文： 戴（帽子等） 例句：帽ぼう子しを被かぶる。→ 戴帽子。 📘补充：用于头上穿戴的动作，如帽子、头盔。

👓 掛かける【②】

中文： 挂上、戴上（眼镜等） 例句：メガネを掛かける。→ 戴眼镜。 📘补充：用于脸部、墙壁、肩膀等“挂、搭”类动作。

👔 着きる【①】

中文： 穿（上半身衣服、连体衣） 例句：スーツを着きる。→ 穿西装。 📘补充：用于上半身衣物，如衬衫、西装、连衣裙等。

👖 履はく【①】

中文： 穿（裤子、鞋、袜等） 例句：靴くつを履はく。→ 穿鞋。 📘补充：用于下半身衣物。

💍 嵌はめる【⓪】

中文： 戴上、套上（戒指、手表等） 例句：指ゆび輪わを嵌はめる。→ 戴戒指。 📘补充：用于环状物、需要“套上去”的物品。

⌚ 付つける【②】

中文： 佩戴、贴上（饰品） 例句：ピアスを付つける。→ 戴耳环。 📘补充：用于“附着在身体上的物体”，如胸针、耳环、徽章。

💎 する【⓪】

中文： 戴（项链、耳环、化妆等） 例句：ネックレスをする。→ 戴项链。 📘补充：泛用型动词，指所有“装饰性穿戴、使用”行为，如香水・ネクタイ・メイク等。
