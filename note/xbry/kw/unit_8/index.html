新标初_29 卡拉OK

カラオケ

加か藤とう：森もり君くん、次つぎ、歌うたいなさい！

中文： 加藤：森君，下一个，你唱！ 语法解析：

  • Vます語幹＋なさい：对晚辈/下属的“温和命令”。比「Vてください」更强，但仍属礼貌。
    • 变化：歌う → 歌いなさい／歌いなさいよ；否定命令常用「Vないで」而不说「～なさるな」。
  • 呼语 N君：对同辈/后辈男性的称呼，亲近不正式。
戴たい：そう、そう。森もりさん、歌うたいなさい！

中文： 戴：对，对。森先生，唱吧！ 语法解析：

  • 重复「そう」起附和/催促效果。
  • 用「なさい」对上辈/不熟是不合适的（为下句铺垫）。
李り：戴たいさん、自じ分ぶんより年とし上うえの森もりさんに「歌うたいなさい」と言いうのは失しつ礼れいですよ。

中文： 李：戴先生，对比自己年长的森先生说“唱吧（なさい）”是失礼的哦。 语法解析：

  • Nに 言う：对N说。
  • V辞書形＋のは＋N：把动作名词化作主语，“……这件事是失礼”。
  • 年上：年长者；より 表比较基准。
加か藤とう：そうだね。 この場ば合あいは「歌うたってください」と言いわないとね。

中文： 加藤：是啊。这种情况下要说“请唱一下”。 语法解析：

  • Vてください：标准礼貌请求。
  • Vないと（いけない） 口语省略后半，“不那样说可不行/应该那样说”。
戴たい：はい、分わかりました。 森もりさん、すみませんでした。

中文： 戴：好的，我明白了。森先生，抱歉。 语法解析：

  • 过去式「すみませんでした」= 为刚才的行为道歉。
  • 「わかりました」表领会对方意见。
李り：日に本ほんにいる時とき、日に本ほん人じんの男だん性せいが友とも達だちに、「来こい」「遠えん慮りょするな」などと言いっているのをよく聞ききました。 これは親したしい人ひとに使つかうんですね。

中文： 李：在日本时，我常听到日本男性对朋友说“来！”“别客气！”之类的话。这个是对亲密的人用的吧。 语法解析：

  • 命令形：来る→来い；V辞書形＋な（禁止）：遠慮するな。
  • 「〜と言っているのを聞きました」：把正在说的动作名词化作宾语（のを），接感官动词。
  • 親しい人に使う：对象格に。
  • 〜んですね：说明＋确认语气。
森もり：ええ。 でも、女じょ性せいはあまり使つかいません。 「来きてください」「遠えん慮りょしないでください」と言いったほうがいいですよ。

中文： 森：嗯。但女性不太用这些。说“请过来”“请不要客气”更好。 语法解析：

  • 否定习惯：Vません。
  • Vたほうがいい：给建议“还是……为好”。
  • Vないでください：否定请求（“请不要……”）。
李り：「ください」を取とって、「来きて」「遠えん慮りょしないで」などという言いい方かたをしている女じょ性せいもいました。

中文： 李：也有把“ください”去掉，说“来”“别客气”这样说法的女性。 语法解析：

  • Nを取って：把N去掉/拿掉。
  • 〜という言い方：表达方式。
  • もいました：也有……的人（存在）。
戴たい：そうですか。 中ちゅう国ごく語ごでも、親したしい相あい手てには、“来、来！” “別客气！”と言いって“请”を付つけませんから、同おなじですね。

中文： 戴：这样啊。中文里对熟人也会说“来来！”“别客气！”，不会加“请”，所以是一样的。 语法解析：

  • Nに は：就……而言（话题化）。
  • Vて＋N：引用外语/口号也用「と言って」。
  • Nを付ける：加上N；否定：付けません。
  • から表原因 → だから／ので也可。
李り：そうですね。 それから、野や球きゅう場じょうで女おんなの人ひとが「頑がん張ばれ！」と言いうのを聞ききました。

中文： 李：是的。另外我还听到球场上女性在喊“加油！”。 语法解析：

  • 頑張る→頑張れ：五段动词命令形（〜え）。
  • V辞書形＋のを聞きました：同上，名词化作宾语。
  • それから：补充话题“另外/然后”。
森もり：危き険けんな時ときや丁てい寧ねいに言いう暇ひまがない時ときは使つかってもいいんですよ。

中文： 森：在危险时候或没空讲客气时，用这些也是可以的哦。 语法解析：

  • AやB：并列。
  • V辞書形＋暇がない：没有时间做……
  • Vて（も）いい：许可“可以……”。
  • 〜んですよ：说明＋轻提示/安慰的语气。

小结（礼貌/命令表达梯度）

强度形式用法示例
最柔Vてください「歌ってください」「来てください」
建议Vたほうがいい「言ったほうがいい」
许可Vて（も）いい「使ってもいい」
命令（对后辈）Vます語幹＋なさい「歌いなさい」
命令（直截了当）来い／頑張れ／するな男性间、亲密关系或紧急场面

高频形态

  • 言う→言います／言っている／言った／言わないと
  • 聞く→聞きます／聞きました
  • 使う→使います／使いません
  • 付ける→付けます／付けません

新标初_30 春天的郊游

春はるのピクニック

李り：今こん度どの週しゅう末まつ、みんなでピクニックに行いこうと思おもっているんですが…

中文： 这周末，大家打算去郊游…… 语法解析：

  • V(意向形)＋と思っている：表示当前的打算/商量中的计划（行く→行こう）。
  • 〜んですが：说明+转折/铺垫，请对方回应。
森もり：いいですね、どこへ行いくんですか。

中文： 不错啊，要去哪里呢？ 语法解析：

  • N/辞書形＋んですか：带说明色的提问，更柔和。
  • 行く → 行くんですか（解释询问）。
李り：香こう山ざんへ行いこうと思おもいます。

中文： 想去香山。 语法解析： 行こうと思います：一时的意向/决定（行く→行こう）。

加か藤とう：森もり君くん、遅おそいよ。

中文： 森君，你来晚了啊。 语法解析： 口语评价句尾よ加强语气；遅い形容词直接作断定。

森もり：すみません、出でかけようとした時ときに、電でん話わがあったんです。

中文： 不好意思，刚要出门的时候来电话了。 语法解析：

  • V(意向形)＋とした時：正要……时（出かける→出かけようとした）。
  • Nがあったんです：带んです的解释说明口吻。
加か藤とう：そうか。それじゃ、まあ、仕し方かたないな。馬ばさん、みんなそろったから、そろそろ出しゅっ発ぱつしましょうか。

中文： 这样啊。那就没办法了。马先生，大家都到齐了，我们差不多出发吧？ 语法解析：

  • それじゃ＝それでは（那么）。
  • 仕方ない：没办法（固定说法）。
  • Vましょうか：提议/征求同意“要不要……”。
  • そろそろ：差不多、快要。
馬ば：ええ、そうしましょう。

中文： 好，就这么办吧。 语法解析： そうしましょう：对对方提议的赞同表达。

李り：近ちかくに北ぺき京ん植しょく物ぶつ園えんがあるので、帰かえりに寄よろうと思おもうんです。

中文： 附近有北京植物园，所以想回来的时候顺便去一下。 语法解析：

  • Nがあるので：因为……（原因）。
  • 帰りに：在回程时。
  • 寄る→寄ろうと思うんです：意向＋んです（说明自己计划）。
戴たい：いいですね、私わたしは花はなが好すきなので、楽たのしみです。

中文： 不错呢，我喜欢花，所以很期待。 语法解析：

  • Nが好き：喜好。
  • 〜ので：原因。
  • 楽しみです：期盼、很期待（名词性）。
森もり：人ひとが多おおいですね。

中文： 人真多啊。 语法解析： 形容词多い＋ですね表示感叹/共有观察。

陳ちん：今いまはちょうどピクニックのシーズンですから。

中文： 现在正是郊游季嘛。 语法解析：

  • ちょうど〜ところ／シーズン：正好……
  • 〜ですから：因为（带理所当然语气）。
馬ば：この季き節せつは、どの行こう楽らく地ちも人ひとでいっぱいですよ。

中文： 这个季节，哪里景点都挤满了人啊。 语法解析：

  • Nでいっぱい：被N塞满/很满。
  • どのNも〜：无论哪个都……（普遍性）。
李り：秋あきの香こう山ざんもいいですよ、今こん度どは紅こう葉ようを見みに来きようと思おもっています。

中文： 秋天的香山也很好，这次我想来看红叶。 语法解析：

  • V(ます)＋に来る：来做……（見に来る）。
  • 来る→来ようと思っています：持续的、较确定的打算（比「思います」更强）。

重点语法速记

  • 意向形＋と思う／と思っている：当下想法／更确定的计划。
  • Vようとした時：正要……的时候。
  • Vましょうか／そうしましょう：提议／赞同。
  • 〜んです（が／か）：说明、委婉提问或铺垫。
  • 帰りに／Nでいっぱい／〜ので：回程时／被……挤满／原因。

形态变化示例

  • 行く → 行こう（意向）／行くんですか（解释问）
  • 出かける → 出かけようとした（正要去）
  • 寄る → 寄ろうと思う（打算顺便去）
  • 見る → 見に来る／見に来ようと思っています（来观看的计划）

新标初_31 散步

散さん歩ぽ

馬：ここを曲まがると、小ちいさい公こう園えんがあって、子こ供どもたちがよく卓たっ球きゅうをしているんですよ。

中文： 从这里一拐就有个小公园，孩子们经常在那儿打乒乓球呢。 语法解析：

  • V-辞書形＋と…：**一…就…**的客观结果。「曲がると公園がある」。
  • 〜ている：习惯/持续。「している」← する → して → している。
  • 〜んです／んですよ：说明与提示语气。のだ → んだ → んです。
森：卓たっ球きゅうですか？　公こう園えんで？

中文： 乒乓球吗？在公园里？ 语法解析： 省略谓语的重复确认；第二句是省略主谓的追问（口语自然）。

馬：そうです。その公こう園えんにはコンクリートでできた卓たっ球きゅう台だいがあるんです。

中文： 是的。那个公园里有用混凝土做的乒乓球台。 语法解析：

  • N（场所）にはNがある：存在表达，强调地点（对比/提示的「は」）。
  • できたN：できる → できた（完成形，作定语）。
  • 〜んです：带背景的说明。
森：コンクリートの卓たっ球きゅう台だい？　へえ、面おも白しろいですね。

中文： 混凝土的乒乓球台？哎，很有意思呢。 语法解析：

  • NのN：名词定语。
  • へえ：表示惊讶/感叹。
  • 〜ですね：共同感受的确认。
馬：ええ、卓たっ球きゅうをしたいときにいつでもできるんです。たまに公こう園えんのそばを通とおることがあるんですが、必かならずだれかがやっていますね。

中文： 嗯，想打的时候随时都可以。偶尔从公园旁经过时，总会看到有人在打。 语法解析：

  • Vたい＋とき：「想做…的时候」。
  • いつでも＋可能形：「随时能…」。「できる」为可能动词。
  • V-辞書形＋ことがある：有时会…（经验/偶发）。
  • 〜んですが：先抑后扬/转折铺垫。「のだ → んだ → んですが」。
  • 〜ている：「やっています」← やる → やって → やっている。
  • 必ず／だれか：副词与不定代词，语义增强。
馬：ここから３００メートルほど行いくと、スポーツセンターがあるんです。

中文： 从这里再走大约三百米，就有个体育中心。 语法解析：

  • 数词＋ほど：大约。
  • V-辞書形＋と…：自然结果。「行くと…ある」。
  • 〜んです：说明/提出信息。
森：スポーツセンター？　だれでも自じ由ゆうに利り用ようすることができるんでしょうか。

中文： 体育中心？谁都可以自由使用吗？ 语法解析：

  • だれでも＋可能：任何人都能…
  • Nを利用する：サ变复合动词。
  • V-辞書形＋ことができる：可能表达。
  • 〜んでしょうか：のだ→んだ＋でしょう（推量/委婉）＋か（疑问）。
馬：ええ、だれでも利り用ようすることができます。ただし、有ゆう料りょうですが。

中文： 是的，任何人都能使用。不过是收费的。 语法解析：

  • ただし：转折说明“不过/但”。
  • 有料だ → 有料です：名词述语礼貌形。
  • 〜ですが：句尾轻转折，语气留白。
森：馬さんもよく利り用ようするんですか。

中文： 你也经常去用吗？ 语法解析：

  • よく＋V：频率副词“经常”。
  • 〜んですか：带背景的确认提问。
馬：たまにプールで泳およぐことがあります。でも、会かい員いんじゃないと夜よる８時じ以い降こうは利り用ようすることができないんです。だから、会かい員いんになろうと思おもっているんです。

中文： 偶尔会在泳池里游泳。不过不是会员的话，晚上八点以后就不能用了。所以我打算入会。 语法解析：

  • 泳ぐことがある：泳ぐ（辞）→ 泳ぐことがある“有时会…”。
  • AじゃないとB：如果不是A，就B不成立（口语条件）。
  • 以降：以后的时间范围名词。
  • できない：できる → できない（否定）。
  • 〜んです：说明缘由/情况。
  • 会員になる → なろう：なる（辞）→ なろう（意向形）。
  • V-意向形＋と思っている：持续的打算/计划表达。
  • 思っている：思う → 思って → 思っている（状态持续）。
森：会かい員いんになると、何なにかほかにも特とく典てんがあるんですか。

中文： 成为会员的话，还有别的优惠吗？ 语法解析：

  • V-辞書形＋と…：一…就…/如果…就…（普遍/既定结果）。
  • ほかにも：还有另外的。
  • 〜んですか：说明性询问。
馬：会かい員いんの家か族ぞくも安やすく利り用ようすることができるんです。

中文： 会员的家属也可以更便宜地使用。 语法解析：

  • 安い → 安く：イ形容词连用形作方式副词修饰动词。
  • V-辞書形＋ことができる：可能表达。
  • 〜んです：补充说明。
森：いま何時でしょうか。

中文： 现在几点了呢？ 语法解析：

  • 〜でしょうか：礼貌推量+疑问，语气委婉（自问/问他）。
馬：もうすぐ１時１５分になります。

中文： 快到1点15分了。 语法解析：

  • もうすぐ：再过不久。
  • Nになります：状态变化。「なる（辞）→ なります（丁寧）」。
森：じゃあ、早はやく帰かえらないと、午ご後ごの会かい議ぎが始はじまりますね。

中文： 那就得快回去了，不然下午的会议要开始了呢。 语法解析：

  • **V-ないと（いけない／困る）**的省略：不…就…（不行/会出问题）。这里省略后文表催促。
  • 始まります：始まる（辞）→ 始まります（丁寧）。
  • 〜ね：共识、轻度确认。

小结（常见变形一览）

  • する → して → している（进行/习惯）
  • やる → やって → やっている（进行）
  • 思う → 思って → 思っている（心理状态持续）
  • なる → なろう（意向形）／なります（丁寧）
  • できる → できない（否定）
  • 安い → 安く（副词性用法）
  • NのN（名词修饰）／場所＋には＋Nがある／いる（存在）
  • V-辞書形＋と…（一…就…／客观结果/既定规律）
  • V-辞書形＋ことがある（有时会…）
  • V-辞書形＋ことができる（可能）
  • V-意向形＋と思っている（打算）
  • AじゃないとB（不A就B不成立）
  • 〜んです／んですが／んでしょうか（说明/转折/委婉推量）

新标初_32 五一黄金周

連れん休きゅう

森：李りさん、今こん度どの連れん休きゅうの予よ定ていは？

中文： 李，你这次连休的打算呢？ 语法解析：

  • 名词＋は？：省略谓语的简略问句，用于询问安排/意见。
  • 今度（这次）、予定（计划）均为名词述语，无形态变化。
李：小お野のさんを北ぺ京きんのいろんな所ところへ案あん内ないするつもりです。

中文： 我打算带小野去北京各个地方转转。 语法解析：

  • いろいろな + 名词 ，口语中= いろんな + 名词

    • 语气差别：いろんな更随意、会话味；いろいろな更中性/书面。

  • 「人を 場所へ/に 案内する」（把人带到某地）。

    • 另一型是「人に 地点を 案内する」。

  • N を 案内する：带路、陪同参观。する（基本形）。

  • V-辞書形＋つもりだ：打算、意图。する → するつもりだ → するつもりです（礼貌）。

  • 所读ところ，处所名词。

（私は）
 └─小野さんを────────────┐（宾语）
 └─北京のいろんな所へ────┘（方向补语）
                 ↓
           案内する（谓语动词）
                 ↓
           つもりです（意向表达）
森：天てん気きは大だい丈じょう夫ぶでしょうか。

中文： 天气应该没问题吧？ 语法解析：

  • 〜でしょうか：推量＋疑问，委婉询问。
  • 大丈夫だ → 大丈夫でしょうか（名词述语＋推量）。
李：天てん気き予よ報ほうによると、連れん休きゅう中ちゅうはずっと晴はれだそうですよ。

中文： 根据天气预报，连休期间会一直晴呢。 语法解析：

  • N によると：根据N（信息来源）。
  • N中：期间。
  • Nだそうだ：传闻/转述。「晴れだ → 晴れだそうだ → 晴れだそうです」。
  • ずっと：一直（副词）。
  • 〜よ：提示、加强语气。
李：小お野のさんは、友とも達だちの太おお田たさんの家いえに行いくそうです。わたしもいっしょに行いくことになったんですが、森もりさんは？

中文： 听说小野好像要去朋友太田家。我也决定一起去了，你呢？ 语法解析：

  • V-辞書形＋そうだ（传闻）：行く → 行くそうです。
  • V-辞書形＋ことになる：客观决定/形成的安排。行く → 行くことになる → 行くことになった（过去）。
  • 〜んですが：说明背景后抛出问题/转折。「のだ→んだ」。
森：ぼくにもメールがあって、いっしょにお邪じゃ魔ますることにしましたよ。

中文： 我也收到邮件了，决定一起去打扰一下。 语法解析：

  • V-辞書形＋ことにする：主观决定。する → することにする → することにしました。
  • お邪魔する：自谦语（拜访/打扰）。
李：小お野のさんのメールによると、太おお田たさんの家いえで餃ギョ子ウパーティーをするそうです。だから、わたしにも手て伝つだってほしいって言いっていました。

中文： 根据小野的邮件，说要在太田家办饺子派对。所以她还说也想让我去帮忙。 语法解析：

  • N によると：据……（信息源）。

  • するそうだ：传闻/转述。

  • V-てほしい：希望别人做……。「手伝う → 手伝ってほしい」。

  • 〜って言っていました：口语「と」→「って」，言う → 言っている → 言っていました（过去进行/转述）。

  • わたし に も：助词组合，「に」标愿望的执行者（想让谁做），「も」是“也”。

  • 手伝っ て ほしい：

    • 动词手伝う→て形「手伝って」+ ほしい（形容词“想要”），合起来表示“希望（某人）帮忙”。

    • 结构是**（Aは）Bに Vてほしい**：A（希望方）想让B去做V。

    • 〜って言っていました：口语「と」→「って」，言う → 言っている → 言っていました（过去进行/转述），语气比「と言いました」更柔和，像转述邮件或多次提及。

    • と言いう

    • って = と、という、という人は （仅限口语）

森：餃ギョ子ウパーティー？　それはいいですね。

中文： 饺子派对？那真不错啊。 语法解析：

  • それはいいですね：赞同/赞赏的固定说法。
李：ところで、太おお田たさんって、どんな人ひとですか？

中文： 对了，太田是个怎样的人？ 语法解析：

  • Nって：口语题述标记（＝は）。
  • どんな人：询问性质/类型。
森：北ぺ京きんに来きてから一いち度ど会あったことがありますが、 とても気きさくな人ひとですよ。 スポーツ用よう品ひんの会かい社しゃで、 宣せん伝でんを担たん当とうしているそうです。

中文： 来北京以后见过她一次，是个很随和的人。在一家运动用品公司，好像负责宣传工作。 语法解析：

  • V-てから：自……以来。来る → 来てから。
  • V-たことがある：曾经有过……的经历。会う → 会ったことがある。
  • 気さく：随和（ナ形容词）。
  • Nを担当する：负责……。
  • V-ているそうだ：听说正在/持续从事……。「担当する → 担当している → 担当しているそうです」。
李：じゃあ、これからいっしょに仕し事ごとをすることになるかもしれませんね。

中文： 那以后也许会一起工作呢。 语法解析：

  • V-辞書形＋ことになる：形成的安排/客观决定。
  • 〜かもしれない：也许、说不定。**知れる（可能）**的固定语法：かもしれません（礼貌）。
  • 组合：することになる＋かもしれない → 可能成为既定安排。
森：ええ、だから、できるだけ連れん絡らくを取とろうと思おもっています。

中文： 嗯，所以我打算尽量保持联系。 语法解析：

  • できるだけ：尽可能地。
  • V-意向形＋と 思っている：打算/计划（自己持续的意向）。取る → 取ろう（意向形）→ 取ろうと思っている。
  • 思っている：思う → 思って → 思っている（状态持续）。

小结：本课关键形态变化

  • する → するつもりだ／するそうだ／することにする／することになる
  • 行く → 行くそうだ／行くことになる
  • 言う → 言っている／言っていました／って言う（口语）
  • 手伝う → 手伝ってほしい（对他人愿望）
  • 担当する → 担当しているそうだ（传闻进行/习惯）
  • 取る → 取ろう（意向形）→ 取ろうと思っている
  • 会う → 会ったことがある（经验）
  • 形：Nによると（根据信息源）；N中（期间）；V-てから（自……以来）；かもしれない（可能性）；〜でしょうか（委婉推量问）
