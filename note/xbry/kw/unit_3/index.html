新标初_09 温泉

温おん 泉せん

李り：小お 野のさん、この 浴ゆ 衣かたは ちょっと 小ちいさいです。

小野小姐，这件浴衣有点小。 语法解析：

  • 「このN」指示+名词：“这件……”。
  • 「ちょっと」表示稍微、有点（多带负面评价）。
  • 形容词敬体判断：い形容词「小さい」＋です → 礼貌表达。
  • 否定、过去形举例：小さい → 小さくない（否定）／小さかった（过去）。
小お 野の：あら、それは 子こ 供ども 用ようですよ。これを どうぞ。

哎呀，那是儿童用的哦。给你用这个。 语法解析：

  • 「N＋用」表示用途/用法（子供用＝给孩子用）。
  • 句末「よ」提示新信息、说话人断定。
  • 「これを どうぞ」：宾语标记「を」+ 递物固定表达「どうぞ」（请收下/请用）。
李り：それは ちょうど いいです。

那个正合适。 语法解析：

  • 「ちょうど」副词=正好、刚好。
  • 判断句：い形容词「いい」＋です（礼貌）。
小お 野の：李りさん、熱あつく ないですか。

李小姐，不会烫吗？ 语法解析：

  • い形容词否定敬体：熱い → 熱く ないです（≈更正式「熱く ありません」）。
  • 句末「〜ないですか」作确认/担心的反问：“不会……吗？”
李り：いいえ、ちょうど いいです。とても 気き 持もちが いいですね。

不，正好。感觉非常舒服呢。 语法解析：

  • 「とても」副词=很、非常。
  • 固定搭配：「気持ちがいい」= 舒服、惬意；主语由「が」标记。
  • 「ですね」带共识/感叹语气（希望对方赞同）。
李り：小お 野のさん、すばらしい 眺ながめですね。

小野小姐，景色真棒啊。 语法解析：

  • 名词「眺め」= 景色、视野。
  • 句末「ですね」：带感叹地陈述并寻求认同。
小お 野の：ええ、本ほん 当とうに。

嗯，真的。 语法解析：

  • 「本当に」副词= 真的、确实。此处省略了述语（＝すばらしいです）。
李り：小お 野のさん、それは 何なんですか。

小野小姐，那是什么？ 语法解析：

  • 指示词「それ」指靠近对方或对方持有的事物。
  • 「何ですか」= “是什么？”的敬体询问。
小お 野の：ああ、これは 温おん 泉せんの お湯ゆです。李りさんも 飲のみますか。

啊，这是温泉的热水。李小姐也要喝吗？ 语法解析：

  • 「A の B」：所属/来源（温泉的水）。
  • 尊敬前缀「お」＋名词「湯」＝礼貌说法。
  • 动词「飲む」→ 飲みます（敬体）→ 飲みました（过去）／飲みません（否定）。
  • 助词「も」表示“也”。
  • 「Vますか」礼貌询问意愿。
李り：ええ。あまり おいしく ないですね。

嗯。不怎么好喝呢。 语法解析：

  • 「あまり ＋ 否定」= 不太……/不怎么……。
  • い形容词否定：おいしい → おいしく ない（≈ おいしく ありません）。
  • 「ですね」表示带点委婉的评价。

本课要点速记

  • い形容词变形（以 熱い/おいしい 为例）
    • 肯定：熱い／おいしい
    • 否定：熱くない／おいしくない（礼貌可接「です/ありません」）
    • 过去：熱かった／おいしかった
    • 过去否定：熱くなかった／おいしくなかった
  • 常用副词
    • ちょっと（稍微/有点）│ ちょうど（正好）│ とても（很）│ 本当に（真的）│ あまり＋否定（不太）
  • 固定搭配与表达
    • 気持ちがいい：舒服、惬意（「が」标主语）。
    • N＋用：用途（子供用・旅行用）。
    • これをどうぞ：请收下/请用。
    • 〜ないですか：带关切的确认（不……吗？不会……吧？）。
  • 指示词复习
    • これ/それ/あれ：这/那/那边那个；ここ/そこ/あそこ：这里/那里/那边。

新标初_10 美术馆

美び 術じゅつ 館かん

長なが 島しま：昨きの 日う　京きょう 都とへ　行いきました。

昨天去了京都。 语法解析：

  • 方向助词「へ」表示去向/方向。
  • 「行く」基本形 → 行きます（敬体） → 行きました（过去敬体）。
  • 时间副词放句首作话题：昨日…。
小お 野の：京きょう 都とですか。京きょう 都との　紅も 葉みじは　どうでしたか。

是京都吗？京都的红叶怎么样？ 语法解析：

  • 「A の B」= 所属/关于（京都的红叶）。
  • 「どうでしたか」：どうですか（现在）→ どうでしたか（过去）。
長なが 島しま：とても　きれいでした。　でも、町まちは　あまり　静しずかじゃ　ありませんでした。

非常漂亮。不过，城里并不怎么安静。 语法解析：

  • い形容词过去：きれい（な形容词词干）→ きれいでした（过去肯定）。
  • な形容词否定过去：静かだ → 静かではありません → 静かじゃありませんでした（口语「じゃ」）。
  • 「あまり＋否定」= 不太……/并不怎么……。
  • 对比连接词：「でも」= 但是。
李り：えっ？

咦？（表示惊讶/不解） 语法解析： 感叹词，承上启下，无语法变化。

長なが 島しま：今いま　ちょうど　修しゅう 学がく 旅りょ 行こうの　シーズンです。　とても　にぎやかでした。

现在正好是修学旅行的季节。非常热闹。 语法解析：

  • 「ちょうど」= 正好。
  • 名词判断：シーズン＋です（现在判断）。
  • 叙述过去情况用句二：にぎやかでした（な形容词过去）。
李り：この　通とおりは　静しずかですね。

这条街很安静呢。 语法解析：

  • 「このN」= 这……。
  • 评价 + 「ですね」：带确认/共识的感叹。
長なが 島しま：ああ、今きょう 日うは　平へい 日じつですね。　でも、休やすみの　日ひは　観かん 光こう 客きゃくが　多おおいです。　とても　にぎやかですよ。

嗯，今天是工作日呢。不过，休息日游客很多，非常热闹哦。 语法解析：

  • 「今日は」话题化当前日期（礼貌写法维持逐字注音）。
  • 「A の 日」：休息日。
  • 「N が 多い」：以「が」标主语（多）。
  • 句末「よ」：说话人提供新信息/强调。
李り：そうですか。

这样啊/是吗。 语法解析： 信息确认的通用应答。

李り：あれえ、だれも　いませんね。

咦，一个人也没有呢。 语法解析：

  • 「だれも＋否定」= 谁也不……/一个人也没有。
  • 「いません」：动词「いる」（有生命）→ いません（敬体否定）。
小お 野の：そうですね。

是啊。 语法解析： 赞同/附和。

李り：ところで、この　美び 術じゅつ 館かんには　どんな　作さく 品ひんが　ありますか。

对了，这家美术馆里有什么样的作品？ 语法解析：

  • 「ところで」转话题：对了/话说。
  • 「場所＋に は」= 地点＋に表示存在位置，「は」话题化。
  • 「どんな＋名词」= 怎样的/什么样的。
  • 「あります」：无生命存在。结构：N が あります。
長なが 島しま：世せ 界かい 上じょうの　有ゆう 名めいな　作さっ 家かの　彫ちょう 刻こくです。

有世界级著名作家的雕塑。 语法解析：

  • 「A の B」连缀三次：世界上的＋著名的＋作家的＋雕刻（名词串联修饰）。
  • 「有名な」是な形容词作定语时带「な」。
長なが 島しま：美び 術じゅつ 館かんの　庭にわにも　いろいろ　おもしろいのが　ありますよ。

美术馆的庭院里也有各种有趣的东西哦。 语法解析：

  • 「〜にも」= 在……也（在别处有之外，这里也有）。
  • 「おもしろいの」：名词化指代“有趣的那些（作品）”。
  • 结句「ありますよ」：存在＋提示/强调。

本课要点速记

  • 存在表达：
    • 無生命：N が あります（場所に）。
    • 有生命：N が います（否定：いません）。
  • い/な形容词过去与否定
    • い形：きれい（な词干）→ きれいでした； 静か（な）→ 静かじゃありませんでした（正式：ではありませんでした）。
    • 「あまり＋否定」：あまり 静かではない／じゃない／じゃありません。
  • 并列/修饰链：「A の B の C」逐层限定：京都の紅葉、作家の彫刻。
  • 量与主语：「N が 多い／少ない」主语用「が」。
  • 提气粒子：「ね」（寻求认同）／「よ」（提示新信息/加强）。
  • 疑问：「どうでしたか」（过去的感觉/情况）、「どんな＋名词」。

形态变化速表

  • 行く → 行きます → 行きました
  • いる → います → いません（否定）
  • ある → あります → ありません
  • にぎやかだ → にぎやかでした／にぎやかではありませんでした
  • 静かだ → 静かです → 静かじゃありませんでした

新标初_11 礼物

お土み 産やげ

李り：箱はこ 根ねは ホテルや 旅りょ 館かんが 多おおいですね。

箱根酒店和旅馆真多啊。 语法解析：

  • AやB：列举“不完全并列”（A、B等）。
  • 形容词「多い」现在陈述；过去/否定：多かった／多くない。
  • 句末「ですね」带同感确认。
小お 野の：そうですね。会かい 社しゃの 別べつ 荘そうも たくさん あります。

是啊。公司的别墅也很多。 语法解析：

  • 「A の B」所属（公司的别墅）。
  • 無生命存在：N が あります；否定：ありません。
  • 「も」= “也”。
李り：長なが 島しまさんや 小お 野のさんも、よく 来きますか。

长岛先生和小野小姐也经常来吗？ 语法解析：

  • 「よく」副词= 经常。
  • 来る → 来ます（敬体）→ 来ました／来ません。
長なが 島しま：ええ、車くるまで 時とき々 来きます。

嗯，时常开车来。 语法解析：

  • 「で」表手段/工具（乘车）。
  • 「時々」= 有时、不时。
小お 野の：わたしは 運うん 転てんが できませんから、いつも 電でん 車しゃです。

我不会开车，所以总是坐电车。 语法解析：

  • N が できません：不会/不能做N（这里 N=運転）。
    • できる（能）→ できます／できません。
  • から：表示原因。“因为不会开车，所以……”。
  • 交通方式名词＋です（省略动词“乘坐”）。
長なが 島しま：ぼくは 来らい 月げつ また 来きますよ。

我下个月还会来哦。 语法解析：

  • 时间名词作状语：「来月」。
  • 「また」再次；句末「よ」提示信息/强调。
李り：どうしてですか。

为什么呢？ 语法解析：

  • 原因疑问词「どうして」。
長なが 島しま：この 近ちかくの ホテルで 友とも 達だちの 写しゃ 真しん 展てんが ありますから。

因为这附近的酒店有朋友的摄影展。 语法解析：

  • 「AのB」连体修饰：近く的酒店／朋友的摄影展。
  • 無生命存在：N が あります。
  • 〜から结句给出理由（解释型）。
李り：箱はこ 根ねの お土み 産やげは 何なにが いいですか。

箱根的特产，买什么好呢？ 语法解析：

  • 评价型问法：何が いいですか（什么比较好）。
  • 「お〜」礼貌前缀：お土産。
小お 野の：うーん、そうですね……、迷まよいますね。

嗯——这个嘛…真让人难选呢。 语法解析：

  • 「迷います」：迷う→ 迷います（犹豫/拿不定主意）。
  • 「ね」带共感语气。
長なが 島しま：寄よ 木せぎ 細ざい 工くは いかがですか。とても 有ゆう 名めいですよ。

寄木细工怎么样？非常有名哦。 语法解析：

  • 「〜は いかがですか」= 郑重推荐/征询。
  • 「有名な」作定语时带「な」；句尾加「よ」加强语气。
小お 野の：きれいな 模も 様ようですね。

花纹真漂亮啊。 语法解析：

  • 形容动词做定语：「きれいな 模様」。
  • 「ですね」共感感叹。
長なが 島しま：ええ。気きに 入いりましたか。

嗯。你喜欢吗？ 语法解析：

  • 固定表达：気に入る（中意、喜欢）。
    • 入る→ 入ります → 入りました（过去完成：是否已喜欢上？）。
李り：はい、 とても。

是的，非常喜欢。 语法解析：

  • 省略述语的简洁回答（＝とても 気に入りました）。

本课要点速记

  • 列举：A や B（不完全列举）／A と B（并列完整）。
  • 存在：無生命 N が あります；有生命 N が います。
  • 能力：N が できる → できます／できません（運転ができません）。
  • 原因：句末 〜から（解释/回答“どうして”）。
  • 频度副词：よく（经常）、時々（有时）、いつも（总是）、また（再、又）。
  • 推荐：〜はいかがですか（更礼貌，比「どうですか」郑重）。
  • 评价问法：何が いいですか（买哪个好/做哪个好）。
  • 固定搭配：気に入る（喜欢上）— 気に入りました／気に入りません。

动词与形容词形态链

  • 来る → 来ます → 来ました／来ません
  • ある → あります → ありません
  • できる → できます → できません
  • 迷う → 迷います → 迷いました
  • 入る → 入ります → 入りました
  • 多い → 多くない／多かった
  • 有名だ → 有名です（现在）／有名でした（过去）

新标初_12 酒与茶

お 酒さけと　お 茶ちゃ

李り：長なが 島しまさんは　よく　お 酒さけを　飲のみますか。

你经常喝酒吗？ 语法解析：

  • 频度副词「よく」= 经常。
  • 宾语助词「を」标“お酒”。
  • 动词：飲む→飲みます（敬体）→飲みました（过去）／飲みません（否定）。
  • 疑问：ます形＋か。
長なが 島しま：ええ。お 酒さけは　大だい 好すきですから、毎まい 晩ばん　飲のみますよ。

是啊。我非常喜欢酒，所以每天晚上都喝。 语法解析：

  • 「N は 大好きです」：な形名词性形容词，表示很喜欢。
  • 接续「〜ですから」= 因为…所以…（礼貌因果）。
  • 「毎晩」时间副词；动作习惯「飲みます」。
  • 句末「よ」提示信息/语气加强。
李り：ビールと　日に 本ほん 酒しゅと、どちらが　好すきですか。

啤酒和清酒，哪个更喜欢？ 语法解析：

  • 「A と B と、どちらが 好きですか」= 二选一比较问法。
  • 「N が 好き」结构：主语用「が」。
長なが 島しま：どちらも　好すきです。　でも　焼しょう 酎ちゅうが　いちばん　好すきです。

都喜欢。但最喜欢烧酒。 语法解析：

  • 「どちらも」= 两者都……
  • 「いちばん」= 最高级“最……”。结构：いちばん＋形容/好み。
小お 野の：焼しょう 酎ちゅうは　最さい 近きん　人にん 気きが　ありますね。

烧酒最近很有人气呢。 语法解析：

  • 固定搭配：「人気がある」= 受欢迎。
  • 「最近」时间词；句末「ね」寻求认同。
李り：小お 野のさんは　焼しょう 酎ちゅうが　好すきですか。

小野你喜欢烧酒吗？ 语法解析：

  • 主语用「が」接「好き」（名词性形容词）。
小お 野の：いいえ。わたしは　焼しょう 酎ちゅうより　ワインの　ほうが　いいです。　でも　ビールも　よく　飲のみますよ。

不。我比起烧酒，更喜欢葡萄酒。不过啤酒也常喝。 语法解析：

  • 比较：Aより Bのほうが いい／好き。
  • 「〜も」= 也；「よく」= 经常。
  • 動詞 飲む 变化同上；句末「よ」提示。
長なが 島しま：中ちゅう 国ごくは　お 茶ちゃの　種しゅ 類るいが　多おおいですね。

中国的茶种类很多啊。 语法解析：

  • 「N の 種類」= ……的种类。
  • い形容词「多い」现在叙述；否定：多くない；过去：多かった。
李り：ええ。ウーロン茶、ジャスミン茶、緑りょく 茶ちゃなど、たくさん　あります。

对。乌龙茶、茉莉茶、绿茶等等，有很多。 语法解析：

  • 列举：「A、B、Cなど」= 等等（不完全）。
  • 無生命存在：N が／は あります。
小お 野の：どの　お 茶ちゃが　いちばん　人にん 気きが　ありますか。

哪种茶最受欢迎？ 语法解析：

  • 选择疑问「どの＋名词」。
  • 最高级：いちばん＋形/名词性表达。
  • 「人気がある」固定搭配。
李り：やっぱり　ウーロン茶です。

果然还是乌龙茶。 语法解析：

  • 「やっぱり」= 果然/还是（说话人判断）。
小お 野の：李りさんも　よく　飲のみますか。

你也经常喝吗？ 语法解析：

  • 「も」= 也；「よく」= 经常；ます形疑问。
李り：ええ。毎まい 日にち　ウーロン茶や　ジャスミン茶を　飲のみます。

嗯。每天喝乌龙茶和茉莉茶（等）。 语法解析：

  • 「A や B」= 不完全并列（A和B等）。
  • 宾语用「を」；飲む 变形同上。
長なが 島しま：どちらが　おいしいですか。

哪个更好喝？ 语法解析：

  • 「どちらが〜」比较问法；形容词「おいしい」敬体判断。
李り：どちらも　おいしいですよ。　わたしは　ジャスミン茶の　ほうが　好すきですが…

两种都好喝哦。我个人更喜欢茉莉茶…… 语法解析：

  • 「どちらも」= 两者都……
  • 比较偏好：B の ほうが 好き。
  • 逆接缓和：「…ですが」= 虽然/不过（后续省略）。

重点语法与形态速记

  • 比较与选择
    • Aより Bのほうが いい／好き。
    • 「どちらが いいですか」→「どちらも いいです」。
    • 最高级：いちばん＋形容词／好み。
  • 喜欢结构：Nが 好き／大好き（な）。
  • 列举：A と B（并列、完整）；A や B（列举、还有别的）；A、B、C など。
  • 存在/受欢迎：N が あります；人気が あります。
  • 频度与习惯：よく／毎日／毎晩＋Vます。
  • 动词变形（本课）：
    • 飲む→飲みます→飲みました／飲みません
    • ある→あります→ありません
