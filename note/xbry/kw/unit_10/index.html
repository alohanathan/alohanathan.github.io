新标初_37 万里长城

万ばん里りの長ちょう城じょう

小お野の：「八は達たつ嶺れい」までどのぐらいですか。

中文： 到“八达岭”大概有多远/要多久？ 语法解析：

  • まで：终点助词“到……为止”。
  • どのぐらい：数量/程度疑问“多大/多久”。
  • ですか：敬体疑问。
  • 省略了“時間が”，口语常用：（到……）どのぐらいですか＝“要多久？”。
李り：高こう速そく道どう路ろを利り用ようすれば、だいたい１いち時じ間かんぐらいです。

中文： 如果走高速，大概一个小时左右。 语法解析：

  • N を利用する：“利用N”。基本形 利用する → 条件形 利用すれば（する→すれば）。
  • だいたい／〜ぐらい：大约/左右。
  • 敬体判断 です。
（続き）鉄てつ道どうなら２に時じ間かんちょっとですね。

中文： 坐火车的话，大概两个多小时呢。 语法解析：

  • N なら：话题限定“若是N的话”。
  • 〜ね：征求认同/缓和语气。
小お野の：鉄てつ道どうでも行いくことができますか。

中文： 坐火车也可以去吗？ 语法解析：

  • N でも：“连N也／N也可以”。
  • 行くことができる：可能表达=行ける的书面型。基本形 行く → 名词化 行くこと → 可能表达 〜ができる。
  • 敬体疑问 〜ますか。
李り：ええ、鉄てつ道どうで行いくなら、北ぺ京きん北ほく駅えきが便べん利りです。直ちょく通つう列れっ車しゃが出でていますから。

中文： 嗯，坐火车的话，北京北站比较方便。因为有直达列车开行。 语法解析：

  • N で行く：用N交通方式去。
  • 〜なら：条件/话题。
  • N が便利です：Aな形容词「便利だ」→ 敬体 便利です。
  • V-ている：进行/状态「出ています」=“正在发车/有车开出”。
  • 〜から：原因说明“因为……”。
李り：「八は達たつ嶺れい」は万ばん里りの長ちょう城じょうの観かん光こうスポットの１ひとつです。北ぺ京きんから近ちかくて、いちばん有ゆう名めいな場ば所しょです。

中文： “八达岭”是万里长城的观光景点之一。离北京近，是最有名的地方。 语法解析：

  • N の一つ：……之一。
  • X から近い：离X近，近い→近くて（连用形并列）。
  • 一番＋形容词：最高级“最……”。
  • 有名な場所：形容动词连体修饰名词。
小お野の：テレビとか写しゃ真しん集しゅうとかで見みたことがありますが、実じっ際さいに見みると、本ほん当とうに規き模ぼが大おおきいですね。さすが「世せ界かい遺い産さん」です。

中文： 在电视或画册上见过，不过实际一看，规模真的很大。不愧是“世界遗产”。 语法解析：

  • N とか N とか：列举“……啦……啦”。
  • V-たことがある：曾经有过……的经验。見る→見たことがある。
  • V-る と：一……就……（客观条件）。
  • さすが：果然/不愧。
  • ですね：感叹并求同。
小お野の：どうして「万ばん里りの長ちょう城じょう」と言いうんですか。

中文： 为什么叫“万里长城”呢？ 语法解析：

  • どうして：为什么。
  • N と言う：称为……。言う→言うんですか（のです说明+疑问，带解释语气）。
李り：長ちょう城じょうの全ぜん長ちょうは５ご０れい０れい０れいキロメートル以い上じょうあります。中ちゅう国ごくでは「１いち里り」は０れい．５ごキロメートルですから、中ちゅう国ごくの単たん位いに換かん算さんすれば、１いち万まん里りを超こえるんですよ。

中文： 长城的总长度在五千公里以上。在中国，“一里”是0.5公里，所以按中国单位换算，超过一万里呢。 语法解析：

  • N 以上：以上。
  • ですから：因此/所以。
  • N に換算する：换算为N。する→すれば（假定形）表示“如果换算的话”。
  • V-る ＋ んです：解释说明语气；超える（超过）→ 超えるんです。
小お野の：何なにかお土みや産げを買かおうと思おもうんですが…

中文： 想买点儿纪念品…… 语法解析：

  • V-おうと思う：意志形＋思う，表示意图/打算。買う→買おう（五段意志形）＋と思う。
  • 〜んですが：带解释/征询的转折语气。
李り：お土みや産げなら、Tシャツとかメダルとか、いろいろありますよ。

中文： 纪念品的话，有T恤、纪念章之类的，挺多的哦。 语法解析：

  • N なら：就N来说/如果是N的话。
  • N とか N とか：举例。
  • いろいろある：有很多；敬体提示 ありますよ 带提示语气。
小お野の：そうですか。じゃあ、メダルでも買かおうかな。

中文： 这样啊，那我买个纪念章吧。 语法解析：

  • でも：用在名词后，表示“……也行/来个……吧”（宽泛选择）。
  • V-おうかな：意志形＋かな，自言自语式的犹豫/考虑。買う→買おうかな。
李り：ゆっくり探さがせば、ほかにもいいお土みや産げがあると思おもいますよ。

中文： 慢慢挑的话，我想还有别的不错的纪念品哦。 语法解析：

  • V-ば：假定形。「探す→探せば」。
  • N も：“也/还”。
  • A いい N：好的N。
  • 〜と思います：表达说话人的推量/看法。

重点词形与变化总表

词/表达基本形变化形/用法说明
利用するする利用すれば假定形“如果……”
行く行く行くことができる可能表达（书面）
出る出る出ています持续/状态
近い近い近くて连用并列
見る見た見たことがある经验
言う言う言うんですか说明＋疑问
換算するする換算すれば假定形
超える超える超えるんです解释说明
買う買う買おう（意志形）／買おうかな意图／自问
探す探す探せば假定形

新标初_38 胡同

胡フー同トン

下面内容严格按你之前的所有规则＋“所有出现汉字均逐字注音、纯假名不再注音”的规则重做：

  • 日语原文用五级标题（#####）。
  • 每句后给“一一对应”的中文释义。
  • 逐句进行“详细语法解析＋形态变化（给出基本形与变化过程）”。
小お野の：この路ろ地じが「胡フー同トン」ですか。

中文释义：小野：这条小巷就是“胡同”吗？ 语法解析

  • この：指示词“这”。
  • 路ろ地じ：名词“巷子、小胡同”。
  • 「……」：名称引用。
  • ですか：判断（礼貌体）＋疑问。 形态变化：无活用（名词判断句）。
李り：ええ、この辺あたり北ぺ京きんの伝でん統とう的てきな町まちです。

中文释义：李：对，这一带是北京的传统街区。 语法解析

  • 辺あたり：名词“附近、一带”。
  • 北ぺ京きんの伝でん統とう的てきな町まち：形动（的てき）连体＋名词。
  • です：判断礼貌体。 形态变化：无活用。
人ひとが実じっ際さいに生せい活かつしている所ところですから、写しゃ真しんは撮とらないようにしてくださいね。

中文释义：因为这里是人们实际生活的地方，所以请不要拍照喔。 语法解析

  • 〜ている：进行/状态。“生活している”。
  • 〜ですから：原因。
  • 撮とらないようにする/してください：V-ない＋ように（避免）＋请求。 形态变化
  • 撮とる（基本形）→ 撮とらない（否定辞书形）→ 撮とらないようにする → 撮とらないようにしてください（请求形）。
小お野の：分わかりました。ところで、この道みちは通とおり抜ぬけられますか。

中文释义：小野：明白了。话说，这条路能穿过去吗？ 语法解析

  • ところで：话题转换。
  • 可能形：通とおり抜ぬけられます（能穿过）。 形态变化
  • 分わかる → 分わかりました（过去礼貌体）。
  • 通とおり抜ぬける（基本形，一段）→ 通とおり抜ぬけられる（可能形）→ 通とおり抜ぬけられますか（礼貌疑问）。
李り：大だい丈じょう夫ぶです。入いり組くんでいますが、通とおり抜ぬけられますよ。

中文释义：李：没问题。虽然道路曲折，但可以穿过去的。 语法解析

  • 入いり組くむ→入いり組くんでいる：连用/状态。
  • 〜が：逆接。
  • 〜ますよ：说明+肯定语气。 形态变化
  • 入いり組くむ → 入いり組くんでいます（进行）。
  • 通とおり抜ぬける → 可能形 通とおり抜ぬけられる → 礼貌体 通とおり抜ぬけられます。
小お野の：胡フー同トンは、古ふるい北ぺ京きんが感かんじられて、楽たのしいですね。

中文释义：小野：在胡同里能感受到老北京，很有意思呢。 语法解析

  • 受身自发：感かんじる → 感かんじられる（“能感到/令人感到”）。
  • 〜て：顺接并列（感到…而且…有趣）。 形态变化
  • 感かんじる → 感かんじられる（受身自发）。
李り：ええ。だから、時じ間かんがある時ときや、外がい国こく人じんの友とも達だちが来きた時ときは、胡フー同トンを歩あるくようにしているんですよ。

中文释义：李：对。所以有时间，或者外国朋友来的时候，我都会尽量在胡同里走走。 语法解析

  • 〜や：并列（……或……）。
  • Vた＋時とき：过去事实修饰“时候”。
  • V-辞书形＋ようにしている：形成习惯/尽量做。 形态变化
  • 来くる → 来きた（过去）→ 来きた時とき。
  • 歩あるく（基本形）→ 歩あるくようにしている。
李り：小お野のさん、「油ヨウ条ティアオ」は食たべられますか。

中文释义：李：小野，你能吃“油条”吗？ 语法解析

  • 「……」：外来/特定食物名引用。
  • 可能形：食べられます（能吃）。 形态变化
  • 食たべる → 食たべられる（可能/被动）→ 食たべられますか（礼貌疑问）。
小お野の：ええ、大だい好すきです。

中文释义：小野：嗯，非常喜欢。 语法解析

  • 形动 大好きだ → 礼貌体 大好きです。 形态变化：无活用（名词/形动判断）。
李り：熱あついですから、火や傷けどしないように気きをつけてくださいね。

中文释义：李：因为很烫，请注意不要烫到。 语法解析

  • 〜ですから：原因。
  • V-ない＋ように：避免。
  • V-てください：请求。 形态变化
  • 気きをつける（基本形）→ つけてください（请求）。
  • 火や傷けどする → しないように（否定＋规避）。
李り：最さい近きん、マンションやビルが建たって、胡フー同トンがだいぶ減へってしまいました。

中文释义：李：最近小区和大楼盖了不少，胡同也减少了很多。 语法解析

  • 〜て、〜：并列顺接。
  • V-てしまう：完了/遗憾（“……掉了/不幸……”）。 形态变化
  • 建たつ → 建たって（て形）。
  • 減へる → 減へってしまいました（てしまう＋过去敬体）。
小お野の：なんだか残ざん念ねんですね。

中文释义：小野：总觉得有点遗憾呢。 语法解析

  • なんだか：副词“总觉得”。
  • 〜ですね：确认/感叹。 形态变化：无。
李り：そうなんです。今いまでは胡フー同トンが見みられる場ば所しょは本ほん当とうに少すくなくなりました。

中文释义：李：确实如此。现在能看到胡同的地方真的变少了。 语法解析

  • 可能/受身：見みられる（能被看到）。
  • A く＋なる：形容词连用形＋なる（变化）。
  • では：对比界面“在……则”。 形态变化
  • 見みる → 見みられる（可能/受身）。
  • 少すくない → 少すくなくなる（连用形＋なる）→ 少すくなくなりました（过去礼貌体）。

小结（常用表达）

  • V ないようにしてください：请尽量不要……
  • V られる（可能形）：能……
  • V ようにしている：养成习惯/尽量……
  • V てしまう：做完/不慎……（语气带遗憾）
  • A くなる：变得……

新标初_39 故宫

故こ宮きゅう

李り：この門もんは、昔むかし、火か事じのために２に度ども焼やけたことがあるんですよ。

中文释义：李：这道门以前因为火灾有过两次被烧毁的经历哦。 语法解析

  • 「〜ために」表示原因/目的，这里作“因为火灾”。
  • 「V-たことがある」表示曾经有过的经历。
  • 「焼ける」自他动对：此处用自动词“着火/被烧”。
  • 终尾「んです」提示说明语气，「よ」加强提示。 形态变化
  • 焼やける（基本形）→ 焼やけた（过去形）→ 焼やけたことがある。
小お野の：そうなんですか、それにしても立りっ派ぱな建た物てものですね。中なかに入はいれるんですか。

中文释义：小野：这样啊；即便如此也真是气派的建筑呢。能进去吗？ 语法解析

  • 「それにしても」转折强调“即便如此”。
  • 「立派な」形动连体形修饰名词。
  • 「V-れる」可能形表“能……”，这里是「入る → 入れる」。
  • 句尾「んですか」礼貌、求确认。 形态变化
  • 入はいる → 可能形 入はいれる。
李り：ええ、入はいれますよ。チケットを買かってきましょうか。

中文释义：李：能进去。我去买票过来吧？ 语法解析

  • 「V-ますよ」断定+提示。
  • 「V-てくる／V-てきましょうか」表示去做后带回来；「〜ましょうか」提出建议/征询。 形态变化
  • 買かう → 買かって（て形）→ 買かってくる → 〜きましょうか（意向疑问）。
李り：かばんを持もっては入はいることができませんから、ここに預あずけていきましょう。

中文释义：李：不能带包进去，所以把包寄存在这里再进去吧。 语法解析

  • 「V-てはならない／できない」：禁止/不能。此处「持っては入ることができません」。
  • 「〜から」表原因。
  • 「V-ていく」表示做了再去/带着去；这里是“把包寄存后再走”。
  • 「〜ましょう」提议。 形态变化
  • 持もつ → 持もって（て形）；
  • 入はいる → 入はいることができない（可能否定）；
  • 預あずける → 預あずけていく → 〜いきましょう。
小お野の：天てん安あん門もんからはいろんな建た物てものが見み渡わたせるんですか。

中文释义：小野：从天安门能眺望到各种建筑吗？ 语法解析

  • 「見渡す」他动：眺望/俯瞰。
  • 「〜せる」使役？此处是可能形「見渡せる」（五段→连用音便 す→せる）。
  • 「んですか」说明性提问。 形态变化
  • 見み渡わたす → 可能形 見み渡わたせる。
李り：ええ、でも今け朝さは霧きりがかかっているために、遠とおくまで見みえないと思おもいますが…。

中文释义：李：嗯，不过今天早上起雾了，恐怕看不到很远…… 语法解析

  • 「〜ために」原因。
  • 「V-ている」状态“笼着雾”。
  • 「〜ないと思います」推量。
  • 「が…」转折+语气延宕。 形态变化
  • 見みえる（基本形）→ 見みえない（否定）→ 見みえないと思おもいます。
李り：ここは、別べつ名めい「紫し禁きん城じょう」と言いって、映えい画がの舞ぶ台たいになった所ところです。

中文释义：李：这里又叫“紫禁城”，是电影的取景地。 语法解析

  • 「A と言って」相当于“称作……”。
  • 「N になる」表示成为；「〜になった」过去。
  • 「所」名词化地点“……的地方”。 形态变化
  • 言いう → 言いって（引用连接）；
  • なる → なった（过去）。
小お野の：そうそう。私わたしは映えい画がを見みて、絶ぜっ対たい来きたいと思おもっていたんです。

中文释义：小野：对对，我看了电影后就一直想着一定要来。 语法解析

  • 「V-て、…」顺接因果：看了电影→产生想法。
  • 「V-たい」愿望；「V-たいと思う」表意愿；「〜ていた」过去持续（一直在想）。 形态变化
  • 来くる → 意愿 来きたい；
  • 思おもう → 思おもっていた。
（李り）せっかく北ぺっ京きんへ来きて、ここを見みないで帰かえることはできませんよ。

中文释义：李：（千里迢迢）都来北京了，不看这里就回去可不行啊。 语法解析

  • 「せっかく〜」表示“好不容易/难得”。
  • 「V-ないで」不……就……；
  • 「V-ることはできない」不允许/不能做某事。 形态变化
  • 見みる → 見みないで；
  • 帰かえる → 帰かえることはできない。
小お野の：屋や根ねが黄き色いろで、とてもきれいですね。

中文释义：小野：屋顶是黄色的，真漂亮呢。 语法解析

  • 「〜で」并列说明（黄色 + 漂亮）。 形态变化：无活用。
李り：ええ、あれは「瑠る璃り瓦がわら」と言いいます。青あおい空そらにとてもよく合あいますね。

中文释义：李：嗯，那叫“琉璃瓦”。和蓝天非常相配呢。 语法解析

  • 「N と言う」定义命名。
  • 「V-ます」礼貌叙述；「合う」自动词“相配/协调”。 形态变化
  • 言いう → 言いいます；
  • 合あう → 合あいます（礼貌体）。
小お野の：李りさん、あの建たて物ものをバックにして、一いっ緒しょに写しゃ真しんを撮とりませんか。

中文释义：小野：李先生，以那座建筑当背景，一起拍张照吧？ 语法解析

  • 「N をバックにして」固定表达：以……作背景。
  • 「V-ませんか」邀请“要不要一起……”。 形态变化
  • 撮とる → 撮とりませんか（否定疑问邀请）。
李り：いいですね。

中文释义：李：好啊。 语法解析：赞同响应的常用句。 形态变化：无。

重点语法速查

  • V-たことがある：曾经有过……的经历。
  • V-れる（可能形）：能……（五段多为「a段＋れる」；一段为「られる」）。
  • V-ていく／V-てくる：边做边去／做了带回来。
  • V-てはならない／できない：禁止／不能。
  • N をバックにして：以……为背景。
  • せっかく〜のに／せっかく〜から：难得……却／既然难得……就。

新标初_40 京剧

京きょう劇げき

森もり：すみません。お待またせして。

中文释义：森：不好意思，让你们久等了。 语法解析

  • 「おV-せする」＝「お待たせする」的连用形，表示使对方等待的自谦表达。此处省略了「すみませんが」「してしまって」等后续。
  • 基本形：待またせる（使役）→ お待またせする（自谦）→ 连用形 お待たせして（接续道歉）。
小お野の：いいえ、私わたしたちもちょうど前まえに着ついたところです。

中文释义：小野：不会，我们也刚刚到。 语法解析

  • 「V-たところだ」表示刚做完；时间点紧邻现在。
  • 基本形：着つく → 着ついた（过去）→ 〜たところ（刚刚）。
森もり：そうですか。よかった。チケットは？

中文释义：森：这样啊，太好了。票呢？ 语法解析

  • 省略主语宾语的口语简洁问法。「は？」提示主题“票”。
李り：これから買かうところです。

中文释义：李：正要去买。 语法解析

  • 「これから」表从现在起/接下来。
  • 「V-ところだ」可表三个阶段：直前（V-るところ）、最中（V-ているところ）、刚完（V-たところ）。这里是 V-るところ → “正要……”。
  • 基本形：買かう → 買うところ。
森もり：じゃあ、ぼくが買かってきますよ。ここで待まっていてください。

中文释义：森：那我去买吧。请在这里等着。 语法解析

  • 「V-てくる」表示去做后回来；「〜ますよ」提出主动、安慰对方。
  • 「V-ている＋ください」= 请保持某状态（一直等着）。
  • 形态：買かう → 買かって（て形）→ 買ってくる／買ってきます；待まつ → 待まっている→ 待っていてください。
小お野の：この劇げき場じょう、新あたらしそうですね。

中文释义：小野：这个剧场看起来挺新呢。 语法解析

  • 「〜そうだ」接形容词词干，表样态推测“看起来……”。
  • 基本形：新あたらしい → 新しそう（样态）。
李り：ええ、去きょ年ねん改かい築ちくしたばかりです。

中文释义：李：嗯，去年刚改建过。 语法解析

  • 「V-たばかり」表示刚刚（说话人主观上时间不长）。
  • 基本形：改かい築ちくする → 改かい築ちくした → したばかり。
：それまでは、清しんの時じ代だいにできた建たて物ものを修しゅう理りしながら使つかい続つづけていました。

中文释义：在那之前，一直一边修缮一边使用清代建成的建筑。 语法解析

  • 「V-ながら」一边……一边……（同一主体同时进行）。
  • 「V-続ける」表示持续反复动作；「〜ていました」过去进行/过去状态。
  • 基本形：
    • 修しゅう理りする → 修しゅう理りしながら；
    • 使つかう → 使つかい続ける → 使い続けていました。
李り：小お野のさん、いかがでしたか。

中文释义：李：小野，感觉怎么样？ 语法解析

  • 「いかが」=「どう」的礼貌形；「〜でしたか」询问过去感受/结果。
小お野の：すばらしかったです。見み終おわったばかりで、まだ耳みみの奥おくに音おん楽がくが残のこっています。

中文释义：小野：太精彩了。刚看完，耳畔里还回荡着音乐。 语法解析

  • 「V-たばかり」= 刚刚；
  • 「V-ている」表状态持续（音乐余音犹在）。
  • 基本形：
    • 見み終おわる → 見み終おわったばかり；
    • 残のこる（自动）→ 残っています。
森もり：あれっ、小お野のさんは？

中文释义：森：咦，小野人呢？ 语法解析

  • 「は？」提示主题+追问去向，口语省略动词。
李り：小お野のさんなら、あそこです。

中文释义：李：要找小野的话，在那边。 语法解析

  • 「N なら」提示话题条件“若说到N的话……”。
今いま、舞ぶ台たいの上うえで、役やく者しゃさんと一いっ緒しょに写しゃ真しんを撮とっているところですよ。

中文释义：现在他正在台上和演员一起拍照呢。 语法解析

  • 「V-ているところだ」表示动作进行的正当中。
  • 终助词「よ」提示/强调。 形态变化
  • 撮とる → 撮とって（て形）→ 撮っているところ → ですよ（提示）。

重点语法总结

  • おV-する／おV-せする：自谦，给对方添麻烦（お待たせして）。
  • V-たところ／V-るところ／V-ているところ：刚做完／正要做／正在做。
  • V-てくる：去做后回来；V-ていてください：请持续保持该动作/状态。
  • Aそうだ（样态）：看起来……。
  • V-たばかり：刚刚；V-続ける持续。
  • V-ながら：一边……一边……（同一主语）。
  • なら提示话题/条件。
