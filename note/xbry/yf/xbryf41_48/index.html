新标初_41 小李收到部长的表扬

☆受身形、被动形式

被动形式是表示做主语的人或事物承受某种动作或影响的表达方式。

  • 一类动词：把基本形的最后一个音变成相应 あ 段上的音，再加 れる。

  • 二类动词：把基本形的 る 变成 られる 。

  • 三类动词：把 把 する 变成 される ，来くる 变成 来 こられる 。

基本形读音被动
するするされる
来くるくる来こ られる
  • (ら)れまる 的活用方式与 二类动词 相同。

☆名は 名に 动(ら)れる

表示被动时，动作对象做主语，而动作主体用助词 に 表示。

部ぶ 長ちょう は 李りさん を ほめました。

部长表扬了小李。

李りさん は 部ぶ 長ちょう に ほめられました。

小李被部长表扬了。

森もりさん は 小お 野のさん に 食しょく 事じ に 誘さそわれました。

森先生被小野女士邀请去吃饭了。

李りさん は 部ぶ 長ちょう に ほめられませんでした。

小李没有被部长表扬。

☆~てもらう

表示受益时，一般不用被动形式，而是用 ~てもらう 的形式。

李りさん は 友とも 達だち に 助たすけられました。

小李得到了朋友的帮助。

李りさん は 友とも 達だち に 助たすけて もらいました。

小李让朋友帮了忙／得到了朋友的帮助。

☆名は 名に 名を 动(ら)れる

日语中有一种物主做主语的被动句，即某事物的拥有者在被动句里做主语，而该事物在被动句里仍然充当宾语。

这种被动句一般表示该事物的拥有者遭受了某种麻烦或损失。

森もりさん は 馬まさん の カメラ を 壊こわしました。

森先生把马先生的相机弄坏了。

馬まさん は 森もりさん に カメラ を 壊こわされました。

马先生的相机被森先生弄坏了。

李りさん は 昨きの 日う 犬いぬ に 手て を かまれました。

小李昨天被狗咬了手。

森もりさん は 田た 中なかさん に 本ほん を 汚よごされました。

森先生的书被田中先生弄脏了。

☆名は 名に 动(ら)れる

日语中还有一种纯粹表示受害的被动句，在这种被动句里，受害者不是直接的而是间接的承受某种事态的影响。

不仅是他动词，自动词也可以用于这种被动句。

另外，这种被动句一般不具有否定形式。

小こと 鳥り が 逃にげました。

小鸟逃走了。

陳ちんさん は 小こと 鳥り に 逃にげられました。

陈先生被小鸟跑掉了。（= 小鸟逃走了，使陈先生感到困扰。）

張ちょうさん は 隣となり の 人ひと に 夜よる 遅おそく まで 騒さわがれました。

小张被邻居吵到很晚。（这是 迷惑受身：对方的行为让自己受到困扰。）

在这种被动句里，助词 に 前面一般是人或动物，但有时候也可以是 雪 和 雨 。

不过这时候的动词必须是 降ります 。

わたしたち は 雨あめ に 降ふられました。

我们被雨淋了。

☆名が/は 动(ら)れる

日语中还有一种用事物来做主语的被动句。这种被动句的动作主体一般是某一不确定的人的群体，一般不在句中出现。

2010年ねん に 上しゃん 海はい で 万ばん 博ぱく が 開ひらかれます。

2010 年在上海举办世博会。

駅えき 前 に 高たかい ビル が 建たてられました。

车站前建起了一栋高楼。

☆名は 名によって 动(ら)れます

在用事物做主语的被动句里，有时动词的主体是特定的，这时候动作主体用 によって 来提示。

によって 不能用 に 来替换。

この 車くるま は 日に 本ほん の 有ゆう 名めいな デザイナー によって 設せっ 計けいされました。

这辆车是由日本的著名设计师设计的。

（“〜によって”用于受身句中，强调动作的执行者，多用于正式、客观的说明。）

この 本ほん は 山やま 田だ 先せん 生せい によって 書かかれました。

这本书是山田老师写的。

（受身形＋「によって」＝ 强调动作的执行者，多用于正式说明或书面语。）

新标初_42 开着电视就出门

⭐️まま

表示将理应改变的状态保持着去进行另外的动作。

接续：

  • 小句1(动た形/ない形)まま，小句2

  • 名 + の + まま，小句

  • まま 前为肯定形式时，用 动た形 + まま 。

  • まま 前为否定形式时，用 动ない形 + まま 。

テレビ を つけた まま、 出でかけて しまいました。

把电视开着就出门了。（带有“结果上不太好／不应该”的懊悔语气。）

李りさん は コート を 着きた まま、 ベッド で 寝ねて しまいました。

小李穿着大衣就在床上睡着了。

馬まさん は 行いき先さき を 告つげない まま、 家いえ を 出でて しまいました。

小马没告诉去向就出门了。

わたしたち は 空くう 腹ふく の まま、 出しゅっ 発ぱつしました。

我们保持着肚子空空的状态就出发了。

⭐️のに

事实性逆接，“明明……却……”，后项一般是事实

  • 如果 小句1 为二类形容词小句和名词小句时，要用 二类形/名词 + な + のに 的形式。
  • のに 前一般不用敬体形。
目め 覚ざまし を かけて おいた のに、 今け 朝さ は 起おきられませんでした。

明明已经把闹钟设好了，今天早上还是没能起床。

今きょ 日う は こんな に 寒さむい のに、 森もりさん は 寒さむくない と 言いいました。

明明今天这么冷，森先生却说不冷。

この 家いえ は 駅えき から 近ちかくて 便べん 利り なのに、 買かう 人ひと が いません。

这房子离车站近又方便，然而却没有人买。

森もりさん は 昨きの 日う 病びょう 気きだった のに、 出しゅっ 勤きんしました。

森先生明明昨天生病了，却还是去上班了。

田た 中なかさん は 作さっ 家か なのに、 読どく 書しょ が 嫌きらい です。

田中先生明明是作家，却讨厌读书。

~が 、 ~けど 的意思与 ~のに 相近，但其前后的两个小句独立性强

因此 小句1 句尾可以使用 ~です 、 ~ます 、 ~でしょう 等形式，并且 小句2 可以为 命令句 。

与此相反， ~のに 前后的两个小句独立性弱，因此 小句1 句尾不可以使用 ~です 、 ~ます 、 ~でしょう 等形式，并且 小句2 也不可以是祈使、请求句式。

⭐️はず

表确信的推断，“理应”

はず，形式名词

小句为二类形容词小句和名词小句时，分别使用 二类形 + な + はずです 、 名词 + の + はずです 的形式。

会かい 議ぎ は 5時じ まで です から、 もうすぐ 終おわる はず です。

会议开到 5 点，所以应该快结束了。

「〜はずです」＝ 根据理由或常识做出的理所当然的推测。）

この 映えい 画が は 人にん 気き が あります から、 観かん 客きゃく は 多い はず です。

这部电影很有人气，所以观众应该会很多。

由于 ~はずです 表示的是根据某种理由作出的判断，因此既可以用于说话人自身经历过，但记忆已经不十分准确，也可以用于实际情况与自己的考虑不一致。

⭐️ はずがありません / はずはないです

~はずがありません 是 ~はずです 的否定形式

表示根据某种理由作出某种否定性的推断，也可以说成 ~はずはないです 。

小句为二类形容词小句和名词小句时，使用 二类形 + な + はずがありません 、 名词 + の + はずがありません 的形式。

張ちょうさん は 入にゅう 院いん 中ちゅう です から、 旅りょ 行こう に 行いく はず が ありません。

小张正在住院，所以不可能去旅行。

7月がつ の オーストラリア は 冬ふゆ です から、 暑あつい はず が ありません。

澳大利亚的 7 月是冬天，所以不可能热。

新标初_43 老陈让儿子去美国留学

⭐️ 使役形

使役形式是表示使役主体强制或指示动作主体进行动作的语法形式，也可以用来表示使役主体对动作主体的行为的许可或放任。

  • 一类动词：把 ない形 的 ない 变成 せる 。
  • 二类动词：把 ない形 的 ない 变成 させる 。
  • 三类动词：把 来る 变成 来させる ( こさせる )，把 する 变成 させる 。

注意，使役形式与可能形、被动形一样，都是二类动词。

⭐️ 名は 名を 自动(さ)せます

在使役句里不是动作主体，而是使役主体做主语，

  • 如动词是 自动词 时，动作主体用助词 を ・に　表示。
李りさん は 出しゅっ 張ちょうします。

小李要出差。

部ぶ 長ちょう は 李りさん を 出しゅっ 張ちょうさせます。

部长让小李去出差。

陳ちんさん は、 息むす 子こ を アメリカ に 留りゅう 学がくさせます。

陈先生让儿子去美国留学。

⭐️ 名は 名に 名を 他动(さ)せます

在使役句里不是动作主体而是使役主体做主语，如动词是 他动词 时，动作主体用助词 に 表示。

这里的动作主体不用 を 而用 に 表示，是因为在日语中一般尽量避免在同一个句子里重复使用 を 的缘故。

森もりさん は 歌うた を 歌うたいます。

森先生唱歌。

陳ちんさん は 森もりさん に 歌うた を 歌うたわせます。

陈先生让森先生唱歌。

部ぶ 長ちょう は 太おお 田たさん に レポート を 書かかせました。

部长让太田先生写了报告。

⭐️ 使役被动形

由使役形式加被动形式构成，一般表示一种被迫性的动作。其具体构成方式如下：

  • 一类动词：把 ない形 的 ない 变成 される 。
  • 二类动词：把 ない形 的 ない 变成 させられる 。
  • 三类动词：把 来る 变成 来させられる ( こさせられる )，把 する 变成 させられる 。
選せん 手しゅ は 走はしります。

选手（运动员）跑步／奔跑。

監かん 督とく は 選せん 手しゅ を 走はしらせます。

教练让选手跑步。

選せん 手しゅ は 監かん 督とく に 走はしらされます。

选手被教练逼着跑步。

（这是 使役受身形：“被迫做……”，语气比普通受身更带无奈感。例如：被逼跑、被迫参加训练。）

⭐️ 动(さ)せてください

不太礼貌

  • 请求别人允许自己做某事。

在这个句型里，说话人一般不出现。如需要特别强调，可以用 わたしを 、 わたしに 的形式。

疲つかれました。 （わたし を） 少し 休やすませて ください。

累了。请让我休息一下。

⭐️ 动(连用形)やすいです/にくいです

安やすい

難にくい

  • ~やすいです 表示事物有 易于~ 的倾向
  • ~にくいです 表示事物有 难于~ 的倾向
この ボールペン は とても 書かきやすい です。

这支圆珠笔很好用。

この 本ほん は 読よみやすい です。

这本书很容易读／读起来很轻松。

（「〜やすい」表示“易于……、好……”，强调阅读体验很顺畅。）

新标初_44 门厅那儿好像有人

⭐️ 推测表达 ~（简）+ ようだ

様よう：样态

接续：

  • ~（简）+ ようだ
  • AN + な+ ようだ
  • N + の + ようだ
玄関げんかんのところにだれかいるようです。

玄关那边好像有谁在。

  • ~ようだ：一句状况作出推测，助动词
よくよく 見みえるように、大おおきく書かきました。

为了能看清楚，我写得大一点。

  • ~ように：表目的，“为了”
  • Vる、Vない、V可能形 + ように
エンジンえんじんが故障こしょうしたようようです。

好像是发动机出故障了。

店みせの前まえには行列ぎょうれつができています。あの店の料理りょうりはおいしいようようです。

店门口排起了长队，那家店的菜似乎很好吃。

値段ねだんはほかの店の倍ばいです。ここの料理はかなり豪華ごうかなようようです。

价钱是别家店的两倍，这里的菜看起来相当豪华。

遠とおくから何なにか飛とんできました。飛行機ひこうきのようようですね。

有什么东西从远处飞过来了——好像是飞机呢。

——そのようですね。

是那样的啊（表示附和或同意对方的推测）。

⭐️ ~（简）みたいだ

表示根据说话人感知的某种情况进行推测，是比较随便的说法。

小野おのさんは森もりさんがすきみたいです。

中文释义： 小野小姐好像喜欢森先生。

语法说明：

  • 「〜みたいです」：表示根据主观印象的推测，多用于口语，比「ようです」随意。 （区别：ようです偏客观、みたいです偏主观）
  • ~（简）みたいだ
  • AN/N だ + みたいだ
小野おのさんは森もりさんをよく見ていますね。小野おのさんは森もりさんが好すきみたいです。

小野小姐经常看着森先生呢，好像喜欢他哦。

雨あめの音おとが聞きこえません。雨あめがやんだみたいです。

听不到雨声了，好像雨已经停了。

外そとの人はみんなコートを着きています。外は寒さむいみたいです。

外面的人都穿着大衣，看来外面很冷。

雲くもが広ひろがっています。明日あしたは雨あめみたいですよ。

天上布满了云，明天好像要下雨呢。

⭐️ らしい 【推测】【 传闻 】

  • らしい ：助动词
  • 依据观察到的情况做出推测
  • 传闻，委婉地表示听来的信息

接续：

  • ~(简)らしい
  • AN/N だ + らしい
今度こんどの社員旅行しゃいんりょこうは韓国かんこくへ行くらしいです。

中文释义： 听说明次员工旅行要去韩国。

语法说明：

  • 「〜らしいです」：表示传闻（听别人说）。 例如：「田中さんはけっこんしたらしいです。」（听说田中结婚了。）
李りさんは頭あたまが痛いたいらしいです。

听说小李头疼。

林はやしさんはお酒さけが好すきらしいですよ。

听说林先生喜欢喝酒哦。

電気でんきが消きえています。李りさんは出でかけたらしいです。

灯关着呢，听说小李出门了。

パトカーのサイレンが聞きこえます。近所きんじょで事故じこがあったらしいです。

听到警车的警笛声，听说附近发生了事故。

🌸ようだ・みたいだ・らしい・そうだ 的系统整理

①ようだ、みたいだ、そうだ，后项接续「だ」。

  • おいしそうなケーキですね。
    • 看起来很好吃的蛋糕呢。
  • 子供こどもは勉強べんきょうしているようで、静かにして。
    • 孩子好像在学习，请安静点。
  • 休みみたいだから、別の店に行こう。
    • 看样子关门了，去别的店吧。

② らしい可作形容词活用

森もりさんは風邪かぜをひいたらしく（て）、会社に来なかった。

听说森先生感冒了，所以没来公司。

  • ようだ、 みたいだ：依据状况作出的推测，意思基本相同；可以表比喻
    • ようだ 书面语偏多，正式的口语表达
    • みたいだ 比较随意的口语表达
  • らしい ：依据状况做出推测；传闻；
    • 责任感不强（小道消息）
    • 一般多用于小道消息、绯闻等、不负责；
  • そうだ：依据样态做出推测，“看起来”；传闻
    • 一般有明确的消息来源

新标初_48 您的行李我来拿

自谦语是通过压低说话人自身或说话人一方的形式表示敬意的表达方式。如跟别人谈到自己的家人或自己公司的上级时，也使用自谦语。

  1. 公式形：お/ご~する
  2. 特定性：行く、来る →伺うかがう・参まいる
    1. 謙譲語けんじょうご1：表达对动作接受者的敬意（→伺うかがう）
    2. 謙譲語けんじょうご2-丁重语：表达对听话人/读者的敬意（→参まいる）
  3. お・ご+~ 等

公式形

  • 将自己的行为降低，表达对别人的敬意

お + V1/2(连用) + する

ご + V3(汉字) + する

不可用：

  • V1/2(连用) 只有一个音节的词
  • V3： 来る・する
お荷物にもつは 私（わたくし）がお持もちします。
