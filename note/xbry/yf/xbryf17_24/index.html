第17课 我想有（套）新西服

☆愿望句

☆文型1：名 が　ほしいです

解析：ほしい 形1 想要

黒くろいワンピースが欲ほしいです。

我想要一件黑色连衣裙。

ワンピース　③　（名）连衣裙；英语 one-piece

子供こどもが欲ほしいです。

我想要孩子。

私わたしは 古ふるい 本ほんが 欲ほしくないです。

我不想要旧书。

  • 古ふるい　②　（い形）旧的

☆文型2：动ますたいです

解析：想进行某个动作

意思：想……

赤あかいカメラを 買かいたいです。

我想买一台红色的相机。

  • 动ますたい相当于一类形容词，变形同一类形容词
赤あかいカメラを 買かいたかったです。
お酒さけを 飲のみたいです。

我想喝酒。

仕事しごとの後あと、デパートへ　行いきたいです。

工作之后想去百货商店。

薬くすりを　飲のみたくないです。

我不想吃药／不想服药。

☆邀请，劝诱，提议

文型：动ます ましょう、ませんか

解析：邀请，劝诱，提议对方一起做某事

ちょっと休やすみましょう。

稍微休息一下吧。

仕事しごとの後あと、一緒いっしょに帰かえりましょう。

工作之后，一起回去吧。

☆全面肯定

文型：疑问词 + でも+ 肯定

解析：表示任何情况下事态都相同。

どこでも　行いけます。

哪里都能去。

誰だれでも　いいですか。

谁都可以吗？

いつでも　電話でんわをしてください。

随时请打电话。

第18课 手机变小了

☆形容词 + なる

解析：描述事情性质或状态的形容词，表示变化时，要借助动词【なる】。且该变化多为自然而然地变化。

文型：

  • 形1 ~い →く+なる
  • 形2 词干 + に+ なる
電気でんきをつけて、部屋へやは明あかるくなりました。

把灯一打开，房间就亮了。／开了灯后，房间变亮了。

風邪かぜは 良よく なりました。

感冒好起来了。

最近さいきん、 暖あたたかく なりましたね。

最近暖和了呢。

庭にわは きれいに なりました。

院子变得干净/漂亮了。

だんだん 上手じょうずになります。

会逐渐变得熟练/越来越拿手。

  • だんだん：副词，“渐渐、逐渐”。
  • ナ形容詞＋に＋なる：表示状态变化“变得……”。这里是「上手（じょうず）に なる」→“变得擅长”。

☆名 + に + なる

成为……；到了……（时候）

先生せんせい に なりたい です。

我想成为老师。

息子むすこ は 医者いしゃ に なりました。

我儿子成了医生。

春はる に なる。

到了春天／变成春天。

☆形容词 + する

解析：表示事物性质或状态的变化时由人为意志性的动作、作用而引起的。

文型：

  • 形1 ~い →く+する
  • 形2 词干 + に+ する
ジュースを冷たくつめたくします。

把果汁弄凉

静かしずかにしてください。

请安静。

冷たいつめたい　3（い形）冷的

静かしずか　1（な形）安静

説明せつめいを できるだけ 簡単かんたんに してください。

请把说明尽可能简单。

うるさいですから、ラジオの音おとを小さくちいさくしました。
それで、ラジオの音おとが小さくちいさくなりました。

因为很吵，所以把收音机的声音调小了。于是，收音机的声音变小了。

☆名1 を名2　に　する

人为性使名1成为名2的样子或状态

会議室かいぎしつを禁煙きんえんにします。

把会议室设为禁烟。

この点てんを中心ちゅうしんに（して）、半径はんけい5センチの円えんを描かきます。

以这个点为中心，画一个半径 5 厘米的圆。

☆名にする

表示选择或决定的结果

プレゼントはネねクくタたイいにしました。

礼物决定为领带了。

A:何なににしますか。

您要点什么呢？ / 您决定要什么呢？

B: カかレれーーにします。

我要咖喱。

☆~ほうがいいです

接续：

  • 形1 + ほうがいいです
  • 形2 +　な　+　ほうがいいです

解析：两个以上事物的性质，其中一个相对比较好时。

家いえは広ひろいほうがいいですか。

房子是宽敞一点比较好吗？

広ひろい形容词 (い形容词)宽阔的，宽敞的
ほうほう名词名词方面（用于比较）方面（用于比较）
店員てんいんは親切しんせつなほうがいいです。

店员还是亲切一点比较好。

電気製品でんきせいひんは新あたらしいほうがいいですね。

电器产品还是新的比较好呢。

代表的だいひょうてきな建物たてものは立派りっぱなほうがいいでしょう。

有代表性的建筑物还是气派一点比较好吧。

アパートは静しずかなほうがいいです。

公寓还是安静一点比较好。

レストランれすとらんは奇麗きれいなほうがいいです。

餐厅还是干净整洁比较好。

☆名　のほうがいいです

相比较来说还是……为好。

此この仕事しごとは独身どくしんの男性だんせいのほうがいいです。出張しゅっちょうが多おおいですから。

这份工作单身男性来做比较合适。因为出差很多。

独身どくしん名词单身
男性だんせい名词男性
仕事しごと名词工作
面接めんせつなら、スーツのほうがいいです。

如果是面试的话，还是穿西装比较好。

面接めんせつ名词面试
なら助词如果…的话；提起话题
スーツ名词西装，套装

☆自动词vs他动词

  • 自动词：标明主题本身的动作，不需要作用于其他事物，不需要宾语。
  • 他动词：主题的动作需要作用于其他事物，需要宾语を
会かい社しゃが潰つぶれてしまった。（潰つぶれる）

公司倒闭了。（倒闭）

会かい社しゃを潰つぶしてしまった。（潰つぶす）

（我）把公司搞垮了。/（我不小心）让公司破产了。（动词原形：搞垮）

如何区分自动词和他动词？？

  • 意思上分析
  • 单词结构特点
    • ア 相同汉字，相同结尾 “eru”他动 “aru” 自动
始はじめる集あつめる終おえる
始はじまる集あつまる終おわる
  • 单词结构特点
    • 相同汉字，不同结尾时，“su”结尾他动
出です壊こわす汚よぐす動うごかす
出でる壊こわれる汚よぐれる動うごく
  • 单词结构特点
    • “eru"结尾大多为自动词
动词中文释义核心含义说明
壊こわれる坏了，损坏指机器、物品等发生故障或破损。
汚よごれる脏了指物品或场所变得不干净。
切きれる断了；用完了指绳子、线等断开；也指期限已到或东西用完。
折おれる折断了指棒状、条状的东西（如树枝、铅笔）折断。
割われる裂了，碎了指玻璃、盘子等脆性物体破裂。
倒たおれる倒了；倒塌指人病倒或晕倒；也指建筑物等倒塌。

特殊：

为了更系统地学习，强烈建议将这两个他动词与它们的对应自动词一起记忆。

他動自動说明
入れる (放入)入はいる (进入)他动词强调“放”的动作；自动词强调“进入”的状态。
忘れる (忘记)(状态变化)“忘れる”本身已包含状态变化，较少强调对应的自动词。
  • 单词结构特点
    • 词源是形容词，构成“む”结尾的动词，一般是自・他动词。

单词词性中文释义核心含义
痛いたむ自他动词疼痛，苦恼多指身体或内心的疼痛。
苦くるしむ自他动词痛苦，烦恼，受苦指精神上或身体上感到痛苦、难受。
楽たのしむ自他动词享受，快乐，期待指从某事中获得乐趣，或愉快地期待某事。
悲かなしむ自他动词悲伤，惋惜，哀悼因为不幸的事情而感到悲伤。

也有一些动词本身是自他两用的。

单词词性中文释义核心概念
開ひく自他动词（门、窗等）开，打开描述“某物被打开”后的状态或“自动打开”的结果。
引ひく自他动词拉，抽，吸，吸引表示主体向自身方向施加力的动作。

不是所有的自他动词都是对应的。（相对自他动词 绝对自他动词）

单词 (原形)注音词性中文释义
あるある自动词有，存在（用于无生命物）
歩くあるく自动词走，步行
死ぬしぬ自动词死亡
殺すころす他动词杀，杀害
食べるたべる他动词吃
買うかう他动词购买

殺す、食べる、買う 等，是没有对应的自动词，是绝对的他动词！

第19课 请别忘了房间钥匙

☆动词ない形

  • 动词简体否定

  • 礼貌体否定 ません

一类动词：

词尾 → 该行 [あ]段假名　＋　ない

  • 如果词尾是：う　则变成 わ

  • 特殊：ある　→　ない

原形ない形
休やすむ休やすまない休息
会あう会あわない见面
帰かえる帰かえらない返回
终おわる终おわらない结束

二类动词：

去る＋　ない

原形ない形
見みる見みない见
食たべる食たべない吃
起おきる起おきない起床

三类动词：

する　→　しない

来くる　→　来くない

〜する　→　〜しない

☆动ない でください

解析：动てください相应的否定形

意思：请不要……

触さわらないでください。危あぶないですから。

请勿触摸。因为很危险。

傘かさを忘わすれないでください。

请不要忘记带伞。

無理むりをしないでください。

请不要勉强自己。/ 请别太拼命。

ここに入はいらないでください。

请不要进入这里。

☆动なければなりません

解析：表示必须进行前项动作。

意思：必须……

今日きょうは早はやく帰かえら なければ なり ません。

今天必须早点回家。

食事しょくじの前まえに、手てを洗あらわなければなりません。

吃饭前必须洗手。

单词词性中文释义
食事しょくじ名词吃饭，餐食
の助词的（表示修饰）
前まえ名词之前
に助词在（表示时间点）
手て名词手
を助词宾语标记
洗あらわなければなりません动词短语必须洗（“洗う”的义务形）
病院びょういんに行いかなければなりませんか。

我必须去医院吗？/ 我需要去医院吗？

宿題しゅくだいをしなければなりません。

（我）必须做作业。

☆动なくてもいいです

解析：表示不做某事也可以。

意思：即使不……也可以。

名前なまえを書かかなくてもいいです。

可以不写名字。/ 不写名字也没关系。

今日きょうは来こなくてもいいですが、明日あしたは来こなければなりません。

今天不来也可以，但是明天必须来。

これ、返かえさなくてもいいですよ。

这个啊，不还也可以哦。

第20课 丝密斯先生会弹钢琴

☆动（基本型）ことができる

こと把动词名词化

解析：

  • 表示能力
  • 表示条件状况允许
スミスすみすさんはピアノぴあのを弾ひくことができます。

史密斯先生/女士会弹钢琴。

单词词性中文释义
ピアノぴあの名词钢琴
弾ひくことができます动词短语会弹，能弹（“弾く”的能力形）
兄あにはフランス語ふらんすごの歌うたを歌うたうことができます。

我哥哥会唱法语歌。

お酒おさけを飲のみましたから、運転うんてんすることができません。

因为喝了酒，所以不能开车。

図書館としょかんで大声おおごえで話はなすことができません。

在图书馆不能大声说话。（这种条件下不允许）

☆名　は　动（基本型）ことです

解析：当谓语部分的内容是一个动作行为时，使用这个句型

  • 形式体言こと可以将动词名词化
趣味しゅみは切手きってを集あつめることです。

爱好是收集邮票。

单词词性中文释义
趣味しゅみ名词爱好，兴趣
切手きって名词邮票
集あつめる他动词收集
こと形式名词将动词名词化，指代“收集邮票”这件事
私わたしの夢ゆめは外国がいこくで働はたらくことです。

我的梦想是在国外工作。

仕事しごとは英語えいごを教おしえることです。

（我的）工作是教英语。

☆~前に

接续：

  • 名の　前に、〜
  • 动（基本型）前に、〜

解析：表示在此之前进行另外一个动作

来くる前まえに電話でんわをかけてください。

请（打电话）告诉我您要来。/ 请您来电告知您会前来。

单词词性中文释义
来くる动词来（“来る”的原形）
電話でんわ名词电话
を格助词宾语标记
かけてください动词短语请拨打（“かける”的请求形）
出でかける前まえにガスがすを消けしてください。

出门前请关掉煤气。

单词词性中文释义
出でかける自动词出门，外出
前まえ名词之前
に助词在（表示动作发生的时间点）
ガスがす名词煤气，燃气
を格助词宾语标记
消けしてください动词短语请关闭（“消す”的请求形）
食事しょくじの前まえに手てを洗あらいます。

饭前洗手。

单词词性中文释义
食事しょくじ名词吃饭，餐食
の助词的（表示修饰关系）
前まえ名词之前
に助词在（表示动作发生的时间点）
手て名词手
を格助词宾语标记
洗あらいます动词洗（“洗う”的礼貌体现在式）

☆时间数量词 + 前に

2年にねん前まえにここに引ひっ越こしました。

（我）两年前搬到了这里。

单词词性中文释义
2年にねん数量词两年
前まえ接尾词…之前
に助词在（表示时间点）
ここ代名词这里，这个地方
に助词到（表示移动行为的目的地）
引ひっ越こしました动词搬家（“引っ越す”的礼貌体过去式）
一ヶ月前いっかげつまえに佐藤さとうさんに会あいました。

一个月前见到了佐藤先生/女士。

单词词性中文释义
一ヶ月前いっかげつまえ名词一个月前
に助词在（表示动作发生的时间点）
佐藤さとうさん名词佐藤先生/女士（敬称）
に助词表示动作的对象（用于“会う”等动词）
会あいました动词见面，会见（“会う”的礼貌体过去式）
  • 句型： 【时间】に 【人】に 【动词】ました
  • 解析：一ヶ月前いっかげつまえに：这是句子的时间状语。「一ヶ月いっかげつ」是数量词“一个月”。
  • 注意「ヶ」是表记上的特殊用法，读音与「か」相同。

☆疑问词 か

解析：表示不确定

  • 回答时先说：はい、いいえ

意思：

  • 是否……
  • 有没有……
A: 部屋へやに 誰だれが いますか。

A: 房间里有谁？

B: 李りさんがいます。

B: 小李在。

A: 部屋へやに 誰だれか いますか。

A: 房间里有（人）吗？ / 房间里是不是有谁在？

B: はいはい、李りさんがいます。

B: 是的，小李在。

いつか一緒いっしょにテニスてにすをしましょう。

我们什么时候一起去打网球吧。

暑あついですね。何なにか飲のみませんか。

真热啊。我们喝点东西好吗？ / 天气好热，要不要喝点什么呢？

「何なにか」：「何」是疑问词“什么”，后续助词「か」表示不确定，意为“某些东西”或“什么（喝的）”。

A: 冬休ふゆやすみ、何処どこかへ行いきましたか。

（寒假，你去了哪里吗？）

B: はいはい、韓国かんこくに行いきました。

（是的，我去韩国了。）

第21课 我吃过日式牛肉火锅

☆动词的过去式-た形

  • 基本形：書かく
  • ます形：書かきます
    • 过去式：書かきました
  • た形：書かいた

一类动词：

  • う、つ、る　→促音变 った

第24课 （我）想小李马上就来

☆小句（简体形） と 思おもいます

  • AN/N だ と 思おもいます。

  • 我想小李马上就来。
  • 「と」是表示引用的格助词，将其前面的句子内容作为一个整体概念来引用，相当于引号“”的功能。～と 思います
  • 「思います」表示说话人自己的想法。“李さんはもうすぐ来る”是整个想法的内容。
田中たなかさんは 来こない と 思おもいます。
  • 我认为田中先生/女士不会来。
此このの 本ほんは 面白おもしろい と 思おもいます。
  • 我认为这本书很有趣。
日本にほんは 綺麗きれいだ と 思おもいます。
  • 我认为日本很美丽。
明日あしたは 良いい 天気てんきだ と 思おもいます。
  • 我认为明天会是好天气。
