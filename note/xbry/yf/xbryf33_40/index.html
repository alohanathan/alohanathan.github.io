36课

ばかり

だけ：表示限定（客观）

ばかり：表示量、次数多（主观）

日に 本ほん 語ご が できる の は　森もり さん だけ です。（限定）
周まわり は　日に 本ほん 人じん だけ です。（客観叙述）
周まわり は　日に 本ほん 人じん ばかり です。（主観感情）
うそ　ばっかり　言いってる。（主観感情強…口语：ばっかり）
森もり さん は 毎まい 日にち お 酒さけ ばかり飲のんで いる。

37

动词：ば形

優ゆう 勝しょう すれば、 オリンピック に 出しゅつ 場じょう する こと が できます。

如果夺冠，就能参加奥运会。

  • 希望后件成立，叙述其所需要的条件，终点在前件；
  • 表示恒常的依存关系，或一般法则（和と相似）。

① 希望后件成立，叙述其所需要的条件，终点在前件：

授じゅ 業ぎょう を 受うければ、 試し 験けん に 合ごう 格かく できる。

A：どうすれば、いいですか。

B：李り さん に 聞きけば、 わかります よ。

② 表示恒常的依存关系，或一般法则（和と相似）:

  • この ボタン を 押おす と、 電でん 源げん が 入はいります。（説明書‐恒常性）
  • この ボタン を 押おせ ば、 電でん 源げん が 入はいります。（回答询问‐希望）
  • 春はる に なる と、 桜さくら が 咲さきます。（説明）
  • 春はる に なれ ば、 桜さくらが 咲さきます。（希望）

④ 可用于假定条件（同たら），和反事实假定；（假定形）

雨あめ が 降ふれば／降ふったら、 マラソン 大たい 会かい は 中ちゅう 止し です。（假定条件）
あと 100 円えん あれば、 あの 鞄かばん が 買かえる のに。（反事实假定）

要是再有100日元就能买到那只包了

彼かの 女じょ が いれば、 遊ゆう 園えん 地ち へ 行いく の は 楽たのしい けど。

⑤ 原則上不加意志、希望、命令、依頼等表述（和と相似）

質しつ 問もん が あれば、 手て を あげて ください。
親おや が 許ゆるして くれれば、 かのじょ と 結けっ 婚こん する つもり です。（前后主语不同）

注意点：

提取结果（原文）：

  1. 希望条件成立，叙述其所需要的条件，重点在前件；

    ・授じゅ 業ぎょう を 受うければ、 試し 験けん に 合ごう 格かく できる。

  2. 表示恒常的依存关系，或一般法则；（和と相似，自然/机械）

  3. 多用于谚语；

  4. 可用于假定条件（同たら），和反事实假定；（假定形）

  5. （讲义写）原则上不加意志、希望、命令、依赖等表达（和と相似）

    ・質しつ 問もん が あれば、 手て を あげて ください。（状态性谓语）
    ・親おや が 許ゆるして くれれば、 彼かの 女じょ と 結けっ 婚こん する つもり です。（前后主语不同）

〜なら

・承接对方说的话（意志、愿望）； ・表示一种归结，后接说话人的判断；

天てん 安あん 門もん へ 行いく なら、 地ち 下か 鉄てつ が 便べん 利り です よ。

按你的规则（注音；纯假名不注；必要处词语整体标读）：

  • A：今日きょう は 忙いそがしい ん です が…。
    • 我今天有点忙……
  • B：忙いそがしい なら、 行いかなくて も いい です よ。
    • 要是忙的话，就不去也行哦。
  • A：会かい 議ぎ の 資し 料りょう は どこ です か。
    • 会议资料在哪里？
  • B：昨きのう の は 知しりません が、 今け 朝さ の 会かい 議ぎ の 資し 料りょう なら、 机つくえ の 上うえ に 置おいて おきました。
    • 昨天的我不清楚，今早的会议资料的话，我已经放在桌子上了。
今こん 夜や、 もし 暇ひま なら、 いっしょ に 食しょく 事じ に 行いきません か。

今晚要是有空，要不要一起去吃饭？

彼かれ が 犯はん 人にん で ない なら、 すぐ 釈しゃく 放ほう して ください。

如果他不是犯人，请立刻释放。

形容词的假定形

Aい → Aければ
AN/N＋だ → AN/N＋なら
AN/N＋である → AN/N＋であれば

① V・Aい后可用の，也可省略；

天てん 安あん 門もん へ 行いく の／ん なら、 地ち 下か 鉄てつ が 便べん 利り です よ。

要是去天安门，地铁更方便哦。

忙いそがしい の／ん なら、 行いかなくて も いい です よ。

要是忙的话，不去也行。

なら注意点：

提取结果（原文）：

  1. 承接对方说的话（意志、愿望）；

  2. 表示一种归结，后接说话人的判断；

  3. ～简体形＋なら，AN/N＋だ なら；

  4. V・Aい后可用の，也可省略；

    ・天てん 安あん 門もん へ 行いく の／ん なら、 地ち 下か 鉄てつ が 便べん 利り です よ。（～のなら）
    ・忙いそがしい の／ん なら、 行いかなくて も いい です よ。

と・たら・ば・なら

  • と
    • 真理／自然事件、恒常／机械操作、反复／习惯、发现／事实契机；
    • 假定性弱，说明继起关系；
    • 不加意志、希望、命令等；

结论一句话：と＝一到A就必然B（客观连续）；要给建议/命令/愿望，换ば／たら／なら。

  • なら

    • 承接对方说的意志、愿望；
    • 归结，后＋说话人的判断；

  • たら

    • 适用范围广；
    • 假定条件＋确定条件；
    • 可＋意志、希望、命令等；
    • 可用于事实契机（同と）。

  • ば

    • 希望后件成立；
    • 假定条件＋反事实假定；
    • 一般不＋意志、希望、命令等；
    • 恒常／一般（自然／机械）（同と）；

でも 示例

列举几项选择中比较典型的一项

映えい 画が でも 見みに 行いきません か。

一般不用于过去式

提取结果（原文）：

  • 誰だれ でも 参さん 加か できます。（全部）【第17課】
  • その 計けい 算さん は 子こ 供ども でも できます。（極端例子）【第35課】
  • 映えい 画が でも 見み に 行いきません か。（列挙典型）【第37課】

〜とか　〜とか　口语，前面可以接句子

パーティー で、 戴たい さん とか 楊よう さん とか、 いろいろ な 人ひと に 会あいました。

第38课 小戴会说英语

☆动词的可能形

可能形式：表示能够进行某动作的形式。

  • 一类动词：把基本形的最后一个音变成相应え段上的音，再加る。
基本形读音可能形
書くかくかける
急ぐいそぐいそげる
飛ぶとぶとべる
  • 二类动词：把基本形的る变成られる。
基本形可能形
食べる食べられる
寝る寝られる
  • 三类动词：把来る变成来られる，把する变成できる。
基本形读音可能形
来るくるこられる
するするできる

  可能形式的活用形式和二类动词相同。

可能形式(基本形)可能形式(ない形)可能形式(た形)可能形式(なかった形)
書ける書けない書けた書けなかった
食べられる食べられない食べられた食べられなかった
来られる来られない来られた来られなかった

  使用~ことができます时动词前面的助词不发生变化，但使用可能形式时，动词前面的助词を常常变成が。

口语中比较随意时：られる就变成了れる，没有ら

戴たい さん は 英えい 語ご が 話はなせます。

小戴会说英语

V可能形：表示能够做某事（能力、状况）

話はなす(一类动词)　

話はなせる(可能形：基本型，变成二类动词)

話はなせます

V可能形：看作二类动词活用

  • 戴たい さん は 英えい 語ご を 話はなす こと が できます。【第20課】

  • 戴たい さん は 英えい 語ご が 話はなせます。

① 格助词的使用

  • [人] が [もの・こと] が 可能

    • 戴たい さん が 英えい 語ご が 話はなせる。

  • [人] が [もの・こと] を 可能

  • 戴たい さん が 英えい 語ご を 話はなせる。

  • [人] に [もの・こと] が 可能

  • 森もり さん に（は） 英えい 語ご が 話はなせない よ。

②无意志动作不可使用可能形

見みえる 和 見みられる

よく見みえるように、大おおきく書かきました。

为了看得清楚，我写得大一点。

見みえる ：由于眼力、距离等客观因素可以看到；表示自燃地映入眼帘

見みられる：人为条件下可以看到；表示主观的能力

外そとは暗くらくて、何なにも見みえません。

外面太暗了，什么也看不见。

上野動物園うえのどうぶつえんでは、パンダが見みられます。

在上野动物园可以看到熊猫。

見みえる・聞きこえる自发动词、自然而然地

基本形、ない形、可能形 + ように

表目的，“为了~”

様よう

風邪かぜを引ひかないように、気きをつけています。

为了不感冒，我一直在注意。

いつでも出でられるように、準備じゅんびしてあります。

为了随时能够出发，已经准备好了。

子供こどもが触さわれないように、この薬くすりは棚たなのいちばん上うえに置おきましょう。

为了让孩子碰不到，把这药放在架子的最上面吧。

状态性描述 + ように

①非意志变化／状态 ＋ ように → よう（様）

非意志性V：Vる／Vない ＋ ように

  • 自己的意志无法实现的动作
  • 降る、忘れる、なる、ある、見える、聞こえる……

意志性V：V可能形（る／ない）＋ ように

  • 自己的意志可以实现的动作
  • 食べる、行く、起きる……

吃饭时动作，能吃饭是一种状态

ように前后的动作主可以不一致

~ように和~ために区别

為ため：强烈的要求达成目的的心情

明日あした早はやく起おきるために、目覚めざましをかける。

为了明天早点起床，定上闹钟。

様よう：希望事情会变成那样，为之努力

（祈愿）雨あめが降ふるように、神かみに祈いのる。

祈求下雨。

（为之努力）皆様みなさまの支ささえに少すこしでもなれるよう、頑張がんばります。

为了能稍微成为各位的支持，我会努力的。

动作主一致、不一致

ようになります

様よう：样态

~になる：表变化，陈述客观的变化结果

  • 变成了某种状态
  • 某种之前不存在的状态，现在存在了
けがが治なおって、歩あるけるようになりました。

伤好了，已经能走路了。

赤あかちゃんが１歳いっさいを過すぎると、言葉ことばを話はなすようになります。

婴儿一岁以后，就开始说话了。

着物きものを自分じぶんで着きられるようになりました。

已经能够自己穿和服了。

ようにします

様よう：样态

~にする：表变化，强调变化的人为性

  • 努力使某种行为、状态发生；
  • 表述努力维持某种习惯的动作；
陳ちんさんは毎日まいにち、英字新聞えいじしんぶんを読よむようにしています。

陈先生每天都尽量看英文报纸。

毎朝まいあさ、7時しちじに起おきるようにしています。

为了做到每天早上7点起床，我一直在努力（养成这个习惯）。

のどが痛いたいので，今日きょうはあまり話はなさないようにします。

因为喉咙痛，今天尽量不说话。
