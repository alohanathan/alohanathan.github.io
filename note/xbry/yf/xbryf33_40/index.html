新标初_33 电车突然停了

☆自动词 和 他动词

  • 自动词：陈述结果，状态；无宾语
  • 他动词：强调人为性；有宾语
  • 成对的自他动词
  • 有自动词，无对应他动词
  • 有他动词，无对应自动词

☆规则

大部分都没有用，一个字看都别看

记：

  • -aru 自动词
  • -reru 自动词
  • -su 他动词

☆自动词 + ている

ている表示动作正在进行。

  • 如果该自动词是表示动作的，则表示动作正在进行；
  • 如果该自动词是表示变化的，则表示结果的存续。
部へ 屋や の 電でん 気き が 消きえています。

房间的灯已经关着（呈关着的状态）。

壁かべ に は 大おおきな 絵え が 掛かかって います。

墙上挂着一幅大画。

この 自じ 動どう 車しゃ は 壊こわれています。

这辆汽车坏掉了（处于坏的状态）。

自动词て形 + います 和 自动词た形 都可以表示结果存续

  • 自动词た形 偏重表示变化的完成
  • 自动词て形 + います 则偏重表示变化完成后形成的结果状态。
部へ 屋や の 電でん 気き が 消きえています。

房间的灯灭着

部へ 屋や の 電でん 気き が 消きえました。

房间的灯灭了（“灯熄灭了”这个动作刚刚发生）。

☆他动词后续 + ている

他动词后续 ~ています 一般表示现在进行，但也有一些例外，例如：

森もりさん は 車くるま を 持もっています。

森先生有车（拥有的状态）。

吉よし 田ださん は 眼め 鏡がね を かけています。

吉田先生戴着眼镜（处于戴着的状态）。

☆动 て + しまう

でしまう ＝ じゃう

てしまう ＝ ちゃう

しまう 完了， 终了的意思

表示一种后悔、遗憾、责备的语感。

一般表示该动作所产生的结果是令人不愉快的事情。

森もりさん は ボーナス を 全ぜん 部ぶ 使つかって しまいました。

森先生把奖金全都花掉了（带有“完全花光”或“有点后悔/遗憾”的语气）。

大たい 切せつ な 書しょ 類るい を うっかり 忘わすれて しまいました。

重要的文件不小心忘带了／忘记了。（带有懊悔或困扰的语气）

無む 理り を する と、 病びょう 気き に なって しまいます よ。

勉强自己的话，会生病的哦。

但有时候也只用来表示动作的 完成 ，而没有结果令人不愉快的意思，这时候动作一般是有意进行的。

この 本ほん は もう 読よんで しまいました。

这本书我已经读完了。（带“读完了”或“读完了，有点可惜/意外”的语气，根据语境可正可负）

最さい 後ご まで 食たべて しまいましょう。

把它吃到最后吧／干脆把它全吃掉吧。（带“就这样一起完成吧”的语气）

☆动/形 そうです

~そうです 用于描述说话人对某种事物样态的观感，或根据某种情形推测事态的发展。其接续方式为：

  • 动词连用形 + そうです 。
  • Aい + そうです 。
  • AN + そうです 。
この ケーキ は とても 美お 味いしそう です。【样态】

这个蛋糕看起来很好吃。

李りさん は 仕し 事ごと が 大たい 変へんそう です ね。【样态】

小李的工作看起来很辛苦呢。

雨あめ が 降ふりそう です。【推测】

看起来要下雨了。

あの 人ひと は 将しょう 来らい 偉えらく なりそう です。【推测】

那个人看起来将来会有出息／会成为厉害的人。

表示否定时

  • 前接形容词时用 ~そうではありません 的形式
  • 前接动词时则用 ~そうにありません 的形式。
あまり 美お 味いしそう では ありません。

看起来不太好吃。

雨あめ は やみそう に ありません。

雨看上去不像要停的样子

另外，前接 ない 和 いい 这两个双音节词语时，情况特殊，前者用 なさそうだ 的形式，后者用 よさそうだ 的形式。

今きょ 日う は 会かい 議ぎ が なさそう です。

今天好像没有会议。

この 本ほん は とても よさそう です。

这本书看起来很不错。

~そうです 不仅用于词尾，还用于名词、动词的前面。

  • 在名词前面时，使用 ~そうな 的形式；
  • 在动词前面时，使用 ~そうに 的形式。
おいしそうなケーキですね

这蛋糕看上去很好吃啊

子こ 供どもたち は 楽たのしそう に 遊あそんで います。

孩子们看起来很开心地玩着。

新标初_34 墙上挂着挂历

☆补助动词 てある

表示有意进行的动作结果的存续状态。在这个句型里不涉及动作的主体，而只涉及动作的对象。

壁かべ に カレンダー が 掛かけて あります。

墙上挂着日历。（强调“结果状态”，是“有人挂好后的状态”）

窓まど が 開あけて あります。

窗户被打开着。（强调“有人事先把窗户打开了”的状态）

パスポート は バッグ の 中なか に 入いれて あります。

护照已经放在包里了。（强调“事先放好、做好准备”的状态）

手て 紙がみ の 中なか に は どんな こと が 書かいて あります か。

信里写了些什么内容呢？

注意如下两个句子的对比。

  • 壁に私がカレンダーを掛けました (我把挂历挂在墙上了)
  • 壁にカレンダーが掛けてあります (墙上挂着挂历)

☆补助动词 动て + おく

置おく 放置、搁

表示为做某种准备而有意识地进行的动作，有为了某种目的而将动作的结果或效果留存下来的意思。相当于汉语的 事先 、 提前 的意思。

お客きゃくさん が 来くる 前 に、 部へ 屋や を 掃そう 除じして おきます。

在客人来之前，把房间先打扫好。

明あ 日した は 忙いそがしい です、 今夜 は ゆっくり 休やすんで おいて ください。

明天会很忙，集体那晚上你要好好休息

訪ほう 問もんする 前 に、 一度 電でん 話わして おきます。

拜访之前我准备先打个电话

☆在口语中，常常把ておく = とく

そこ に 置おいておいてください 。
そこ に 置おいといて ください。

请先放在那里。（＝置いておいてください 的口语形）

☆补助动词 てみる

見みる

表示尝试做某事。

太おお 田たさん は 中ちゅう 国ごく語 で 手て 紙がみ を 書かいて みました。

太田先生试着用汉语写了一封信

おいしそう です ね、 食たべて みます。

看上去很好吃，我来尝尝。

それ は おかしい、 もう 一いち 度ど 調しらべて みて ください。

那有点奇怪，请再查一次试试看。

☆小句1(基本形)ために

為ため = ために　目的，形式名词

~ために 表示目的，其前后两个小句的主语相同。

日に 本ほん に 留りゅう 学がくする ため に、 お金おかね を ためて います。

为了去日本留学，正在存钱。

運うん 動どう を する ため に、 新あたらしい 靴くつ を 買かいました。

为了运动，我买了新鞋。

林はやしさん は 駅えき へ 行いく ため に、 バス に 乗のりました。

林先生为了去车站，乘了公交车。

前接名词时，用 名词 + の + ために 的形式。

留りゅう 学がく の ため に、 日に 本ほん 語ご を 勉べん 強きょうして います。

为了留学，正在学习日语。

健けん 康こう の ため に、 毎まい 日にち 歩あるいて います。

为了健康，每天都在散步。

☆假定、限定 ：小句1たら，小句2

~たら 用于表示假定条件。

某种特定的、一次性的依存关系；使用范围广！

后面可以接意志、希望、命令、依赖

  • 其接续方式是把过去形式的 た 换成 たら

  • 二类形 和 名词 的过去否定形式 ~ではなかった 后续 たら 时，要去掉其中的 は 变成 ~でなかったら 。

明あ 日した 雨あめ が 降ふったら、 マラソン 大たい 会かい は 中ちゅう 止し です。

如果明天下雨，马拉松大会就中止。

寒さむかったら、 窓まど を 閉しめて ください。

如果冷的话，请把窗户关上。

表ひょう 現げん が 変へんだったら、 教おしえて ください。

如果表达（说法）奇怪的话，请告诉我。

明あ 日した 雨あめ で なかったら、 ハイキング に 行いきます。

如果明天不下雨，就去远足。

☆敬体体形式

礼貌的说法用 ~ます 、 ~です 后续 ~たら 换成 ~ましたら 、 ~でしたら 的形式。

問もん 題だい が ありましたら、 教おしえて ください。

如果有问题，请告诉我。

表ひょう 現げん が 変へんでしたら、 直なおします。

如果表达有问题的话，我会改。

☆逆接 小句1ても，小句2

相当于汉语的 尽管~，也~ 、 即使~，也~ 。

接续方式为把 て形 的 て 、 で 换成 ても 、 でも 。

日に 本ほん へ 帰かえっても、 中ちゅう 国ごく語 の 勉べん 強きょう を 続つづけて ください。

即使回到日本，也请继续学习中文。

薬くすり を 飲のんでも、 この 病びょう 気き は 治なおりません。

就算吃药，这个病也治不好。

仕し 事ごと が 忙いそがしくても、 休きゅう 日じつ に は 休やすみます。

即使工作再忙，休息日也会休息。

準じゅん 備び が 完かん 全ぜん でも、 最さい 終しゅう 点てん 検けん は 必ひつ 要よう です。

即使准备得再完美，最后的检查也是必要的。

休きゅう 日じつ でも 働はたらきます。

即使是休息日也会工作。

~ても 除了用于假定的情况外，还可以用于表示已经发生的情况。

その 言こと 葉ば は、 辞じ 書しょ で 調しらべても 分わかりませんでした。

那个词即使查了字典也没弄明白/ 那个词，查了词典也没有明白

☆名 + だけ

表示 这就是全部，不再有其他 的意思。

今こ 年とし の 夏なつ 休やすみ は 3日か だけ です。

今年的暑假只有三天。

森もりさん と 馬まさん だけ が お酒さけ を 飲のみます。

只有森先生和马先生喝酒。

わたし は その 料りょう 理り だけ を 食たべませんでした。

只有那道菜我没吃。

だけ 后面的助词 が 和 を 有时省略。

森もりさん と 馬まさん だけ お酒さけ を 飲のみます。

只有森先生和马先生喝酒。

わたし は その 料りょう 理り だけ 食たべませんでした。

只有那道菜我没吃。

☆名しか + 否定形式

语法说明：「しか～ない」

结构：名词 + しか + 动词否定形

意思：“只有……”、“除了……以外什么都不……”

表示 除了举出的一项外，其他都不是这样 的意思，和 ~だけ + 肯定形式 的意思基本相同。

两者的细微区别： ~だけ 只用来表示一种客观的限定，而 ~しか~ない 带有一定的主观感情色彩。

会かい 議ぎ 室しつ に は 李りさん しか いません。

会议室里只有小李一个人。

会かい 議ぎ 室しつ に は 李りさん だけ が います。

会议室里只有小李在（存在）。

~しか~ない 和 ~だけ 之间的差异在它们和数量词一起使用时表现得尤其明显。

下例中使用 ~だけ 表示在北京客观上只能停留 3 天，而使用 ~しか~ない 时，则有说话人主观上觉得 3 天时间太短了的语感。

  • 北ぺい 京きん に は 3日か しか 滞たい 在ざいする こと が できません。 (在北京只能停留 3 天)
  • 北ぺい 京きん に は 3日か だけ 滞たい 在ざいする こと が できます。在北京只能停留三天。

新标初_36 我迟到了，真抱歉

☆小句1て，小句2 或 小句1で，小句2

这种形式还可以用来表示原因、理由，即 小句1 是 小句2 的原因、理由。不过， 小句2 不可以是 祈使句 (或称为 命令句 )。

遅おそく なって、 すみません。

来晚了，不好意思／对不起。

最さい 初しょ は 言こと 葉ば が 通つうじなくて、 とても 困こまりました。

一开始语言不通，非常为难／非常困扰。

森もりさん は 頭あたま が 痛いたくて、 会かい 社しゃ を 休やすみました。

森先生因为头疼而请假没去公司。

説せつ 明めい が 上じょう 手ず で、 よく 分わかりました。

说明讲得很好，我完全明白了。

父ちち は 大おお 阪さか の 出しゅっ 身しん で、 いつも 関かん 西さい 弁べん で 話はなします。

我父亲是大阪人，所以平时总是说关西方言。

☆名 + に

~に 也可以用来表示用途和基准。

表示用途时，其前面是具体说明用途的名词，后面一般是 使つかう 等动词。

この 写しゃ 真しん は 何なに に 使つかいます か。

这张照片用来做什么？

パスポート の 申しん 請せい に 使つかいます。

用于申请护照。

~に 表示基准时，其前面的名词是基准，后面则一般是表示评价的形容词。

スーパー が 近ちかい ので、 この マンション は 買かい物もの に 便べん 利り です。

因为超市很近，这个公寓在购物方面很方便。

この 本ほん は、 大おとな人 には 易やさしい です。 しかし、 子こ 供ども には 難むずかしい です。

这本书对大人来说很简单，但对孩子来说很难。

☆动基本形 + のに

可以表示用途和基准，与 名词 + に 相同，其后面一般也是 使つかう 等动词和表示评价的形容词。

この 写しゃ 真しん は パスポート を 申しん 請せいする の に 使います。

这张照片是用来申请护照的。

この フライパン は 卵たまご 焼やき を 作つくる の に 使います。

这个平底锅用来做玉子烧（煎蛋卷）。

この マンション は 買かい物ものする の に 便べん 利り です。

这个公寓在购物方面很方便。

☆ばかり

名词 ＋ ばかり，表示：光是……，净是……，满是……

何なんで 野や 菜さい ばかり 食たべて いる ん です か。

为什么一直只吃蔬菜呢？

林はやしさん は カラオケ で 古ふるい 歌うた ばかり 歌うたいます。

林先生在卡拉 OK 里总是唱老歌。

動て形 ＋ ばかり ＋ いる，表示：老是、净、总是做某事（有点埋怨/否定的语气）

張ちょうさん は 毎まい 日にち お酒さけ を 飲のんで ばかり います。

小张每天就是一直喝酒。

森もりさん は いつも 失しっ 敗ぱいして ばかり います。

森先生总是在失败／老是犯错。

☆ ~ばかり 和 ~だけ

都可以表示限定，但其用法有 3 点差异：

だけ：表示限定（客观）

ばかり：表示量、次数多（主观）

  1. ~だけ 可以用来限定数字，而 ~ばかり 不能！

  2. ~だけ 可以和否定形式相呼应，而 ~ばかり 不能！

  3. ~だけ 只表示没有例外，而 ~ばかり 则含有反复进行同一动作的意思。

  4. 另外， ~ばかり 有一种 不理想 的含义。

☆主语 ＋ が ＋ 見みえる／聞きこえる

“我看到/听到『某个动作正在发生』”，或 “能看到/听到某个动作”。

空くう 港こう の 入いり口ぐち に 警けい 官かん が 立たって いる の が 見みえます。

能看到有警察站在机场入口处。

隣となり の 部へ 屋や で 子こ 供どもたち が 騒さわいで いる の が 聞きこえます か。

能听到隔壁房间孩子们在吵闹吗？

☆小句1て + 小句2 或 小句1ないで + 小句2

前后主语一致

表示进行 小句2 的动作主体处于 小句1 的状态时， 小句1 句尾使用 动词て形 或 动词ない形 + で 。

~ないで 也可以说成 ~ずに ， ~ずに 是古日语残留形式，多用于书面语。

今天没带伞就出门了。

以前我们曾经学过 ~て 的如下 3 种用法：

  1. 表示动作相继发生。
  2. 表示并列。
  3. 表示原因、理由。

在这些用法中， ~て 前后的主语可以不一致。

  • 然而，本课学习的表示动作主体状况的 ~て ，其前后的主语必须一致。
春はる が 過すぎて、 夏なつ が 来きました。（动作相继发生）

春天过去了，夏天来了。

李りさん が 来きて くれて、 わたし は うれしかった です。（原因）

小李能来，我很高兴。

（比较生硬）

接续：

  • 為ため可视为名词
  • Vる + ために
  • N + の + ために
  • Aい + ために
  • AN + な + ために

☆与 ~から 、 ~ので 、 ~て 相比， ~ために 更多地用于表示不情愿的事情的原因。

部ぶ 長ちょう に なった から、 給きゅう 料りょう が 増ふえました。

因为当上了部长，工资涨了。

部ぶ 長ちょう に なった ので、 給きゅう 料りょう が 増ふえました。

因为当上了部长，所以工资涨了。

部ぶ 長ちょう に なって、 給きゅう 料りょう が 増ふえました。

当上部长后，工资涨了。

另外， ~から 、 ~ので/んで 、 ~て 可以用来结句，而 ~ために 没有这种用法。

  • 遅かったね (你来晚了)
  1. うん，事故があったから (嗯，出事故了)
  2. うん，事故があったんで (嗯，出事故了)
  3. うん，事故があって (嗯，出事故了)

表示目的时， ~ために 前面为意志性动词的 基本形 ，而且 小句1 和 小句2 的动作主体相同。
