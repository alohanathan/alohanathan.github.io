新标初_25 这是明天会议要用的资料

☆接续小火车：动[简体形] + 名

  1. 动[简体形] + 名：使用动词小句修饰名词时，需要动词的简体形。

  2. 表示某种习惯或将要发生的动作时，用动词的基本形。

  3. 表示动作已经完成时，用动词的た形。

これは 明あ 日した 会かい 議ぎ で 使つかう 資し 料りょう です。
  • 这是明天会议要用的资料
あれ は 森もり さん が 今こん 夜や 泊とまる ホテル です。
  • 那是森先生今晚要住的酒店。
あれ は 倒とう 産さん した 会かい 社しゃ です。
  • 那是已经倒闭的公司。

☆小句(动简体) + 名は + 名/形です

动词小句修饰名词而形成的名词短语，常常用来做主语。

わたし が 明あ 日した 乗のる 飛ひ 行こう 機き は 中ちゅう 国ごく 航こう 空くう です
  • 我明天要乘坐的飞机是中国航空的（航班）。
小お 野の さん が 作つくる 料りょう 理り は おいしいです
  • 小野先生做的菜很好吃
森もり さん が 買かった 本ほん は これ です。
  • 森先生买的书是这本。

☆小句(动简体) + 名を/に/から 动ます

动词小句修饰名词而形成的名词短语，也可以用做主语以外的其他句子成分。

中ちゅう 国ごく で 買かった CD を 友とも 達だち に 貸かしました。
  • 把在中国买的 CD 借给了朋友。
私わたし が 知しらない 人ひと から 手て 紙がみ が 来きました。
  • 我收到了一个不认识的人寄来的信。

注意，修饰名词的动词小句中的主语，不用は而用が表示。

☆小句(一类形/二类形/名) + 名：名词 + が + 一类形 + です

  • Aい + 名词
  • AN + な + 名词
  • 名词 + の　+ 名词

入いり口ぐち が 広ひろい あの 建たて 物もの が わたし の 会かい 社しゃ です

  • 入口很宽的那栋建筑是我公司的大楼。
操そう 作さ が 簡かん 単たん な パソコン が 欲ほしい です。
  • 我想要一台操作简单的电脑。
数すう 学がく が 専せん 門もん の 先せん 生せい が 休やすみました。
  • 主修（专长）是数学的那位老师请假了／缺席了。

大きな和小さな：与大きい、小さい意思相同，不过大きな、小さな只能用于修饰名词，不能用于结句，其他的一类形容词没有这种情况。

あそこ に ある 大おお き な 白しろ い 建たて 物もの は 何なん です か。
  • 那边那个又大又白的建筑是什么？

☆このあたり

あたり表“附近、四周、左右（时间）”→写「辺り」。

辺あたり

汉语意思是这一附近。あたり表示某一场所的周边

東とう 京きょう 駅えき の 辺り。

—— 东京站一带。

☆の ところ

所ところ 场所

の 所ところ：可以把本来不表示场所的名词变为表示场所的名词。

あの 窓まど の ところ に いる 人ひと は だれ です か。
  • 在那扇窗边的那个人是谁？

☆でしたね

用于表示确认

新标初_26 骑自行车带人很危险

☆动词小句加の使其名词化

自転車じてんしゃに2人にんで乗のるのは危あぶないです。
  • 两个人骑一辆自行车是很危险的。
  • 自転車じてんしゃに：人物 附着点
  • 2人にんで ：主题样态
  • 「～のは」：把动词短语名词化，作为主语或主题。
  • 名词化，作为主语或主题。
  • 「危あぶない」：形容词，意思是“危险的”。
パソコンで表ひょうを作つくるのは楽たのしいです。
  • 用电脑做表格是很有趣的。
李りさんは絵えを描かくのが好すきですね。
  • 小李喜欢画画呢。

第11课学习了好きです、嫌いです、上手です、下手です等表示好恶、擅长或不擅长的形容词。如果名词化形式是这些形容词的对象时，要用助词が来表示。

私わたしは自転車じてんしゃに乗のるのが下手へたです。
  • 我不擅长骑自行车。／我骑自行车骑得不好。

注意，好きです等表示好恶的形容词的对象作为话题时，要用助词は来表示。

絵えを描かくのは好すきですか。
  • 你喜欢画画吗？
手紙てがみを出だすのを忘わすれました。
  • 我忘记寄信了。
森もりさんが発言はつげんするのを聞ききました。
  • 我听到了森先生发言。

注意，这种名词化形式做宾语时，の和こと可以互换。

李りさんは部長ぶちょうに報告ほうこくするのを忘わすれました。
  • 小李忘记向部长汇报了。
李りさんは部長ぶちょうに報告ほうこくすることを忘わすれました。

  如果句尾的动词是知らせます、伝えます、話します等表示语言行为的动词时，要用こと而不能用の。

李りさんは森もりさんが休やすむことを伝つたえました。
  • 小李转告（别人）森先生要请假了。

  但句尾动词是見ます、聞きます等表示感知的动词，或手伝います、やめます、防ぎます等表示直接作用于动作对象的动词时，则要使用の而不能用こと。

李りさんは森もりさんが車くるまから降おりるのを見みました。
  • 小李看见了森先生从车上下来了。

☆小句简体形でしょう

表示说话人对自己和听话人都不能断定的事情进行推测，常与たぶん呼应使用。

でしょう（敬体）对应的简体是： だろう

接续：

  • い形容词 + でしょう
  • N/ANだ + でしょう
  • (简体句)+ でしょう
東とう 京きょうは こんや、風かぜが 強つよいでしょう。

东京今晚风应该会很大吧。

その映えい 画がは 長ながいでしょう。

那部电影大概很长吧。

この道みちは 狭せまいでしょう。

这条路应该挺窄吧。

  • 动词普通体 + でしょう
彼かれは まもなく 到とう 着ちゃくするでしょう。

他很快就会到吧。

あしたは 雨あめが 降ふるでしょう。

明天大概会下雨吧。

値ね 段だんが 上あがるでしょう。

价格大概要涨吧。

  • 名词 + でしょう（相当于“是……吧”）
あの人ひとは 中ちゅう 国ごく 人じんでしょう。

那个人大概是中国人吧。

そこは 銀ぎん 行こうでしょう。

那里应该是银行吧。

  • な形容词 + でしょう
この町まちは 静しずかでしょう。

这座小城应该很安静吧。

その店みせは 便べん 利りでしょう。

那家店应该挺方便吧。

この店みせは 有ゆう 名めいでしょう。

这家店应该很有名吧。

~でしょう的简体形是~だろう。

☆かもしれません

  • か：终助词，本义是“吗/是否”，带不确定、推量的味道。
  • も：系助词“也”，在这里起到“再往后退一步、更含糊一些”的缓冲作用。 → か＋も = かも：把前面的判断变成“也许如此”的框。
  • 知れる：动词「知る」的自他变化系里派生出的自发/可能性的用法，意思是“会被知道、变得明白”。
  • 知れません：在「知れ(连用形)+ます」的敬体上加否定「ません」，= “不能明确”。
  • 全式：（命题）＋かも＋知れません → “也许如此，但不能肯定”。

表示有可能发生某事，其可能性一般为百分之五十左右。

接续和でしょう，一样

与~でしょう相比，~かもしれません表示的概率较低。

☆つい

表示并非有意所为，却形成了某种后果，并含有对意想不到的后果感到后悔的意思。

つい忘わすれました

一不小心就忘了。／无意中忘记了。

バーゲン 会場かいじょうでは，つい何なんでも買かいたくなります。

在特卖会场，总是不由得想买什么都买。

☆それで

表示前面句子的事态为后面句子事态的原因、理由。

日本にほんには握手あくしゅの習慣しゅうかんがありません。それで，つい握手あくしゅするのを忘わすれます。

在日本没有握手的习惯，所以我常常不自觉地忘记去握手。

今日きょうから光ひかりデパートは新春しんしゅんセールです。それで，朝あさからお客きゃくさんがたくさんいました。

从今天起，光百货公司开始新春特卖。因此，从早上开始就有很多顾客。

だから也表示原因、理由，但与それで相比，其语气较强。且后面可以是表示祈使或推测的句子。而それで则不行。

☆もしかしたら

意思是有可能，但可能性比较低。もしかしたら是口语形式，常与~かもしれません、~ではありませんか相呼应使用。

  • もしかしたら ＝ もしか ＋ したら
    • もしか ＝ もし（假设、副词“如果/万一”）＋ か（疑问/不确定语气的助词）→ 合起来产生“或许、说不定”的推量语气。
    • したら ＝ 动词 する 的过去形 した ＋ 条件助词 ら（“如果……的话”）。
  • 合起来的字面感：“如果（有那么一点）万一的话……” → 自然语义就是“也许、说不定”。
馬まさんはまだ来きませんね。

马先生还没来呢。

もしかしたら，今日きょうは来こないかもしれませんよ。

也许今天不会来了哦。／说不定他今天不会来了。

新标初_27 小时候，发生过大地震

☆相对时态 小句简体形 + 時

  • ~する(基本形) + 時 ，表示最近发生的事。
  • ~した(た形) + 時，表示以前发生的事。

一类形容词小句：

忙いそがしいとき，家族かぞくみんなで仕し 事ごとをします。

忙的时候，全家人一起工作。

二类形容词小句：

暇ひまなとき，私わたしは町まちで買かい物ものをします。

有空的时候，我会在街上购物。

名词小句：

子供こどものとき，大おおきな地震じしんがありました。

小时候发生过一次大地震。

动词小句：

日本にほんに行いったとき，たくさんお土産みやげを買かいました。

去日本的时候买了很多纪念品。

也可以使用~時に的形式。

この前，友達ともだちが病気びょうきのときに，私わたしが看病かんびょうしました。

前阵子朋友生病的时候，我照顾了他。

☆动1(连用形) + ながら + 动2

表示同一主体同时进行两个动作，其中后面的动作是主要动作。

李りさんはテレビを見みながら食事しょくじをしています。

小李一边看电视一边吃饭。

そのことを考かんがえながら歩あるいていました。

我一边想着那件事，一边走着。

☆小句简体形 + でしょう？

表示确认的~でしょう有两种用法：

  1. 确认对方比自己更为熟悉的事情，这种用法含有说话人请听话人告诉自己某种信息的语感，读升调。
  2. 用于对方和自己意见不同，或者叮问对方时，这种でしょう读降调。

升调

李りさん，明日あしたパーティーに行いくでしょう？

小李，明天你要去参加派对吧？

李りさんは優やさしくて，親切しんせつでしょう？

小李人很好吧？温柔又亲切，对吧？

降调

お母かあさん，私わたしの日記にっき，見みたでしょう。

妈妈，你看了我的日记吧？（＝你是不是偷看了我的日记！）

「～でしょう」在这里语气较强，带有责备或确认的意味。

ね也用于向对方进行确认，但是ね仅仅用于说话人认为自己和听话人双方意见一致的情况。而~でしょう则可用于对方和自己意见不一致时，所以如果用来确认对方的意见和自己的意见大体一致的事情时，有时候就会显得不礼貌。

在这种情况下，应当使用~ね。如在路上相逢双方都觉得天气热时，使用~ね而不能使用~でしょう。

☆动ている　反复、习惯

除了表示动作正在进行和动作的结果、状态的用法外，还可以表示反复或习惯性的动作。

葉子ようこさんはアルバイトをしながら学校がっこうに通かよっています。

叶子一边打工，一边上学。

北京ペキン 行いきの飛行機ひこうきは1時間じかんに1便びん 飛とんでいます。

飞往北京的飞机每小时有一班。

私わたしは毎日まいにち 散歩さんぽしています。

我每天都散步。

私わたしは毎日まいにち 散歩さんぽします。

表示习惯性动作时，~ています、~ます都可以使用，但如果句子里使用先月から这种表示时间起点的形式时，一般不用~ます形。

私わたしは先月せんげつから毎朝まいあさ 太極拳たいきょくけんをしています。

我从上个月开始，每天早上都练太极拳。

表示现在正在从事的工作，有时可用[表示职业的名词] + を + しています的形式。

私わたしは高校こうこうの教師きょうしをしています。

我是一名高中老师。／我在高中教书。

☆名+で

で加在名词后面还可以表示原因。

仕事しごとで，楊ようさんと会あっていたんですよ。

是因为工作，我刚才在见杨先生啊。／我那时候是在和杨先生谈工作的。

☆と会あいます

第8课学习了~に会います中的に表示对象的用法，而使用表示互动行为的动词时，进行同一动作的另一方要用と来表示。

但用と时含有互动的语感。

仕事しごとで，楊ようさんと会あっていたんですよ。

是因为工作啊，我那时候正在见杨先生。／我是在工作上见的杨先生哦。

☆そう言いえば

由正在进行的对话或说话现场的某种情况，联想到另外一些情况时使用。

常用于：

  1. 想起某件事：

    表示“说起来我突然想起了……”。

    → 对应中文的“对了”、“对啊”、“这么说来”。

  2. 转换话题：

    在谈话中自然地引出另一个话题。

    → 类似中文“话说回来”、“对了说到这件事”。

新标初_28 小马给了我一张地图

☆授受动词

  • あげる

    • 我给别人；我→他（施慧；不用于目上）

    • 客观描述

    • 森もりさんは小野おのさんに花はなを上あげました。
      • 森先生送了花给小野女士。
    • 森もりさんは小野おのさんに花はなを買かって上あげました。
      • 森先生给小野女士买了花（表示出于关心或体贴）。

  • くれる

    • 别人给我；他→我（受惠）

    • 李りさんは母ははに花はなをくれました。
      • 小李送了花给我母亲

    • 李りさんは母ははに花はなを買かってくれました。

      • 小李买了花送给我母亲。

  • もらう

    • 我请别人给我做；他→我（受惠；更委婉、常用）

    • 客观描述

    • 母ははは李りさんに花はなをもらいました。
      • 我妈妈收到了小李送的花。

    • 母ははは李りさんに花はなを買かってもらいました。

      • 我妈妈请小李帮她买了花。／小李帮我妈妈买了花。

☆それに

用于追加、补充同等事物，把形式上不独立的前一个分句和后一个分句连接起来，常用于较为随便的口语中。

陳ちんさんに不動産屋ふどうさんやさんを紹介しょうかいしてもらいました。
それに，李りさんや馬まさんもいろいろと探さがしてくれています。

请陈先生帮我介绍了房产中介。 而且，小李和小马也一直在帮我到处找房子。

☆どうしたんですか

用在不同的场合表达不同的意思，相当于汉语的怎么了、干了什么、怎么处理了，要根据具体的情况来判断。

家具かぐはどうしたんですか。

家具怎么办了？／家具后来怎么样了？

☆~先さき

表示移动性动作的到达地点或归属。

☆辺へん

询问大致的场所，可使用どの辺。可以回答一个大致的场所，也可以回答一个地名。

新住的地方已经决定了。／搬去的地方已经确定了。

比动词的命令形稍微客气一些，多用于老师对学生或者父母对孩子提出要求。

☆动基本形 + な

用于禁止听话人做某事。

不要在这儿停车！

タバコを吸すう な。

不要吸烟！

☆动て(ください)、动ない で(ください)

日常生活中经常使用的表示祈使的说法，不论男性女性都经常使用，但一般用于关系亲密的人之间。

もうちょっと 急いそいで。

再稍微快一点！

ちゃんと 話はなしを 聞きいて。

好好地听我说话！

遠慮えんりょしないで。

别客气！

命令形、~なさい、~な和~て/ないで相比，使用的范围很有限。

三者一般限于上下、尊卑关系非常明确的非正式场合。

不过，在紧急情况下，可以对尊长说逃げろ。另外，虽然是紧急情况，但要求对方、帮助自己时也不能用命令形，而用动て形，比如助けて。

☆名1 + という + 名2

と言いう 本质就是“说～”“叫做～”“所谓的～”。

提供对方不知道的新信息时，在表示新信息的词语后面加：と言いう。

最基础用法：引用“说”

彼は「行いきたくない」と言いました。

他说：“不想去。”

N１というN２ = “叫做N1的N2 / 名叫N1的N2”

田た 中なかさんという人ひとを知しっていますか。

你认识一个叫田中的人吗？

それはフジという 花はなです。

那是一种叫做紫藤的花

わたしは東京とうきょうの世田谷せたがやという所ところに住すんでいます。

我住在东京一个叫做世田谷的地方

田中たなかという名前なまえは、日本人にほんじんの名前なまえです。

田中这个姓是日本人的姓

☆名1は 名/小句 + という + 名2です

という也可以用于给名1下定义，或对名1做解释，这时候名2多为意味等词语。

这里的小句可以是命令形、简体形或简体形 + な。

このマークは、タバコを吸すうなという意味いみです。

这个符号是禁止吸烟的意思

このマークは ここで止とまれという意い 味みです。

这个符号是在这儿停下来的意思

この標識ひょうしきは、駐車禁止ちゅうしゃきんしという意味いみです。

这个标识是禁止停车的意思

☆~ないといけません

~ないといけません可以省略いけません，~なければなりません可以省略なりません。

  • この場合は歌ってくださいと言わないとね(这种场合要说歌ってください)
  • 時間がないから，急がないと~(没时间了，不快一点就~)
  • 日本語のテレビ講座が9時から始まります。見なければ(日语的电视讲座从9点开始，得看)

表示使用某种方式进行动作。

あの人ひとは いつも 変かわった 考かんがえ 方かたをします。

那个人总是有奇怪的想法。／那个人的思考方式总是与众不同。

☆意志形

动词的 意志形 ：动词的意志形是说话人当场表示自己的决心、意志的表达形式。

  • 表示提议（两个人） =　ましょう的简体
    • ましょう的简体 就是动词意向形
  • 表示决心（一个人）

☆变化规则

  • 一类动词：把基本形最后一个音变成相应的 お 段上的音+ う。
基本形读音意志形

急ぐいそぐいそごう
飛ぶとぶとぼう

買うかうかおう

  • 二类动词：把基本形的 る 变成 よう 。

与其说是“动词变形”，不如说：

二类动词：词干 + 一个表示意志的东西「よう」

也就是：

  • 「食べ」＋「よう」
  • 「見」＋「よう」

你可以把「よう」当成一种语法小尾巴（助动词），专门表示：

  • 说话人的打算 / 决心：
    • 今年から貯金ちょきんしよう。
    • 今年开始存钱吧 / 要开始存钱了。

☆相关句型 1：と思おもう

意义：表示决心；“要”

接续：意向形 + と思おもう

今きょ 日う、 会かい 社しゃを 休やすもうと 思おもいます。

我今天不想上班

もう 遅おそいので、そろそろ帰かえろうと思おもいます。

已经很晚了，我该回家了

☆と思おもっている

表示自己已把某种意志持续了一段时间。

明あ 日した 病びょう 院いんへ 行いこうと 思おもっています。

我正在想明天去医院。 （带“已经在考虑／有这个打算一段时间了”的语气。）

今こん 度ど、 長ちょう 江こう 下くだり を しようと 思おもっています。

我想下次坐船游览长江两岸的风光

その 問もん 題だい について、 もう 少し 考かんがえよう と 思おもっています。

关于那个问题我想再考虑考虑

意志形 、 ~ようと思います 、 ~ようと思っています 三者的区别如下：

  1. 表示当场形成的意志时使用 意志形 ，而不能使用 ~ようと思います 和 ~ようと思っています 。
  2. ~ようと思います 和 ~ようと思っています 都表示决心或意志。但由于后者用 ~ています 的形式，所以有一种决心或意志尚未最后形成的语感。

☆小句1ので，小句2

~ので 表示原因、理由，既可以接在简体形后，也可以接在敬体形后，但接简体形较多。

小句1 为二类形容词小句和名词小句时，把其简体形的 だ 变成 な 再加 ので ，成 な + ので 的形式。

荷にも 物つ が 重おもい ので、 宅たく 配はい 便びん で 送おくります。

行李很重，所以用送货上门的方式送达

香ほん 港こん から お客きゃくさん が 来くる ので、 空くう 港こう へ 迎むかえ に 行いきます。

因为有客人从香港过来，所以我要去机场接他们。

ここ は静しずか なので、とても気き に入いっています。

这里很安静，所以我非常喜欢这里。

もう 終しゅう 電でん の 時じ 間かん なので、 帰かえります。

已经是末班电车的时间了，所以我要回去了。

~ので 和 ~から 相比，较为礼貌郑重，所以多用于正式场合。 ~ので 口语中往往发成 ~んで 。

小句2 表示提议时，必须使用 から 。

電でん 車しゃ が 遅おくれた んで、 遅ち 刻こくしました。 すみません。

电车晚点了，所以迟到了。不好意思。

もう時じ 間かん が ない から、早く行いこうよ。

已经没时间了，快走吧。

☆まあ

表示不是十分理想，但能过得去的程度，用于调控自己或对方的不满情绪。

甲： 報ほう 告こく 書しょ ができました。これ で いい です か。

报告做好了。这样可以吗？

乙： 時じ 間かん が ない から、まあ、いい でしょう。

因为没时间了嘛，就这样吧。

（表达一种“算了、够了、差不多了”的宽松判断。）

☆そろそろ

用于表示做某事或某事态发生的时间渐渐迫近。

そろそろ 食しょく 事じ の 時じ 間かん ですよ。

差不多到吃饭时间了哦。

そろそろ梅つ 雨ゆ が 始はじまります ね。

梅雨季差不多要开始了呢。

☆条件：基本形、ない形 + と，小句2

~と 用在表述恒常性状态、真理、反复性状态、习惯等内容的复句里，表示 小句1 是 小句2 的条件。

本质是表达两件事之间的继起关系！

この ボタン を 押おす と、 電でん 源げん が 入はいります。

按下这个按钮就会通电。

食たべもの を 食たべない と、 人にん 間げん は 生いきる こと が できません。

不吃食物的话，人就无法生存。

わたし は、その 店みせ へ行いく と、いつも コーヒー を飲のみます。

我一去那家店，就总是喝咖啡。

~と 还可以表示由于某种行为而发现了新的情况。

この 道みち をまっすぐ行いく と、デパート があります。

沿着这条路一直走，就能看到百货公司。

カーテン を 開あける と、 美うつくしい 山やま が 見みえました。

把窗帘拉开时，看到了美丽的山景。

用 ~と 的复句， 小句2 不能是自己的意志、愿望或向听话人提出要求、劝诱等内容。

夜よる に なる と、電でん 気き がつきます。

到了晚上灯就亮了

☆动(基本形/ない形) ことがある

表示有时会发生某种事态，往往和 たまに 、 時々 等副词呼应使用。

那台电脑偶尔会死机

時とき 々どき 寝ね 坊ぼう する こと が あります。

たまに 朝あさご飯はん を 食たべない こと が あります。

偶尔也会不吃早饭。

☆形容词的 副词性 用法

形容词修饰动词

  • 一类形容词要把词尾的 い 变成 く
  • 二类形容词后面要加 に
一类形容词副词
早い早く
大きい大きく
  • 早く家に帰りましょう (早点儿回家吧)
二类形容词副词
上手上手に
静か静かに

小马出色的整理了报告

静しずか に ドア を 閉しめて ください。

请轻轻地关门。

☆小句简体形 でしょうか

句尾用 でしょうか 来表示疑问的句子，是礼貌程度较高的表达方式。比如 来るでしょうか 与 来ますか 意思相同，但是 来るでしょうか 的礼貌程度比 来ますか 高一些。

接续：和です一样

  • 动词小句和一类形容词小句以简体形后续 でしょうか
  • 二类形容词小句和名词小句则把其简体形的 だ 换成 でしょうか
李りさん は 来くる でしょう か。

小李会来吗？

馬まさん は もう 帰かえった でしょう か。

小马已经回家了吗？

3月がつ の 東とう 京きょう は 寒さむい でしょう か。

三月的东京会冷吗？

これ は 中ちゅう 国ごく の お菓か子 でしょう か。

这是中国的点心吗？

☆打算： 动(基本形/ない形) つもりです

表示说话之前已经形成的意志、打算，和 动词意志形 + と思っています 意思相同。

今こん 度ど の 日にち 曜よう 日び に 遊ゆう 園えん 地ち へ 行いく つもり です。

打算这个星期天去游乐园。

ボーナス で 車くるま を 買かう つもり です。

打算用奖金买车。

もう タバコ は 吸すわない つもり です。

我打算不再抽烟了。

对长辈或者上级用 ~つもりですか 直接提问不太礼貌，需要注意。

☆动(基本形/ない形) ことにします/ことにしました

表示说话人自己决定实施某种行为。

明あ 日した から 毎まい 日にち 運うん 動どう する こと に します。

我从明天起每天做运动

今きょ 日う から お酒さけ を 飲のまない こと に します。

我决定从今天起戒酒

明あ 日した、 友とも 達だち と 映えい 画が を 見みに 行いく こと に しました。

我决定明天和朋友去看电影。

今こ 年とし の 夏なつ は 旅りょ 行こう に 行いかない こと に しました。

我决定今年夏天不去旅行了

☆动(基本形/ない形) ことになる

表示由于某种外在的原因导致形成了某种决定。

来らい 月げつ から 給きゅう 料りょう が 上あがる こと に なりました。

从下个月开始，工资决定要上涨了。（表示“外部决定、非自己决定”）

月げつ 曜よう 日び から 3日か 間かん、 香ほん 港こん へ 出しゅっ 張ちょう する こと に なりました。

从星期一开始，要去香港出差三天。（表示公司或外部已经决定好的行程）

  • ~ことにします 、 ~ことになります 中的 こと 不能用 の 来代替。
  • ~ことになりました 也可用于委婉的表述根据自己的意志决定的事情。
わたしたち、 結けっ 婚こん する こと に なりました。

我们决定要结婚了。（多表示“顺其自然／双方达成共识的结果”）

☆【传闻】小句简体形 そうです

用 ~そうです 明确表示某信息不是源于自己的直接感知，而是从别人那里听到的，提示消息来源时则用 ~によると 。

馬まさん の 息むす 子こさん は、 今こ 年とし 小しょう 学がっ 校こう に 入にゅう 学がく する そう です。

听说明马先生的儿子今年要上小学了。

天てん 気き 予よ 報ほう に よる と、 明あ 日した は 雨あめ だ そう です。

根据天气预报，听说明天会下雨。

うわさ に よる と、 あの 店みせ の ラーメン は おいしくない そう です。

听传闻说，那家店的拉面不好吃。

  • ~そうです 没有过去、否定和疑问的形式。

☆ずっと

其表示在某一期限内动作、状态一直持续的用法。

天てん 気き 予よ 報ほう に よる と、 連れん 休きゅう 中ちゅう ずっと 晴はれ だ そう です よ。

根据天气预报，听说明整个连休期间都会是晴天哦。

佐さ 藤とうさん は 先せん 月げつ から ずっと 入にゅう 院いんして います。

佐藤先生上个月起就一直住在医院

☆动てほしい

表示希望别人做某事，礼貌程度较低，对长辈不要直接使用。

先せん 生せい に わたし の 友とも 達だち を 紹しょう 介かいしたい です。

我想把我的朋友介绍给老师。

田た 中なかさんに 日に 本ほん 人じんの 友とも 達だちを 紹しょう 介かいしてほしいです。
