第9课 四川菜很辣

形容词

一类い形容词： 词干 + い

☆一类形容词做谓语

形1一般现在、将来过去
肯定~いです~かったです
否定~くないです~くなかったです
~くありません~くありませんでした

  • いいです

  • よくないです

  • よかったです

  • よくなかったです

☆名は/が 形容词です

日に 本ほん 料りょう 理りは おいしいです。
单词词性・中文释义
日に 本ほん 料りょう 理り⑤名词｜日本料理
は⓪助词｜提示主题
おいしい③形容词｜好吃、美味
です⓪系词｜礼貌判断

☆一类形容词修饰名词

文型：~い + 名

広ひろい 国くに
单词词性・中文释义
広ひろい②形容词｜宽广、辽阔｜辞书形同形：広ひろい②
国くに⓪名词｜国家、国度
今きょ 日うは いい 天てん 気きですね。
单词词性・中文释义
今きょ 日う①名词/副词｜今天
いい①形容词｜好、不错
天てん 気き①名词｜天气
は⓪助词｜提示主题
です⓪系词｜礼貌判断
ね⓪终助词｜确认、共感（“…呢/对吧”）

☆不完全否定

文型：あまり + 否定

意思：不太……；不是很……

このスープは あまり熱あつくないです。

意思：这碗汤不太烫。

单词词性・释义
この⓪连体词｜这…
スープ①名词｜汤
は⓪助词｜主题提示
あまり③副词｜不太…（常与否定连用）
熱あつい② → 熱あつくない形容词｜烫、热（物体/食物的温度）→ 否定形
です⓪系词｜礼貌判断

第10课 京东的红叶很有名

二类な形容词： 词干 + だ

二类な形容词・形容动词

形1一般现在、将来过去
肯定~です~でした
否定~ではありません~ではありませんでした

☆二类形容词做谓语

文型：名 +は/が + 形容词 +です

桜さくらが きれいです。
  • 樱花真漂亮。
单词词性・中文释义
桜さくら⓪名词｜樱花
きれい①形容动词｜漂亮、干净（辞书形同形：きれい①）

☆二类形容词修饰名词

文型：形2 +　な　+ 名词

田た 中なかさんが　嫌きらいな　料りょう 理りは　なんですか。
  • 田中先生不喜欢的料理是什么？
单词词性・中文释义
田た 中なか⓪专有名词｜田中
さん⓪敬称｜先生、女士
が⓪助词｜主语标记
嫌きらいな形容动词用法｜讨厌的｜辞书形：嫌きらい⓪
料りょう 理り①名词｜料理、菜肴
なん①疑问词｜什么（「なに」① 的连用变体）

☆どんな+名

解析：用来询问人或者事务的性质

意思：怎么样的？

東とう 京きょうはどんな 町まちですか。
  • 东京是个怎样的城市？
单词词性・中文释义
東とう 京きょう专有名词｜东京
どんな连体词｜怎样的、什么样的
町まち名词｜城镇、城市

☆~はどうですか

~はどうですか / でしたか

~はいかがですか/ でしたか

解析：用来询问某人对某状态的意见或感想

意思：……怎么样？

A： 昨きの 日うの フランス料りょう 理りは どうでしたか。
B： とても おいしかったです。
  • A：昨天的法餐怎么样？
  • B：非常好吃。没给你人均刺客，算你走运。
单词词性・中文释义备注（形变化）
昨きの 日う名词/副词｜昨天读「きのう」
フランス料りょう 理り名词｜法餐、法国料理片假名部分不加 ruby
は助词｜提示主题念「わ」别念“は”
どう副词/疑问词｜怎么样询问评价
でした系词过去礼貌｜是…（过去）です 的过去形
とても副词｜很、非常和形容词搭配
おいしかったですい形容词过去礼貌｜很好吃原形：おいしい

☆~はどうですか

※有时也可以用于诱导对方做某事

A：お茶ちゃは どうですか。
B：あっ、すみません。
  • A：要不要来点茶呀？
  • B：啊，不好意思。（婉拒/致歉两可，看场合）
单词词性・中文释义
お茶ちゃ⓪名词｜茶
どう①副词｜怎么样、如何

第11课 小野女士喜欢唱歌

☆[が]表示对象

解析：

  1. 好恶对象
    • すき、きらい
子こ 供どもは　野や 菜さいが　嫌きらいです。
  • 小孩子不爱吃蔬菜。
单词词性・中文释义
子こ 供ども⓪名词｜孩子
野や 菜さい⓪名词｜蔬菜
は⓪助词｜提示主题
が⓪助词｜主语/对象标记（喜好句型用）
嫌きらい⓪形容动词｜讨厌、不喜欢（辞书形同形：嫌きらい⓪）
  1. 擅长与否的对象
    • 上じょう 手ず
    • 下へ 手た
    • 苦にが 手て
田た 中なかさんは中ちゅう 国ごく 語ごが上じょう 手ずです。
  • 田中先生中文很棒。
单词词性・中文释义
田た 中なか⓪专有名词｜田中（姓）
さん⓪敬称｜先生/女士
は⓪助词｜提示主题
中ちゅう 国ごく 語ご⓪名词｜中文、汉语
が⓪助词｜主语/对象标记（能力・好恶句型用）
上じょう 手ず③形容动词｜擅长、拿手
です⓪系词｜礼貌判断
  1. 能力的对象（できる）
小お 野のさんは運うん 転てんができます。
  • 小野会开车。
单词词性・中文释义
は⓪助词｜提示主题
運うん 転てん⓪名词｜驾驶；开车
が⓪助词｜能力・可否句型用的主语标记
できます③ → 辞书形：できる②动Ⅱ｜会、能、可以
  1. 明白懂得的对象（わかる）

中ちゅう 国ごく 語ごがわかりません。

  • 听不懂/不懂中文。

☆[や]不完全列举

文型：名 や 名

解析：表示不完全列举， 常与など连用

意思：……呀……（之类）

吉よし 田ださんはときどき中ちゅう 国ごくや韓かん 国こくへ行いきます。
  • 吉田先生有时去中国和韩国。
店みせで野や 菜さいや果くだ 物ものなどを買かいました。
  • 在店里买了蔬菜、水果等等。

☆句子 + から

解析：表示原因

意思：因为……所以……

森もりさんはお酒さけが好すき②ですから、毎まい 日にち 飲のみます。

  • 因为森先生喜欢喝酒，所以他每天都喝。

第12课 小李比森先生更年轻

☆比较句1

文型：名1 は　名2　より 形

解析：表示名1比名2更具有该形容词的性质

意思：……比……

中ちゅう 国ごくは日に 本ほんより広ひろいです。

☆比较句2

文型：名1より 名2　のほうが 形

解析：表示与名1相比，名2更具有该形容词的性质

意思：与……相比，还是……更……

京きょう 都とより東とう 京きょうのほうがにぎやか②です。
  • 和京都比，东京更热闹。
单词词性・中文释义
より⓪助词｜比较标记“比…”
ほう①名词｜一方、这边（比较侧）

☆比较句3

文型：名1　は 名2　ほど　 形（否定）

解析：表示与名1具有的形容词的性质不及名2

意思：……没有……那么……

京きょう 都と は東とう 京きょう ほどにぎやか② ではありません。
  • 京都没有东京那么热闹。
单词词性・中文释义
ほど⓪副助词｜程度比较的基准“像…那样、那么…”

☆比较句4

文型：名1 の中なかで 名2が　一番いちばん 形　です。

解析：表示在某个范围内，名2最具有该形容词的性质。

意思：

スポーツの 中なかで　サッカーが　いちばん　面おも 白しろいです。

在运动里，足球最有趣的。

在时间、场所の中なかで 中の中なか可以省略

一いち 年ねんで　何なん 月がつが　いちばん　寒さむいですか。

一年当中，几月最冷？

☆比较句5

文型：名1 と名2と　どちらが　 形　ですか

解析：询问两种事物中，哪个更具有形容词性质。

吉よし 田ださんと　森もりさんと　どちらが　背せ①が　高たかいですか。
  • 吉田和森，谁个子更高？

吉よし 田ださんより　森もりさんの ほうが　背せが　高たかいです。

  • 跟吉田相比，森更高。

☆多项事物比较最具形容性质

どの名/いつ/どれ/だれ/何　が一番いちばん　形 ですか

解析：

询问三个以上事物中哪个更具有形容词性质。

询问的事物在眼前或见于上下文时可用

どれが、どの名が。反之用：何が

果くだ 物ものの中なかで、何なにがいちばん好すきですか。
  • 在水果里，你最喜欢哪种？

第13课 桌子上有3本书

☆名（数量）+ 动

本ほんを　二に 冊さつ　買かいました。
  • 买了两本书。
ビールを　三さん 杯ばい　飲のみました。
单词词性・中文释义
ビール名｜啤酒
三さん 杯ばい助数｜三杯（杯数用「～杯はい」→ いっぱい／にはい／さんばい…）
を助词｜宾语标记
飲のむ动1｜喝、饮（例句：飲のみました＝过去敬体）

～杯会连浊和促音，1 杯＝いっぱい，3 杯＝さんばい，8 杯＝はっぱい。

ガレージに　車くるまが　五ご 台だい　あります。
  • 车库里有五辆车。
单词词性・中文释义
ガレージ名｜车库（外来语 garage）
車くるま名｜车、车辆
五ご台だい助数｜五台、五辆（机器车辆的量词「～台」）
に助词｜地点格，在…
が助词｜主语标记
あります动词（敬体）｜有、在（无意志物）｜辞书形：有ある

☆名（时间数量）+ 动

解析：表示动作，状态进行的时长。

李りさんは　毎日まいにち　7時間じかん　働はたらきます。
  • 李先生每天工作七个小时。

☆名（时间段）+ に + 名（次数） + 动

解析：表示时间段内进行的频率

李りさんは　1週間いっしゅうかんに　2回にかい　プールへ　行いきます。
  • 李先生每一周去两次游泳池。
单词词性・中文释义
1週間いっしゅうかん名｜一周的时长
2回にかい名｜两次（回数助数词）
プール名｜游泳池
へ助词｜方向、目的地
に助词｜时间/频率标记，这里是“每一周”
は助词｜主题提示
行いきます动1（敬体）｜去；辞书形：行いく

☆名（数量）+ で

解析：表示总共的数量。数量是1时，不用加で

このケーキは　3さん 個こで　500ごひゃく 円えんです。
  • 这款蛋糕三个卖500日元。

☆に移动行为的目的

接续：名（场所）へ 动/名　に　行く/来る/帰り

午後ごご　郵便局ゆうびんきょくへ　荷物にもつを　出だしに　行いきます。
  • 下午去邮局寄包裹。
小野おのさんは　プールへ　泳およぎに　行いきます。
  • 小野去游泳池游泳。

第14课 昨天去商场买东西

☆动词的て形

  1. 动词中顿
  2. 动作按照先后顺序进行
  3. 前项动词时后项动作进行的方式

☆动词て形变形规则

一类动词：

  • 词尾 う、つ、る, 促音变 -> って
会あう会あって动1｜见面，见到；会见
持もつ持もって动1｜拿，持有；拥有
終おわる終おわって动1｜结束，终了；告一段落
  • 词尾 む、ぶ、ぬ, 拨音变 -> んで
飲のむ飲のんで动1｜喝；服（药）；吞咽
呼よぶ呼よんで动1｜呼喊；称作；叫来
死しぬ死しんで动1｜死亡；没命
  • 词尾 く、ぐ、い音变 -> いて、いで
行いく行いって动1｜去
書かく書かいて动1｜写，书写
急いそぐ急いそいで动1｜着急，赶紧，赶忙
  • 词尾 す　音变 -> して
出だす出だして动1｜拿出；掏出；寄出/发出（信、声、命令）；提交；上菜
消けす消けして动1｜关掉（灯、电器）；擦掉、消除；熄灭（火）

五段动词 → て形／た形一览

词尾て形た形记忆点
う・つ・るってった促音变
む・ぶ・ぬんでんだ拔音变
くいていたい音变
ぐいでいだ浊音配合
すしてした单独门派

二类动词：

  • 去 る　加　て
見みる見みて看、观看
食たべる食たべて吃
寝ねる寝ねて睡觉

三类动词：

  • する　->　して
  • くる　->　きて
  • 散歩する　->　散歩して

☆动作相继发生1

文型：动て、动

朝あさご飯はんを　食たべて、　学がっ 校こうへ　行いきました。
  • 吃过早饭后，去了学校。
シャワーを 浴あびて、 寝ねます。
  • 洗完澡就睡觉。
单词词性・中文释义
シャワー①名｜淋浴、花洒
浴あびて（原形：浴あびる⓪）动2｜洗（淋浴）、浇；流到身上
寝ねます（原形：寝ねる①）动2｜睡觉
を助词｜宾语标记
て接续｜动词て形，连接后续动作

☆动作相继发生2

文型：动 てから、动

解析：表示两个以上的动作依照时间顺序相继发生。

意思：……之后，……

昼ひるご飯はんを 食たべてから、 出でかけます。
  • 吃完午饭之后就出门。
单词词性・中文释义
昼ひるご飯④名｜午饭
食たべて（原形：食たべる②）动2｜吃
から接续｜Vてから：在……之后
出でかけます（原形：出でかける④）动2｜出门，外出
を助词｜宾语标记

☆动てください

解析：表示请求对方完成某动作。

意思：请……

お茶ちゃを 飲のんでください。
  • 请喝茶。

☆助词：を

文型：名（场所）を 动

解析：表示经过、离开、移动的场所。

常用的动词有：

  • 通とおす
  • 渡わたる
  • 過すぎる
  • 出でる
单词词性・中文释义常见搭配/语法补充相关他/自动词对应
通とおす①他动1｜让通过；贯穿；贯通；贯彻ドアを通す 让…通过门；意見を通す 坚持意见自动：通とおる①
渡わたる⓪自动1｜渡过；经过；生活度日橋を渡る 过桥；季節が渡る 时序推移他动：渡わたす⓪
過すぎる②自动2｜过；超过；太过（接在连用形后作补助动词“过于…”）食べ過ぎる 吃太多；三時を過ぎる 过了三点他动：過すごす②（度过）
出でる①自动2｜出去；出现；离开；参加；被发表家を出る 离开家；ニュースに出る 上新闻他动：出だす①（拿出、发出）
バスは 学がっ校こう の 前まえ を 通とおります。
  • 公交车从学校前面经过。

第15课 小野女士现在正在看报纸

☆动词正在进行

文型：动ている

解析：表示动作正在进行

意思：正在……

いる 是简体， 变成礼貌体就是います

鈴すず木きさんは テレビを 見みています。
  • 铃木先生/同学在看电视。
单词词性・中文释义
鈴すず木き⓪名｜铃木（姓氏）
見みています动2｜在看（原形：見みる①）
二人ふたりは コーヒーを 飲のんでいます。

☆助词に

文型：名（附着点） に　动

解析：表示人或者物体的附着点

二人ふたりは　公園こうえんで　ボートボートに　乗のります。
  • 两个人在公园里坐小船（划船）。
床ゆかに　座すわります。
  • （我/他）坐在地板上（榻榻米上）。

语法要点

場所＋に：表示动作的着点/所在位置。与“座る・立つ・住む”等动词搭配，用に。

ます形：礼貌体现在发生的动作。

词汇读音含义原形
床ゆか⓪名地板、榻榻米—
座りますすわります⓪动1坐座る⓪

  • 把报纸放在桌子上。
词汇读音含义原形
新聞しんぶん⓪报纸—
机つくえ⓪桌子、书桌—
上うえ⓪上面、上方—
置きますおきます⓪动1放、摆、搁置く⓪

☆助词に

文型：名（场所）に 动

解析：表示目的地

病院びょういんに　行いきます。

☆形容词的并列、中顿形式

形1： い　->　くて

形2：词干 -> 词干で

解析：与二类形容词一样，名で

山本やまもとさんは　旅行社りょこうしゃの　社員しゃいんです。
山本やまもとさんは　中国人ちゅうごくじんで、旅行社りょこうしゃの　社員しゃいんです。

☆动ている

解析：表示动作结果持续

弟おとうとが　結婚けっこんしています。
  • 名が …ています：动词ている形表示状态的持续或习惯。
  • 我弟弟已经结婚了。（处于已婚状态）
電気でんきが　ついていません。
  • 灯没亮。
  • つける（付ける）②：打开、点（灯）（他动词，原形)
東京とうきょうに　住すんでいます。
  • N（場所）に 住んでいる：表示长期居住的状态。「〜ている」在这里是结果状态（住下来了 → 正住着）。
  • 我住在东京。
  • 住む（すむ）①：居住

☆小句が、小句

解析：

  1. 顺接
  2. 逆接。虽然……但是……
すみませんが、ちょっといいですか。
  • 不好意思，可以打扰您一下吗？
失礼しつれいですが、おいくつですか。
  • 失礼了，请问您多大岁数？
ちょっと聞ききたいんですが、これ、どこどこにありますか。
  • 我想打听一下，这个在哪里呢？
日本語にほんごが難むずかしいですが、面白おもしろいです。
  • 日语很难，但是很有趣。
雪ゆきが降ふりましたが、そんなに寒さむくないです。

  • 下雪了，不过并不那么冷。

☆まだ　+ 否

解析：表示还未到达某种程度

意思：还未（不）

A：何年なんねんぐらい　日本語にほんごを勉強べんきょうしましたか。
  • 你日语学了大约几年？
B：２年間にねんかん 勉強べんきょうしましたが、まだ出来できません。
  • 学了两年，但还不会（还不太行）。
A：小野おのさんは もう 結婚けっこんしていますか。
  • 小野已经结婚了吗？
B：いいえ、 まだ 結婚けっこんしていません。
  • 不，还没有结婚。

语法要点

  • もう＋Vている？ 询问“已经……了吗”。
  • Vている 表示状态（结婚后的已婚状态）或进行。
  • まだ＋Vていません “还没有……”。礼貌否定现在时。
