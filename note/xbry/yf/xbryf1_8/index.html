第1课 小李是中国人

☆~は~です。

接续：名1は名2です

意思：……是……

は：主格助词，提示主语主题

です：相当于汉语的“是”

☆~は~ではありません。

接续：名1は名2ではありません。

意思：……不是……

☆~は~ですか。

一般疑问句

接续：名1は名2ですか。

意思：……是……吗？

  • はい
  • はい、そうです。
  • いいえ、
  • いいえ、ちがいますす。
  • すみません、わかりません。

☆の

接续：名1 の 名2

解析：表示名1是名2从属的机构、国家、或属性

接续：名1 は 名2（场所）です。

意思：~在~（地方，场所）

意思：~也是~

意思：~是~，还是~？

第4课 房间里有桌子和椅子

☆存在句

名（场所）に 名 があります/います。

名は名（场所）にあります/います。

  • 存在句 / ～は～です
王おうさんは庭にわにいますか。
单词词性・中文释义
庭にわ ⓪名词｜庭院，院子
います ②动Ⅱ｜在，存在（具意志者）｜原形：居いる ①

☆并列句

文型：名と名

中ちゅう 国ごく と 日に 本ほん
時と 計けい と 眼め 鏡がね
单词词性・中文释义
中ちゅう 国ごく名词｜中国
日に 本ほん名词｜日本
時と 計けい名词｜钟、表
眼め 鏡がね名词｜眼镜
と助词｜和、与（连接名词与名词）

☆全面否定

疑问词 + も + 动（否）

教きょう 室しつに誰だれもいません。
单词词性・中文释义
教きょう 室しつ名词｜教室
誰だれ代词｜谁
に助词｜表示存在地点（在……）
も助词｜和否定搭配成「誰も〜ない」表示“谁也不……”
いません动Ⅱ｜不在（礼貌否定形）｜原形：居いる

☆名＋の＋方位词

うえ／した／まえ／うしろ

となり／なか／そと

本ほんは机つくえの上うえにあります。
かばんは いすの下したにあります。
図と 書しょ 館かんは銀ぎん 行こうの隣となりです。
ドアの外そとに人ひとがいます。
单词词性・中文释义
上うえ名词・方位词｜上，上面
下した名词・方位词｜下，下面
前まえ名词・方位词｜前，前面
後うし ろ名词・方位词｜后，后面
隣となり名词・方位词｜旁边，隔壁
中なか名词・方位词｜中，里面，中间
外そと名词・方位词｜外，外面
の助词｜连体（所属/限定）：A の B（A 的 B）
に助词｜地点/存在/到达点

第5课 森先生7点起床

☆动词

  • 一类动词
    • 词尾不是る一定是一类动词
    • 词尾是る，る前不是い段或え段的假名
  • 二类动词
    • 词尾是る，る前是い段或え段的假名
  • 三类动词
    • する
    • 来くる
    • 〜する

☆动词礼貌体

一般现在、将来过去
肯定ますました
否定ませんませんでした
  • 一类动词
    • 词尾变成该行い段假名 + ます
  • 二类动词
    • 词尾去る + ます
  • 三类动词
    • する 变成 します
    • 来くる　变成 来きます
    • 〜する 变成 〜します

☆に（精确时间）

接续：名（时间） + に + 动

解析：表示动作进行的具体时间。（含有数字的时间）

注意：星期后に可有可无

  • 何時+に+动词

    • 出现了动词，需要加に

  • 何時+ですか。

毎まい 日にち 7しち 時じに起おきます。
单词词性・中文释义
毎まい 日にち ①名词/副词｜每天
7しち 時じ ②名词短语｜七点（钟）
に ⓪助词｜时间点（在…时）
起おきます ②动Ⅱ｜起床（敬体）｜原形：起おきる ②

☆~から~まで

接续：名（时间）から名（时间）まで

意思：从……到……

映えい 画がは 8はち 時じ 半はんから 10じゅう 時じまでです。
单词词性・中文释义
映えい 画が名词｜电影
8はち 時じ 半はん时间词｜八点半
10じゅう 時じ时间词｜十点
は助词｜主题提示
から助词｜从…（起点）
まで助词｜到…为止（终点）
です系词｜判断、是（礼貌体）

☆いつ　动/名 か

意思：什么时候……？

社しゃ 長ちょうの結けっ 婚こん 式しきは いつですか。
单词词性・中文释义
社しゃ 長ちょう名词｜社长，总经理
結けっ 婚こん 式しき名词｜结婚典礼
の助词｜所属、限定（A の B：A 的 B）
は助词｜提示主题（念作「わ」）
いつ疑问词｜什么时候
です系词｜礼貌判断
か终助词｜句末疑问标记

第6课 吉田先生下个月去中国

☆助词：へ

文型：名（场所）へ 动（移动性）

解析：表示移动的方向。与え读音一样。

学がっ 校こうへ行いきます。
单词词性・中文释义
学がっ 校こう名词｜学校
へ助词｜方向（往…）。读作「え」，别念成“へ”。
行いきます动Ⅰ｜去（敬体）｜原形：行いく

☆助词：で

文型：名（交通工具）で 动（移动性）

解析：表示使用某交通工具移动。

注意：步行时使用：歩あるいて

自じ 転てん 車しゃで 学がっ 校こうへ 行いきます。
单词词性・中文释义
自じ 転てん 車しゃ名词｜自行车
で助词｜手段、方式（“用…/通过…”）
学がっ 校こう名词｜学校
へ助词｜方向、目的地（读作「え」）
行いきます动Ⅰ｜去（敬体）｜原形：行いく
地ち 下か 鉄てつで 会かい 社しゃへ 行いきますか。
单词词性・中文释义
地ち 下か 鉄てつ名词｜地铁
会かい 社しゃ名词｜公司
で助词｜手段、方式（“乘坐…/用…”）
へ助词｜方向、目的地；读作「え」
行いきます动Ⅰ｜去（敬体）｜原形：行いく
か终助词｜疑问标记

☆助词：と

文型：名（人）と 动

解析：表示共同做某事的对象。

恋こい 人びとと　一いっ 緒しょに　見みました。
单词词性・中文释义
恋こい 人びと名词｜恋人、对象
一いっ 緒しょに副词｜一起
と助词｜和、与（共同参与者）
見みました动Ⅱ｜看了（敬体过去）｜原形：見みる

☆~から~まで

接续：名（场所）から 名（场所）まで

意思：从……到……

東とう 京きょうから　広ひろ 島しままで　新しん 幹かん 線せんで　行いきます。

意思：从东京到广岛，坐新干线去。结构老老实实是「A から B まで C で 行きます」。

单词词性・中文释义
東とう 京きょう专有名词｜东京
広ひろ 島しま专有名词｜广岛
新しん 幹かん 線せん名词｜新干线
から助词｜起点，从…
まで助词｜终点，到…为止
で助词｜手段，用…方式
行いきます动Ⅰ｜去（敬体）｜原形：行いく

第7课 小李每天喝咖啡

☆助词：を

文型：名を动

解析：表示动作直接作用的对象

恋こい 人びとと　料りょう 理りを　作つくります。
单词词性・中文释义
恋こい 人びと名词｜恋人、对象
料りょう 理り名词｜料理、饭菜
と助词｜与、和（共同参与者）
を助词｜宾语标记
作つくります动Ⅰ｜做、制作（敬体）｜原形：作つくる
お酒さけを 飲のみません。
单词词性・中文释义
お酒さけ ⓪名词｜酒，酒类
を ⓪助词｜宾语标记
飲のみません动Ⅰ（敬体否定）｜不喝｜原形：飲のむ ①

☆助词：で

文型：名（场所）+ で +动

解析：表示动作发生或进行的场所。

映えい 画が 館かんで　映えい 画がを　見みました。

意思：在电影院看了电影。结构是「场所＋で＋宾语＋を＋动词ました」

单词词性・中文释义
映えい 画が 館かん名词｜电影院
映えい 画が名词｜电影
で助词｜动作发生地点（在…），也可表手段
を助词｜宾语标记
見みました动Ⅱ（敬体过去）｜看了｜原形：見みる
部へ 屋やで　休やすみます。
单词词性・中文释义
部へ 屋や ②名词｜房间，屋子
で ⓪助词｜场所，在…
休やすみます动Ⅰ｜休息（敬体）｜原形：休やすむ ②

☆名 か 名

解析：表示不确定

意思：相当于汉语的“或者”

毎まい 朝あさ、パンか たまごを 食たべます。

意思：每天早上吃面包或者鸡蛋。结构就是「名詞A か 名詞B を 動詞ます」。

单词词性・中文释义
毎まい 朝あさ名词/副词｜每天早晨
パン名词｜面包
たまご名词｜鸡蛋
を助词｜宾语标记
食たべます → 原形：食たべる动Ⅱ｜吃（敬体）→ 吃（辞书形）

☆名 をください

解析：用于花钱购物或者不花钱索取

意思：相当于汉语的“请给我……”

コーラと ケーキ を ください。

意思：请给我可乐和蛋糕。

单词词性・中文释义
コーラ名词｜可乐
ケーキ名词｜蛋糕
を助词｜宾语标记
と助词｜和、与（并列名词）
ください表达｜请给我…；源自动词命令形｜原形：下くださる
水みずを 二ふたつ ください。

意思：请给我两杯/两份水。（「二つ」是通用个数词，杯、个、份都能糊过去，服务员都懂。）

单词词性・中文释义
水みず名词｜水
二ふたつ数量词/助数词｜两个、两份（通用计数）
を助词｜宾语标记
ください表达｜请给我……｜原形：下くださる（敬语动词）

第8课 小李用日语写信

☆助词：で

文型：名 で 动

解析：

  • 第6课 表示交通工具
  • 第7课 动作进行的场所
  • 表手段、工具，原材料
日に 本ほん 語ごで　手て 紙がみを　書かきます。

意思：用日语写信。

单词词性・中文释义
日に 本ほん 語ご⓪名词｜日语
手て 紙がみ⓪名词｜信、书信
で⓪助词｜方式、手段
を⓪助词｜宾语标记
書かきます③动Ⅰ（敬体）｜写｜辞书形：書かく①
新しん 聞ぶん 紙しで　紙かみ 飛ひ 行こう 機きを　作つくりました。

意思：用报纸做了纸飞机。

单词词性・中文释义
新しん 聞ぶん 紙し③名词｜报纸（的纸）
紙かみ 飛ひ 行こう 機き⑤名词｜纸飞机
で⓪助词｜手段/材料（用…、以…）
を⓪助词｜宾语标记
作つくりました③动1（敬体过去）｜做了｜辞书形：作つくる①

授受句

文型1：名1（人）+ は + 名2（人）+ に　+ 名3（物）＋　を　あげる

解析：表示名1把名3给名2

意思：……把……给……

私わたしは　友とも 達だちに　プレゼントを　上あげます。
单词词性・中文释义
私わたし②代词｜我
友とも 達だち⓪名词｜朋友
プレゼント③名词｜礼物
に⓪助词｜给/向（受事者）
を⓪助词｜宾语标记
上あげます③动Ⅱ（敬体）｜给、递给｜辞书形：上あげる②
弟おとうとは　田た 中なかさんに　中ちゅう 国ごく 語ごの　小しょう 説せつを　上あげました。
  • 弟弟把中文小说送给了田中先生。
单词词性・中文释义
弟おとうと④名词｜弟弟
田た 中なか⓪专有名词｜田中（姓氏）
さん⓪敬称后缀｜先生/女士
に⓪助词｜给、向（受事者）
中ちゅう 国ごく 語ご⓪名词｜中文、汉语
の⓪助词｜连体（所属）
小しょう 説せつ⓪名词｜小说
を⓪助词｜宾语标记
上あげました④动Ⅱ（敬体过去）｜送给、给了｜辞书形：上あげる②

文型2：名1（人）+ は + 名2（人）+ に/から　+ 名3（物）＋　を　もらう

解析：表示名1从名2得到名3

意思：……从……得到……

森もりさんは　李りさんに　中ちゅう 国ごくの　お土みや 産げを　貰もらいました。
  • 森先生收到了李先生送的中国特产。
单词词性・中文释义
お土みや 産げ⓪名词｜土特产、礼物
は⓪助词｜提示主题
に⓪助词｜给予/来源（从谁那里）
の⓪助词｜连体（所属）
を⓪助词｜宾语标记
貰もらいました⑤动1（敬体过去）｜收到了、得到了｜辞书形：貰もらう⓪

文型2：名1（人）+ は + 名2（人）+ に　+ 名3（物）＋　を　くれる

解析：表示名1把名2给名3

意思：……把……给……

森もりさんは　弟おとうとに　漫まん 画がを　くれました。
  • 意思：森先生给了我弟一本漫画。
单词词性・中文释义
森もり①名词｜森（姓）
さん⓪敬称后缀｜先生/女士
は⓪助词｜主题提示
弟おとうと④名词｜弟弟
に⓪助词｜给予对象
漫まん 画が⓪名词｜漫画
を⓪助词｜宾语标记
くれました④动Ⅱ（敬体过去）｜给了（站在“我方”立场）｜**辞书形：**くれる⓪

用法顺手拎一下

  • あげる②：我方把东西给别人（A は B に C を あげる）。
  • くれる⓪：别人把东西给我方（A は 私/我方 に C を くれる）。
  • もらう⓪：我方从别人那里得到（私は A に/から C を もらう）。

☆~に 会あいます。

会あいます 见面，会面

に表示动作的对象

  • と 会あいます。
