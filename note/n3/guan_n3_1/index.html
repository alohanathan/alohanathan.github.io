N4语法延伸

2025年12月1日 · 1283 字 · 3 分钟 · 日语 中级上

  • たら
  • ながら
  • ために
  • ために ・ように 目的的区别
  • ようにする・ようになる
  • そうにない
  • 常见的口语缩略形

たら

  • 如果

  • 之后

  • 结果

暇ひまがあったら、見みてください。

如果有空的话，请看一下。

日本にほんに着ついたら、電でん 話わをしてね。

到了日本以后，打个电话给我哦。

わからないところを先生せんせいに 聞きいたら、先生せんせいは熱ねっ 心しんに 説せつ 明めいしてくださいました。

把不明白的地方问了老师之后，老师热心地给我解释了。

ながら

  • 一边、一边

  • 虽然、但是

连用形 + ながら

音おん 楽がくを聴ききながら、ご飯はんを食たべている。

一边听音乐，一边吃饭。

タバコは体からだに悪わるいと知しりながら、吸すってしまう。

明知道香烟对身体不好，却还是忍不住去抽。

ために

  • 目的：為ため ，是名词

    • 接续：
      • する + ために
      • の + ために

  • 原因

    • 接续：连体形 + + ために
医い 学がくを勉べん 強きょうする ために、 日に 本ほん 語ごを習ならっている。

为了学习医学，我正在学日语。

日に 本ほん 語ごが 下へ 手たなために、 その会かい 社しゃに入はいれなかった。

因为日语不好，没能进那家公司。

ために ・ように 目的的区别

  1. ために 前后主语必须统一；ように 前有主语，不一定
  2. ために 前半句必须跟意志表达（主观可控的）；ように 前半句必须跟非意志表达（客观不可控的）

太ふとらない（ために・ように）、 食たべる量りょうを減へらす。

为了不变胖，减少食量。

ない型 + ように

日に 本ほんへ行いく（ために・ように）、 日に 本ほん 語ごを勉べん 強きょうします。

为了去日本，学习日语。

去，是一个意志表达

日に 本ほんへ行いける（ために・ように）、 日に 本ほん 語ごを勉べん 強きょうします。

为了“能”去日本，学习日语。

这里的「行ける」是可能形，能力不可控

ようにする・ようになる

ようにする 尽量（主观意志）

ようになる 变化（客观变化）

遅ち 刻こくしないようにする

为了不要迟到而努力（设法做到不迟到）。

遅ち 刻こくしないようになる

变得不会迟到了；逐渐做到不再迟到。

这里的 ようになる 与 ようにする 的差别很关键：

  • ようにする：主语主动“努力使自己做到”
  • ようになる：状态自然变化为“逐渐能做到”

一个是主动施力，一个是状态转变。

そうにない

意义：看样子不会

= そうもない

= そうではない

接续：参照样态助动词接续

雨あめが止やみそうもない

雨看起来一点也没有要停的样子。（“完全不像要停”的语气，比 そうにない 更强烈。）

常见的口语缩略形

  • ている → てる

  • でいる → でる

愛あいしてる

  • ては → ちゃ

  • では → じゃ

  • なくては ならない → なくちゃ

触さわっちゃだめ

不能碰 / 别碰。

じゃない

不是。

  • てしまう → ちゃう

  • でしまう → じゃう

忘わすれちゃった　

忘掉了／我忘了。

死しんじゃった

死掉了。

  • なければ → なきゃ

  • と、という、というのは → って

帰かえらなきゃ

得回去了／必须回去。

僕ぼくは 谷たに 安やすって言いいます。

我叫谷安。

这里的「〜って言います」是自我介绍的常用说法，比「〜と申します」更口语、更随意。

  • とおく → とく
    • とく → といて、といた、とかない、とこう
やめとこう

算了吧／还是别做了吧。

やめる → やめておく（事先停止／放弃）→ やめておこう（意志形）→ やめとこう（口语省音）

  • ても → たって

  • でも → だって

誰だれだって知しっている

谁都知道。

  • でしだか → っけ

  • ましたか → っけ

今日きょうは 何なん 曜よう 日びだっけ？

今天星期几来着？
