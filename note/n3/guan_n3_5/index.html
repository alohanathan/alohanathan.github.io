といえば

意义：说起~

接续：名词 + といえば

日に 本ほんといえば、 私わたしは 桜さくらを イメージします。

一说到日本，我就会联想到樱花。

夏なつといえば、 花火はなびですね。

一说到夏天，就会想到烟花呀。

というのは

意义：下定义 = とは

接续：名词 + というのは

東とう 大だいというのは 東とう 京きょう 大だい 学がくのことだ。

所谓“东大”，指的是东京大学。

東とう 大だいとは 東とう 京きょう 大だい 学がくのことだ。

ということだ

  • 也就是说

    • 接续：简体小句 + ということだ

    • かのじょは まだ 来きていない。つまり、 遅ち 刻こくということだ。
      • 她还没来。也就是说，她迟到了。

  • 听说

    • 接续：简体小句 + ということだ

    • 天てん 気き 予よ 報ほうによると、 明あ 日したは 雨あめが 降ふる ということだ。
      • 根据天气预报，明天会下雨。

というほどではない

という + ほど + ではない　

意义：并没有达到~程度

接续：简体形 + というほどではない

病びょう 状じょうは 手しゅ 術じゅつが 必ひつ 要よう というほどではない。

病情还不至于需要做手术。 （程度没有严重到需要手术的地步。）

に+动词+て

  • に 対たいして

    • 面对

    • 目め 上うえ の 人ひと に 対たいして 敬けい 語ご を 使つかう べき だ。
      • 对上级（地位高的人）应该使用敬语。

  • にとって

    • 对于

    • この 荷にもつ 物もの は 私わたし にとって 重じゅう 要よう だ。
      • 这件行李对我来说很重要。

  • について

    • 关于

    • 給きゅう 料りょう について 検けん 討とう する
      • 就薪资方面进行讨论 / 研究。

  • に 関かんして

    • 关于（较正式）

  • において

    • 在……；等于に、で

    • 会かい 場じょう において
      • 在会场、在会场这个场合
    • 2000年ねん において
      • 在2000年、于2000年
    • 言げん 語ご 学がく 分ぶん 野や における 研けん 究きゅう

    • 在语言学领域的研究

  • に 代かわって

    • 代替

    • 課か 長ちょう に代か わって、 会かい 議ぎ に出しゅっ 席せき した。
      • 代替课长（部门经理），出席了会议。

  • に 比くらべて

    • 和~比较

    • 去きょ 年ねん に比くら べて、 今こ 年とし の 冬ふゆ は 寒さむ い。
      • 和去年相比，今年的冬天更冷。

から〜にかけて

意义：从~到~ = から~まで

接续：名词 + にかけて

十じゅう 一いち 月がつ から 三さん 月がつ にかけて

从十一月到三月这段时间。

北ほっ 海かい 道どう から 九きゅう 州しゅう にかけて

从北海道到九州这一带

によって

1、意义：表示依据；“根据”

成績せいせき によって 順位じゅんい をつける。

根据成绩来排定名次。

人ひと によって 違ちが う。

因人而异。

2、意义：表示手段；“通过”

ネット によって 情報じょうほう を 知し る。

通过网络来获取新闻 / 信息。

3、意义：表示原因；“因为”

事故じこ によって、 電車でんしゃ が 遅おくれた。

因为事故，电车晚点了。

4、意义：表示主体；“由”

この 本ほん は 木村きむら さん によって 翻訳ほんやく された。

这本书是由木村先生翻译的。

たびに

度に

意义：每次

接续：连体形 + たびに

この 写真しゃしん を 見みる たびに、 故郷こきょう の こと を 思おもい 出だ}す。

每当看到这张照片时，就会想起故乡。

くせに

意义：明明~ 等于 のに

接续：连体形 + くせに

かれ は 新しん 入にゅう 社しゃ 員いん の くせに、 生なまい 気き だ。

他明明是个新员工，却很嚣张／却很狂。

すえに

末に

意义：结果

接续：た形 + 末に / 名词 + の + 末に

いろいろ 考かんがえた 末すえ に、 今こん 回かい の 旅りょ 行こう は 行いかない と 決きめた。

经过各种考虑之后，我决定这次旅行不去了。

最さ 中ちゅう に

意义：正在

接续：

ている + 最さ 中ちゅう に

名词 + の + 最さ 中ちゅう に

電でん 話わ を して いる 最さ 中ちゅう に、 誰だれ か が 家いえ に 来きた。

正在打电话的时候，有人来我家了。

食しょく 事じ の 最さ 中ちゅう に、 お客きゃく さん が 来きた。

吃饭正进行的时候，客人来了。

整个语气是在吃饭的关键时刻被打断。

含义：

  • 某个动作或状态正在进行的正当中
  • 出现另一个意料之外/突发事件

常见语气：

  • “正……的时候”
  • “偏偏在……的时候”

有时带困扰、麻烦的语感。

代かわり に

  • 作为交换

    • 接续：连体形 + 代かわり に
    • ノート を 貸かして もらった 代かわりに、 奢おごり ましょう。
      • 因为对方把笔记借给了我，作为回报，我来请客吧。

  • 虽然……但是

    • 接续：连体形 + 代かわり に

    • この仕し 事ごとは安あん 定ていしている代かわりに、給きゅう 料りょうが高たかくない。
      • 这份工作虽然很稳定，但工资不高。

間あいだに・間あいだ(は)

意义：在……期间

接续：连体形 + 間あいだ

しばらく見みない間あいだに、ずいぶん逞たくましくなったね。

逞たくましい

好久没见，你变得强壮了呢。

間あいだに 在一段时间内的一个时间点，完成了一个动作

間あいだ(は) 在一段时间段持续这个一个状态

もとで

下もとで

意义：在……之下

接续：名词 + の + もとで

先せん 生せいの指し 導どうのもとで、日に 本ほん 文ぶん 学がくを研けん 究きゅうしている。

在老师的指导下，我正在研究日本文学。

て以い 来らい

= てから・てこのかた

意义：自从~

接续：て形 + て以い 来らい

結けっ 婚こんして以い 来らい、家か 事じに追おわれる日ひ 々びが続つづいている。

てはじめて

初めて

意义：知道~才~

接续：て形 + てはじめて

病びょう 気きになってはじめて、健けん 康こうの大たい 切せつさがわかった。

生病之后才第一次明白健康的重要性。

中ちゅう 心しんに

意义：以……为中心

接续：名 + 中ちゅう 心しんに

地ち 球きゅうは 太たい 陽ようを 中ちゅう 心しんに 回っている。

地球以太阳为中心旋转。（地球绕着太阳转。）

を込こめて

意义：充满~

接续： 名词 + を込こめて

愛あいを込こめて

倾注爱意／带着爱

心こころを込こめて

倾注心意／全心全意

感かん 謝しゃの気き 持もちを込こめて

带着感激之情／怀着感谢的心情

をはじめとする

初め

とする　把……当做

意义：以~为首；以……为首的一类事物

接续：名词 + をはじめとする

この町まちはこのお寺てらをはじめ、いろんな古ふるい建けん 物ぶつがある。

这座城镇以这座寺庙为代表，有许多古老的建筑。

にしろ〜にしろ／にせよ

意义：无论是~还是~

接续：简体形 + にしろ〜にしろ

好すきにしろ、嫌きらいにしろ、健けん 康こうのため、野や 菜さいを食たべます。

不管喜欢还是讨厌，为了健康，都要吃蔬菜。

おきに・毎ごとに

相同：划分时间，距离时没有区别

３メートルおきに木きを植うえる。

每隔 3 米种一棵树。

３メートルごとに木きを植うえる。

每 3 米种一棵树。

表达含义语感
おきに每隔…强调“空隙”
毎ごとに每…强调“单位循环”

以上いじょうは

意义：既然

接续：连体形、Nである + 以上いじょうは

こうなった以上いじょう、 最後さいごまで責任せきにんを持もつ。

既然变成这样了，就要负责任到底。

学生がくせいである以上いじょうは、 いっしょうけんめい勉強べんきょうするべきだ。

既然是学生，就应该拼命学习。

同义替换：-

  • こうなった 以上いじょうは、 最後さいごまで責任せきにんを持もつ。
  • こうなった 上うえは、 最後さいごまで責任せきにんを持もつ。
  • こうなった からには、 最後さいごまで責任せきにんを持もつ。
  • 既然变成这样，就要负责到底。

おかげで・せいで

辨析：

  • おかげで 好的原因
  • せいで 不好的原因

接续：连体形

あなたのお蔭おかげで、 風邪かぜが完全かんぜんに治なおった。

多亏了你，感冒完全好了。

あなたの所為せいで、風邪かぜを引ひいてしまった。

都怪你，我感冒了。
