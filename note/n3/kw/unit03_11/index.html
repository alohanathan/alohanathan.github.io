会話 若者の意識

1）あのポスター、張ちょう一いっ心しんですね。

中文：那张海报，是张一心吧。 语法：句末 ですね 确认/共感；专有名用片假名/中文名汉字直接读音。

2）ええ。新しん作さく映えい画がの「友とも達だち」が今こん週しゅう公こう開かいされたばかりです。興こう行ぎょう成せい績せきは現げん在ざいのところは第だい二に位いですが、来らい週しゅうはトップになる見み込こみです。

中文：是的。新片《朋友》刚在本周上映。票房目前排第二，但预计下周会登顶。 语法：

  • V-たばかり：刚刚……。
  • 名＋です/ですが：转折承接。
  • N になる：变成……。
  • V-辞＋見込み：预期/展望。
3）張ちょう一いっ心しんは中ちゅう国ごくでは大だいスターですが、日に本ほんでも、すごい人にん気きがあるんですね。

中文：张一心在中国是大明星，在日本也很有人气啊。 语法：

  • N では…が、…でも：对比两个范围。
  • ～んです：说明/感叹语气。
4）はい。彼かれが主しゅ演えんする映えい画がは、アクションでも恋れん愛あいドラマでも、必かならず当あたります。

中文：是的。他主演的电影，无论动作片还是爱情剧，必定会大卖。 语法：

  • N でも N でも：无论……还是……。
  • V-ます/辞＋当たる：走红、命中（引申为“卖座”）。
  • 必ず：必然，一定。
5）中ちゅう国ごく映えい画がの人にん気きはどうなんですか。

中文：那中国电影的人气怎么样？ 语法：

  • どうなんですか：委婉询问“情况如何”。
6）ええ。この１か月げつで３本ほんの中ちゅう国ごく映えい画が公こう開かいされました。どれも高たかく評ひょう価かされています。

中文：嗯，这一个月上映了三部中国电影，都获得了很高评价。 语法：

  • 期間＋で：在……期间内。
  • V-られる（される）：被动；評価される“被评价”。
  • どれも：都、无一例外。
7）中ちゅう国ごく以い外がいではどの国くにの映えい画が人にん気きがあるんですか。

中文：除了中国之外，哪个国家的电影更有人气？ 语法：

  • N 以外では：在……之外。
  • どの N：哪一个……
8）アメリカや韓かん国こく…そうそう、日に本ほん映えい画がの人にん気きも高たかまっていますよ。

中文：美国和韩国……对了，日本电影的人气也在上升呢。 语法：

  • ～や…：举例。
  • V-ている（高まっている）：趋势进行“正在上升”。
  • 口语插入 そうそう：想起补充。
9）そうすると、外がい国こくのものだから人にん気きがあると言いうわけではないんですね。

中文：这么说来，并不是因为“外国的”就有人气啊。 语法：

  • そうすると：由此可见/那么。
  • ～わけではない：并非……（否定整体理由）。
10）ええ、若わか者ものは日に本ほんのものと外がい国こくのものを区く別べつしていないような気きがします。彼かれらは、映えい画がにしても音おん楽がくにしても、いいものならすぐに受うけ入いれます。

中文：对，年轻人似乎并不区分“日本的/外国的”。无论电影还是音乐，只要是好东西就会立刻接受。 语法：

  • ～ような気がする：似乎/感觉。
  • N にしても N にしても：无论……还是……。
  • V-ます/辞＋受け入れる：接受、吸纳。
11）そうですね。中ちゅう国ごくの若わか者ものも同おなじだと言いう気きがします。じゃあ、彼かれらの心こころをつかむには、どうすればいいんでしょうか。

中文：确实。中国的年轻人也差不多。那么，要抓住他们的心，该怎么做才好呢？ 语法：

  • ～には：为了……（目的/条件提示）。
  • どうすればいい：应当怎么办（假定形ば＋いい）。
  • 句尾 ～んでしょうか：委婉请教。
12）うーん…、難むずかしい質しつ問もんですね。やはり本ほん当とうにいいものを作つくることではないでしょうか。

中文：嗯……这个问题很难。果然还是要做出真正好的东西吧。 语法：

  • やはり：果然/还是。
  • V-ることではないでしょうか：不是应该……吗（委婉提案）。
13）本ほん当とうにいいもの、ですか。

中文：真正好的东西吗？ 语法：重复对方关键词＋升调确认。

14）ええ。今いまの若わか者ものは、テレビCMより、同どう世せ代だいの人ひとの口くち込こみを信しんじます。映えい像ぞうでも音おん楽がくでも食たべ物ものでも、本ほん当とうにいいものを提てい供きょうすれば、口くち込こみで広ひろがっていくはずです。

中文：是的。现在的年轻人，比起电视广告，更相信同龄人的口碑。无论影像、音乐还是食物，只要提供真正好的东西，就会靠口碑自然扩散。 语法：

  • A より B を信じる：相比 A 更相信 B。
  • ～でも～でも：无论……还是……。
  • V-ば：只要……就……。
  • V-ていく：向将来扩展的趋势。
  • ～はずだ：按理会/应该会（说话人的合理推断）。
15）なるほど。有ありがとう難がとうございます。とても参さん考こうになりました。

中文：原来如此。谢谢您，非常有参考价值。 语法：

  • なるほど：表示理解/恍然。
  • 参考になる：对我有参考价值。

小结・表达要点

  • わけではない：并非（全面否定某理由/常识）。
  • V-たばかり（刚刚）、V-ている（进行/状态）、V-ていく（向今后发展）。
  • 口コミ（口碑）在年轻群体影响力 > テレビCM。
  • 把握年轻人：不看出身地，看“是否真的好” → 好内容/好产品自带扩散。

課文 漫画とあり目

1）日に本ほんに来きた外がい国こく人じんは、電でん車しゃなどでサラリーマンが夢む中ちゅうになって漫まん画がを読よんでいる光こう景けいを見みて驚おどろくと言いう。

中文：来到日本的外国人常说，会在电车里看到上班族沉迷看漫画的景象而惊讶。 语法：

  • ～になって：状态转变“变得……”；夢中になる“入迷”。
  • Vている：动作进行/习惯。
  • と + 言う：转述。
2）日に本ほんの漫まん画がは、子こ供どもの頃ころから成せい人じん向むけまで、対たい象しょうの年ねん代だいごとにさまざまな種しゅ類るいがある。

中文：日本漫画从儿童到成人向，按读者年龄层分成各种类型。 语法：

  • N から N まで：范围从……到……。
  • N ごとに：按……（分类/每……）。
3）会かい社しゃ員いんの生せい活かつや、経けい済ざいに関かんするものなど、成せい人じんにしか楽たのしめない内ない容ようの漫まん画がも多おおくある。

中文：也有很多只适合成人欣赏的题材，如社员生活、经济相关等。 语法：

  • N にしか V-ない：只有……才……（限定+否定）。
4）一いっ方ぽう、子こ供ども向むけの漫まん画がでも、成おと人なが楽たのしめるものも多おおい。

中文：另一方面，儿童向的漫画也有不少是大人也能享受的。 语法：一方表示对照；V-可能形表达可享受的对象。

5）近きん年ねん、中ちゅう国ごくでは「クレヨンしんちゃん」「ドラえもん」「ちび丸まる子こちゃん」などの漫まん画が子こ供どもだけでなく大おお人となからも人にん気きを集あつめている。

中文：近年在中国，《蜡笔小新》《哆啦A梦》《樱桃小丸子》等不仅孩子喜欢，也深受大人欢迎。 语法：

  • A だけでなく B からも：不但 A，也从 B（处/群体）获得……。
  • V-ている：持续状态。
6）日に本ほんのテレビアニメも外がい国こくでの評ひょう価かが高たかい。

中文：日本的电视动画在海外评价也很高。 语法：N での N：表示领域/场合的“在……的”。

7）日に本ほんで最さい初しょのテレビアニメは１９６３年ねんの「鉄てつ腕わんアトム」だ。

中文：日本第一部电视动画是 1963 年的《铁臂阿童木》。 语法：最初の N：最早/第一的……。

8）漫まん画がをアニメ化かしたものだが、その画が質しつは今いまとは比くらべ物ものにならないほど悪わるかった。

中文：那是把漫画动画化的作品，但画质和现在没法比，当时相当差。 语法：

  • ～化する：……化。
  • N とは比べ物にならないほど A：与……无法相比地 A。
9）その後ご、デジタル技ぎ術じゅつの開かい発はつが進すすむとともに、高こう画が質しつの生いき生いきとした映えい像ぞうが生うみ出だされるようになった。

中文：其后，随着数字技术的发展，高画质、活灵活现的影像逐渐被制作出来。 语法：

  • V-とともに：随着……同时。
  • V-れるようになった：开始能够/常态化地被……。
10）「ドラえもん」「クレヨンしんちゃん」などの漫まん画がも、テレビ番ばん組ぐみ内ないにアニメ化かされ、高こう視し聴ちょう率りつを獲かく得とくした。

中文：诸如《哆啦A梦》《蜡笔小新》等也被改编成电视动画，取得了高收视率。 语法：

  • N 内：在……之内/内部。
  • を獲得する：获得。
11）やがて、最さい先せん端たんのデジタル技ぎ術じゅつを使つかったアニメが映えい画が館かんのスクリーンに登とう場じょうし、娯ご楽らく性せいや芸げい術じゅつ性せいの高たかい作さく品ひんが次つぎ々つぎに製せい作さくされるようになった。

中文：不久，采用最前沿数字技术的动画走上影院大银幕，具有高娱乐性与艺术性的作品接连被制作出来。 语法：

  • V-たアニメ：过去分词作定语。
  • 次々に：接连地。
  • V-れるようになった：逐渐成为常态。
12）２００１年ねんに公こう開かいされた「千せんと千ち尋ひろの神かみ隠かくし」は、その豊ゆたかな想そう像ぞう力りょくと高たかい表ひょう現げん力りょくが世せ界かい的てきにも高たかく評ひょう価かされた。

中文：2001 年上映的《千与千寻》，以其丰富想象力和高超表现力，在世界范围内也获得了高度评价。 语法：

  • N 的にも：在……范围/立场上也……。
  • V-られる（評価される）：被动。

小结/记忆点

  • N だけでなく N からも：不仅……也受到（来自……）……。
  • V-とともに：随着……（并行变化）。
  • V-れるようになった：逐渐形成/能够……。
  • 比べ物にならないほど A：A 到无法相提并论的程度。
  • 次々に：一个接一个。
