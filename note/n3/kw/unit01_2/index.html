新标日中级上_第1单元_第2课

会话

大おお山やま：すみません、お待またせしました。

中文：不好意思，让您久等了。 语法解析

  • おV-ます根 + せする／して + しまう → お待たせしました：
    • 来自动词 待つ（基本形）→ 待たせる（使役「让……等待」）→ お待たせする（美化/自谦）→ 过去礼貌 しました；整句是向对方致歉的固定表达“让您久等了”。
  • すみません：道歉/搭话的客气用语。
佐さ藤とう：どうもごぶさたしています。

中文：久疏问候（好久不见）。 语法

  • ご無沙汰しています（惯用句）：字面“荒废问候”，表示久未联系。持续体 〜ています 表示“处在…的状态”。
大おお山やま：本ほん当とうにお久ひさしぶりです。佐さ藤とうさんと一いっ緒しょに仕し事ごとをするのは、４よん年ねんぶりですね。

中文：真的好久不见了。和佐藤先生一起工作，已经是时隔四年了呢。 语法

  • お久しぶりです：固定寒暄“好久不见”。
  • 名词＋ぶり：表示相隔的时间。“四年ぶり＝相隔四年”。
  • V-辞書形＋のは〜：将动作名词化作主语主题；するのは〜“做这件事是……”。
  • 助词 と（与…一起）+ 一緒に（一起）。
佐さ藤とう：ええ、早はやいものですね。その節せつはお世せ話わになりました。

中文：是啊，时间过得真快。承蒙您当时的关照。 语法

  • 早いものですね：“真快啊”（主观感叹）。
  • その節はお世話になりました：商务常用“当时承蒙关照”（世話になる 自下而上受惠表达，过去式 になりました）。
大おお山やま：いいえ、こちらこそ。

中文：哪里，是我才受您照顾。 语法

  • こちらこそ：对对方致谢/致歉/寒暄时的客套回应“彼此彼此/应该我说才对”。
佐さ藤とう：今こん回かいもよろしくお願ねがいします。

中文：这次也请多多关照。 语法

  • よろしくお願いします：请求/拜托的定式。
  • も：在“这次也”的语感上承接过去合作。
同どう僚りょう：お話はなし中ちゅう、失しつ礼れいします。JC企き画かくの李りさんがいらっしゃいました。

中文：打扰你们谈话了。JC企划部的李先生到了。 语法/形态

  • お話中：名词＋中表示“正在……之中”。
  • 失礼します：进入/打扰时的礼貌用语。
  • いらっしゃいました：尊敬语（来る・行く・いる 的尊敬形 いらっしゃる → 过去礼貌）。
  • N の 人：所属与限定，“企画の李さん”。
大おお山やま：あ、そうですか、じゃあ、こちらにお願ねがいします。

中文：啊，是吗？那就麻烦让他到这边来吧。 语法

  • じゃあ：口语“那么、那”。
  • こちらにお願いします：に 表示方向/目的地，“请安排到这边来/这边请”。
李り：こんにちは。

中文：你好。 语法：日常问候（中午前后）。

大おお山やま：こんにちは、李りさん。紹しょう介かいします。こちらは、今こん度どの「金きん星せい」プロジェクトの責せき任にん者しゃで、佐さ藤とうです。

中文：你好，李先生。我来介绍一下。这位是这次“金星”项目的负责人，佐藤。 语法

  • 紹介します：他动词 紹介する 的礼貌形。
  • こちらは A で、B です：で 表示身分/状态，“是 A（身份），名叫 B”。
  • 外来语 プロジェクト（项目）。
李り、佐さ藤とう：初はじめまして……。あれっ、昨きの日うの……！

中文：初次见面……咦，昨天的……！ 语法/注意

  • 初めまして：初次见面寒暄（来自动词 始める 的连用形名词化）。
  • あれっ：表示惊讶/想起的语气词。
  • 昨日の〜：后接省略，暗示“昨天见过的那位/那件事”。

重点语法与形态一览（N3 向）

  • 让对方久等：お待たせしました ← **待つ（基本形）**→ **待たせる（使役）**→ **お待たせする（美化/自谦）**→ 〜しました。
  • ぶり（隔时）：四年ぶり＝相隔四年。句式 V/名 の は〜 可把动作名词化作主题。
  • その節はお世話になりました：对过去受照顾的客套表达；世話になる（受关照）过去式。
  • こちらこそ：对等回应。
  • よろしくお願いします：请求/合作固定语。
  • お話中／〜中：名词＋中＝“正在……”。
  • いらっしゃる（尊敬）→ いらっしゃいました（过去礼貌）：对应 来る/行く/いる。
  • A は B で、C です：で作系词前的身分/属性说明（「責任者で、佐藤です」）。
  • 初めまして：初次见面定式，后常接よろしくお願いします。

课文

日に本ほん人じん同どう士しがあいさつする時ときは、普ふ通つうお辞じ儀ぎをします。

中文：日本人彼此打招呼时，一般都会行礼（鞠躬）。 语法解析

  • 名詞＋同士：表示“同类之间”，“日本人同士＝日本人之间”。
  • V-辞書形＋時：表示时间点“……的时候”。
  • 普通（ふつう）：副词/名词，“通常、一般”。
  • お＋サ变/名词＋します：美化/礼貌表达；お辞儀をする→お辞儀をします。
中ちゅう国ごくではあいさつをする時とき、握あく手しゅをするのは普ふ通つうですが、日に本ほん人じんは、握あく手しゅよりもお辞じ儀ぎをするのが一いち般ばん的てきです。

中文：在中国打招呼时通常握手，但日本人比起握手，更一般的是行礼。 语法解析

  • 〜のは普通ですが：提出对比的让步“虽然……很常见，但……”。
  • AよりB：比较，“与其 A 不如 B”。
  • 〜のが一般的です：名词化 + 判断，表示社会常态。
お辞じ儀ぎの角かく度どには、首くびを曲まげるだけの軽かるいものから深ふかく頭あたまを下さげるものまでで、いろいろあります。

中文：鞠躬的角度有很多种，从只是点头的轻微礼到深深低头的礼都有。 语法解析

  • 〜には：话题提示 + 范围界定。
  • A から B まで：范围“从……到……”。
  • いろいろあります：存在表达（ある），说明种类多。
  • V-ます根＋下げる：向下、降低之意；頭を下げる“低头”。
普ふ通つう、初しょ対たい面めんの時ときには深ふかく頭あたまを下さげて、丁てい寧ねいなお辞じ儀ぎをします。

中文：一般来说，初次见面时会深深低头，做一个郑重的鞠躬。 语法解析

  • 名＋の時には：特指“在……这种情况下”。
  • 丁寧な：形动作定语“郑重的、礼貌的”。
  • て：并列动作，“低头并行礼”。
仕し事ごとで初しょ対たい面めんの人ひとに会あった時ときは、名めい刺しを出だします。

中文：在工作中与陌生人第一次见面时，要递上名片。 语法解析

  • で：场合/领域，“在工作上”。
  • V-た＋時：过去经历时间点“当见到时”。
  • 名刺を出す：固定搭配“递名片/拿出名片”。
名めい刺しの出だし方かたと受うけ取とり方かたにも決きまりがあります。

中文：递名片和接名片也有规定。 语法解析

  • N の V-ます根＋方：做法/方式，“出し方、受け取り方”。
  • 〜にも〜がある：强调“连……也有……（规定/规则）”。
相あい手てが目め上うえの時ときは、まず自じ分ぶんから先さきに名めい刺しを出だして深ふかいお辞じ儀ぎをし、次つぎに相あい手ての名めい刺しをお辞じ儀ぎをしながら受うけ取とります。

中文：对方是长辈/上司时，先由自己递上名片并深鞠躬，然后一边行礼一边接过对方名片。 语法解析

  • 目上：名词“上级、长辈”。
  • まず…、次に…：顺序连接。
  • V-ます根＋ながら：一边……一边……（同时动作）。
  • V-て、V：动作连贯。
出だす時ときも受うけ取とる時ときも、両りょう手てを使つかう方ほうが丁てい寧ねいです。

中文：不论递出还是接收，都用双手更显得礼貌。 语法解析

  • 時も…時も：并列强调。
  • V-辞書形＋方がいい／です：比较说法，“……做法更好/更得体”。
また、話はなす内ない容ようにも気きをつけなければなりません。

中文：另外，说话内容也必须注意。 语法解析

  • 〜なければなりません：义务/必要“必须……”。
  • 気をつける：注意、留意。
初しょ対たい面めんの時ときは、個こ人じん的てきなことを聞きかないのが普ふ通つうです。

中文：初次见面时，不问私人问题是常规做法。 语法解析

  • 〜ないのが普通だ：对行为的普遍规范性判断。
  • 個人的なこと：形动 + 連体化。
特とくに仕し事ごとの場ばで出で会あった相あい手てには、個こ人じん的てきな質しつ問もんは避さけたほうがいいでしょう。

中文：尤其在工作场合认识的人，最好避免私人提问。 语法解析

  • 名詞＋には：就……而言/对……。
  • V-たほうがいい：劝告“最好……”。
  • でしょう：婉转推量/建议口吻。
結けっ婚こんしているかどうか、給きゅう料りょうはいくらか、年ねんはいくつかなどは、普ふ通つうは聞ききません。

中文：像“是否结婚、工资多少、年龄几何”等，一般不去打听。 语法解析

  • 〜かどうか：是否。
  • 〜か、〜か：并列疑问内容。
  • などは：列举+话题提示。
  • 普通は〜ません：一般性的否定规范。
大だい事じなのは、相あい手てに不ふ快かい感かんを与あたえないことです。

中文：最重要的是，不要给对方带来不适感。 语法解析

  • 大事なのは〜ことです：提要句型，“关键在于……”。
  • V-ないこと：名词化作主语内容。
  • 与える：给予/造成（负面也可用）。
いくら丁てい寧ねいな言こと葉ばであいさつしても、怖こわい顔かおをしていては、相あい手てにいい印いん象しょうを与あたえることはできません。

中文：即便说得再客气，如果板着脸，也无法给对方留下好印象。 语法解析

  • いくら〜ても：让步“无论再怎么……也”。
  • V-ている＋ては：条件让步（处于……状态就会）。
  • 〜ことはできません：可能否定“做不到……”。
気き持もちがこもっているかどうかは、予よ想そう以い上じょうに相あい手てに伝つたわってしまうものです。

中文：感情是否投入，往往会超出想象地传达给对方。 语法解析

  • 〜かどうかは：将“是否……”整体名词化做主题。
  • V-てしまう：完了/遗憾或不可逆语气；这里偏“自然而然地传达出去”。
  • ものです：普遍道理/经验判断，“往往会……”。

补充词汇与表达（N3 重点）

  • お辞儀（おじぎ）：鞠躬、行礼
  • 名刺（めいし）：名片／名刺交換（こうかん）
  • 目上（めうえ）⇄ 目下（めした）
  • 両手（りょうて）で受け取る：用双手接
  • 避ける（さける）：避免
  • 印象（いんしょう）を与える：给出印象
  • 〜かどうか：是否
  • 〜たほうがいい／〜ないほうがいい：劝告
  • いくら〜ても：无论多么……也
  • 〜てしまう：无意完成／遗憾语气
