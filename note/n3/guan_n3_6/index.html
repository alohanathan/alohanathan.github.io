一方いっぽうで、一方いっぽうだ

  • 一方いっぽうで ： 一方面……另一方面

    • 接续：连体形 + 一方いっぽうで

    • 科学かがくの発展はってんは便利べんりな生活せいかつをもたらす一方いっぽうで、 深刻しんこくな環境かんきょう 問題もんだいを引ひき起おこした。
      • 科学的发展一方面带来了便利的生活，另一方面造成了严重的环境问题。

  • 一方いっぽうだ：一个劲地……；不断……

  • 接续：连体形 + 一方いっぽうだ

  • 給料きゅうりょうは下さがる一方いっぽうだ。
    • 工资一直在下降 / 只会继续下降。
  • 人口じんこうは増ふえる一方いっぽうだ。
    • 人口一直在增加 / 只会继续增加。

て済すむ

意义：~就解决了

接续：て形 + て済すむ

ガラスを割わってしまったが、謝あやまって済すんだ。

把玻璃打碎了，但道歉就解决了事情。

也可以理解为： “虽然弄坏了玻璃，但已经道歉完事了。”

古语的否定 = ない

否定

  • ぬ 接名词or结句
  • ず 接动词or句子

ずに済すむ

意义：不……也可以（解决）ずに = ないで

接续：

  • ない形 + ずに済すむ
  • する→せ+ ずに済すむ
風かぜ 邪じゃの症しょう 状じょうが軽かるかったので、病びょう 院いんに行いかずに済すんだ。

因为感冒症状很轻，不用去医院就解决了。

たまえ

給

意义：轻微命令

接续：连用形 + たまえ

来きたまえ ＝ 来きてください
読よみたまえ ＝ 読よんでください
表达语气
来てください礼貌请求
来たまえ命令、强势、上对下、男性用

てほしい・てちょうだい

てちょうだい

  • 意义：请~ てください　女性、小孩用语

  • 接续：て形 + てちょうだい

  • ここへ 来きてちょうだい。
    • 到这里来吧（来这边一下）。
  • その 新しん 聞ぶん をとってちょうだい。
    • 把那份报纸拿给我。

てほしい

  • 请 = てもらいたい

  • 接续：て形 + てほしい

  • 明日うちへ来きてほしい。
    • 明天希望你来我家。
  • 雨あめが降ふってほしい。
    • 希望下雨。

んだって

意义：听说 んだ = のだ　　って＝ということだ

接续：名词接续 + んだって

午後から雨あめが降ふるんだって。

听说明天下午开始要下雨。（语气比较口语、随便）

午後から雨あめが降ふるそうです。

据说明天下午开始要下雨。（语气中性、比较客观礼貌）

んじゃない

  • 不是~吗？（升调）

    • 良いいんじゃない？
      • 不是挺好吗？
    • あの人ひとが好すきなんじゃない？
      • 你不会喜欢那个人吧？

  • 不许（降调）

    • 動うごくんじゃない！
      • 别动！
    • 油断ゆだんするんじゃない！
      • 不要大意！

ないかな

意义：多希望~啊

接续：ない形 + ないかな

早はやく 夜よが 明あけないかな。

真希望天快亮啊 / 不知道天能不能早点亮呢。

よう とする

意义：正要~

接续：意向形 + とする

出でかけようとした時とき、電でん 話わが鳴なった。

正要出门的时候，电话响了。

エレベーターに乗のろうとした時とき、ドアが閉しまった。

正要上电梯的时候，门关上了。

てよかった

意义：幸亏~

接续：て形 + てよかった

先せん 生せいに相そう 談だんしてよかった。

和老师商量真是太好了。

携けい 帯たい、見みつかってよかった。

手机被找到了，真是太好了。

ではないか

意义：双重否定表肯定；不是~吗？ = ではないでしょうか

接续：简体形 + ではないか

外そとは大おお 雪ゆきじゃないか。　外そと 出しゅっするのは危き 険けんだ。

外面不是在下大雪吗？外出很危险啊。

あの子こはまだ子こ 供どもではないか。　親おやはどうしてかれを遊あそばせないのだろう。

那个孩子还只是个小孩吧？家长为什么不让他去玩呢。

恐おそれがある

意义：恐怕

接续：连体形 + 恐おそれがある

地じ 震しんの影えい 響きょうで津つ 波なみの恐おそれがありますから、緊きん 急きゅうに避ひ 難なんしてください。

由于地震的影响，有发生海啸的可能，请紧急避难。

无济于事

  • ても仕し 方かたがない

    • 再怎么做也没办法。

  • ても始はじまらない

    • 再怎么做也没用（事情不会开始推进）。

  • てもどうにもならない

    • 不管怎样也无法改变现状。
今いま 更さら 後こう 悔かいしても仕し 方かたがない
今いま 更さら 後こう 悔かいしても始はじまらない
今いま 更さら 後こう 悔かいしてもどうにもならない

现在后悔也无济于事。

てはいられない

  • て
  • は 强调
  • いる 存在
  • られる 能
  • ない 不

意义：不能……

接续：て形 + てはいられない

時じ 間かんがないから、もう待まってはいられない。

没时间了，已经不能再等了。

こんな無む 意い 義ぎな問もん 題だいに時じ 間かんをかけてはいられない。

这种无意义的问题不能再花时间在上面了。
