こと

  • こと 命令、感叹
  • ことがある 有时、有过
  • ことができる 能力、可能
  • ことに 令人~
  • ことにする 决定
  • ことになる 结果
  • ことになっている 规定

こと 命令、感叹

大おお 声ごえを出ださない こと

不要大声喧哗。（命令、规定用语）

素す 敵てきな 洋よう 服ふくです こと

真是件漂亮的洋装啊。（带感叹、优雅语气）

ことがある 有时、有过

日に 本ほんに行いくことがある

有时候会去日本；偶尔会去日本。

日に 本ほんに行いった ことがある

去过日本；曾经去过日本。

ことができる 能力、可能

日に 本ほん 語ごを 話はなす ことができる

能说日语；会说日语。（表示能力）

ここでタバコを吸すう ことができる

这里可以抽烟。（表示许可）

ことに 令人~的是

驚おどろいたことに

令人吃惊的是；出乎意料的是。

嬉うれしいことに

令人高兴的是；值得开心的是。

残ざん 念ねんなことに

遗憾的是；可惜的是。

ことにする 决定

カメラを買かわないことにする。

决定不买相机。

ことになる 结果

明あ 日した 出しゅっ 張ちょうする ことになりました。

已经决定明天出差。（不是自己决定的，是外部安排）

ことになっている 规定

日に 本ほんでは 車くるまは左ひだり 側がわを 走はしる ことになっている。

在日本，汽车规定要在左侧行驶。（这是制度／规则）

とおり 按照~

通り

按照~

接续：

  • 动词、名词 + の　+ とおり
  • 名词 + どおり
見みた通とおりに、話はなしてください。

请按照你看到的那样说；请照着看到的情形讲述。

説せつ 明めい 書しょのとおりにやってください。

请按照说明书上的内容来操作。

説せつ 明めい 書しょどおりにやってください。

请照说明书原样操作。

はず

  • はずだ 推测；应该

    • 连体形 + はず

  • はずがない 不可能

    • 连体形 + はずがない
彼かれは家いえに着ついたはずです。

他应该已经到家了。

彼かの 女じょは嘘うそをつくはずがない。

她不可能说谎。（按她的性格／情况来看，绝对不会说谎。）

うえ

  • 上うえ で

    • する 上うえ で 在……方面

    • した 上うえ で 在做了……基础上 → 在……之后

      • 辞じ 書しょは 言こと 葉ばの学がく 習しゅうする上うえで 欠かかせないものだ。

      • 在学习语言时，词典是不可或缺的东西。

      • この問もん 題だいは 慎しん 重ちょうに検けん 討とうした上うえで 決けっ 定ていします。

      • 这个问题会在慎重讨论之后再做决定。

  • 上うえ に 加上 → 不仅 而且

    • うちは 狭せまい 上うえに、 子こ 供ども も 多おおくて、ゆっくり 休やすめません。
      • 我们家不仅地方小，而且孩子也多，根本没法好好休息。

  • 上うえ は 既然 = 以い 上じょう は

    • 日に 本ほん に 留りゅう 学がくした 上うえは、 早はやく 日に 本ほん の 生せい 活かつ に 慣なれてください。
      • 既然已经来日本留学了，请尽快适应日本的生活。

うち

  • うちに

    • 趁着……

      • 接续：连体形 + うちに

      • 若わかいうちに いろいろ経けい 験けいしたほうがいいよ。

      • 趁年轻，多经历一下各种事情比较好。

    • 在……期间；侧重：某个时间段内，不知不觉动作完成了

      • 接续：连体形 + うちに

      • 今いまは上じょう 手ずに話はな}せなくても、 練れん 習しゅう}を重かさ}ねるうちに 上手になります。

      • 现在即使说得不太好，随着不断练习，慢慢就会变得熟练。

  • うちは

    • 在……期间；侧重：某个时间段内，一种状态持续着

      • 接续：连体形 + うちに

      • 学がく 生せいの うちは 結けっ 婚こんできない。

      • 在还是学生的时候，不能结婚。

  • ないうちに

    • 还没有……就已经……

    • 接续：ない形 + ないうちに

      • 半はん 年とし 経たたないうちに、 交こう 通つう 事じ 故こ 死し 者しゃは ５０００人にんを 突とっ 破ぱした。

      • 还不到半年，交通事故死亡人数就突破了 5000 人。

わけ

  • わけだ 说明

  • わけだ 原因

  • わけがない 不可能

  • わけではない 并不是

  • わけにはいかない 不可以

わけだ 说明

表示就是说明 = のです

一ひと 人り 二に 百ひゃく 円えんなら、 5人にんだと千せん 円えんになるわけだ。

如果一个人是200日元的话，5个人就是1000日元嘛。

わけだ 原因

今きょう 日はの 面めん 接せつは 失しっ 敗ぱいした、準じゅん 備び 不ふ 足そくだったわけだ。

今天的面试失败了，原来是因为准备不足。

わけがない 不可能

不可能 = はずがない

こんな 漢かん 字じが 多おおい 本ほんは 子こ 供どもが 読よむわけがない。

这么多汉字的书，孩子不可能读得懂。

わけではない 并不是

おお 金かねがあれば 幸しあわせになれる わけではない。

有钱并不代表就能变得幸福。

わけにはいかない 不可以

今きょう 日はは 大だい 事じな 会かい 議ぎがあるから、 休やすむ わけにはいかない。

今天有重要的会议，所以不可能请假休息。

つもり

  • 打算（N4）

    • 接续：原形 + つもり

    • 日に 本ほんへ行いくつもりです。

    • 打算去日本。

  • 就当做（N3）

    • 接续：た形 + つもり

    • 死しんだつもりで、 一いっ 生しょう 懸けん 命めいやりましょう。

    • 就当是拼上性命，全力以赴干吧。

  • 自认为（N2）

    • 接续：ている + つもり

    • た形 + つもり

    • 名词 + の + つもり

    • 彼かの 女じょは 美び 人じんのつもりだけど、 あまりもてない。
      • 她自以为是美人，不过其实并不怎么受欢迎。
    • 私わたしは 彼かの 女じょに 親しん 切せつにしたつもりだけど、 恨うらまれました。
      • 我自以为对她很亲切，结果却被她怨恨了。
