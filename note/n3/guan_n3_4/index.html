くらい／ぐらい

1、意义：作为名词，表示：数量单位

十じゅうの位くらい
百ひゃくの位くらい

2、作副词用：大约，左右 = ほど

3時じ 間かんくらい

大约三个小时。

20人にんくらい

大约二十个人。

3

  • 表示程度 一致

    • この鼠ねずみは猫ねこと同おなじくらいの大おおきさだ。
      • 这只老鼠和猫差不多大。（＝大约同等程度）

  • 表示程度 之高

    • 死しぬくらい疲つかれた ＝ 死しぬほど疲つかれた
      • 累得要死。这里的 くらい ＝ ほど，表示程度之强烈。

  • 表示程度 之低

    • 挨あい 拶さつくらいはできるでしょう。
      • 打招呼这种程度应该能做到吧。
      • 「くらいは」表示 最低限度、至少这种程度带一点点轻微的责备或期待。
  1. くらい ＝ ほど 都可以表示程度，基本可以互换；
  2. 但是ほど 更多用于描述程度高，くらい更多描述程度低
  3. 而且ほど偏书面语，くらい则偏口语化；

まで

  • 终点
  • 甚至；连……
君きみ まで 僕ぼくを 疑うたがうの？

连你也要怀疑我吗？

てまで

意义：即使……也要

接续：て形 + てまで

借しゃっ 金きんしてまで 新あたらしい 携けい 帯たいを 買かいます。

甚至贷款也要去买新手机。

语感说明：〜てまで

「Vてまで」表达一种 极端的、牺牲性的手段，带有轻微否定评价：

  • 为了达到目的
  • 甚至不惜做出不太正常、不值得的行为

所以本句强调：

买手机重要到连借钱这种事都做了。

だけでは

意义：光是~的话

接续：连体形、名词 + だけでは

見みる だけでは つまらない。

光是看一看的话，很无聊。

だけでかく

意义：不仅

接续：连体形、名词 + だけでかく

彼かれは 日に 本ほん 語ごが 上じょう 手ずなだけでなく、 英えい 語ごもぺらぺらだ。

他不仅日语很好，英语也很流利。

さえ

意义：甚至

接续：名词 + さえ

日に 本ほん 人じんでさえ、 敬けい 語ごを 間ま 違ちがえる 場ば 合あいもある。

即便是日本人，也有可能把敬语用错。

さえ~ば

意义：只要~就

继续：名词 +さえ;ば形 + ば

天てん 気きさえよければ出でかける。

只要天气好，我就会出去。

天気さえよければ」，这里使用形容词「いい／良い」的假定形：

  • 良い → よい（基本形）
  • よい → よければ（假定形）

变形说明：

形容词「いい」的变形特殊：

  • 否定：よくない
  • 假定：よければ
  • 过去：よかった

不是直接按「い → ければ」的规则变，而是历史残留的特殊形态。

こそ

意义：正是……，才是……

接续：名词 + こそ

こちらこそよろしく お願（ねが）いします。
今年ことしこそ 試し 験けんに 合ごう 格かくする。

今年一定要考上。こそ（强调助词）

など

など、などか、なんて

意义：~之类的

对别人用——轻视

对自己用——自谦

不同1：名词 + などか、なんて ；动词 + なんて

暴ぼう 力りょくを 振ふるうなんて 怖こわいよ。

动不动就动粗什么的，真吓人啊。/ 使用暴力这种事，太可怕了。

「なんて」带轻蔑、否定、惊讶情绪。

不同2： などか + 助词，なんて不能加

バリ島とう なんかが、 日に 本ほん 人じん に 人にん 気きがある。

像巴厘岛这样的地方，在日本人当中很受欢迎。

不同3：なんて + 名词；などか不可以加

田た 中なか なんて 人ひとは 知しらない。

我才不认识什么“田中”这样的人。（带有轻蔑、否定价值的语气）

ばかりだ

意义：刚刚

接续：た形 + ばかりだ

  • た ところだ (客观)
  • た ばかりだ（主观）
帰かえったばかりだ ＝ 帰かえったところだ

“刚回来的样子”、“刚刚回到家”。

先せん 週しゅう 買かったばかりだ。

“上周才刚买的（还很新）。”

名词化

さ：客观性质

重おもさ、深ふかさ、甘あまさ

み：主观感受

重おもみ、深ふかみ、甘あまみ
  • 「〜さ」：客观性质 表示可量化、可描述的“程度／大小”，偏客观描述。
    • 重さ：重量、沉重程度
    • 深さ：深度
    • 甘さ：甜度
  • 「〜み」：主观感受 表示说话人主观感受到的“味道／意味／韵味”，偏感情、抽象。
    • 重み：分量、沉甸甸的感觉（也可指“份量感、庄重感”这种抽象的重）
    • 深み：深度、深邃感、韵味
    • 甘み：甜味、甜滋味（侧重“入口时感觉到的甜”）

简短记：

  • さ = 可量化的“性质、程度”
  • み = 主观体验到的“感觉、意味”
痛いたみを 感かん じろ

感受痛苦吧。

ぶり

  • ……的样子

    • 接续：连用形、名词 + ぶり
    • 話はなしぶり
      • 说话的样子、说话的方式、口吻
    • 慌あわてぶり
      • 慌あわてる
      • 慌张的样子、慌慌张张的状态
    • 飲のみぶり
      • 喝酒的样子、喝法、酒品

  • 时隔……

    • 名词 + ぶり

    • 彼かの 女じょに会あうのは10年ねんぶりだった。
      • 和她见面，是相隔十年之后的第一次。

~いい

意义：容易~

接续：连用形 + いい

この街まちは住すみ良よいです。

这个城市很好住、适合居住。

この薬くすりは飲のみ良よいです。

这药很好吃（服用起来不难受）。

この本ほんは分わかりいいです。

这本书很好懂。

说明：いい 只能用于积极的场合；やすい可以是积极，可以是消极

不过现代日语更常用：

  • 「〜やすい」：容易做
  • 「〜にくい」：难做

如：

  • 分かりやすい
  • 住みやすい
  • 飲みにくい

「〜良い」听上去更文学、更书面，有一点古风。

がる

意义：第三人称的感觉

接续：形容词词干 + がる

怖こわい → 怖こわがる

怖がる：表现出害怕、感到害怕（别人表现出来的状态）

嫌いや → 嫌いやがる

嫌がる：表现出讨厌、不情愿

～たい → ～たがる

たがる：别人看起来想做、表现出想

ぎみ

気ぎ 味み

意义：感觉有点……（不好的）

接续：名词、动词连用形 + ぎみ

風邪かぜ 気味ぎみ

有点感冒的样子、好像要感冒了。

疲つかれ気味ぎみ

有点累、略感疲惫。

太ふとり気味ぎみ

有点胖、好像变胖了。

がち

勝がち

意义：有不好的倾向，“动不动就~”

接续：名词、动词连用形 + がち

病びょう 気きがちだ。

容易生病；动不动就生病。

曇くもりがちだ。

经常多云；老是阴天。

忘わすれがちだ。

老是忘；很容易忘记。

向むき、向むけ

  • 向むき：适合

    • 子こ 供どもむきの絵え 本ほん

    • “适合孩子的绘本”“为儿童设计的绘本”

  • 向むけ：面向

    • 子こ 供どもむけの絵え 本ほん。

    • “面向儿童的绘本”“专为儿童制作的绘本”。

最さい 近きんの子こ 供ども（向むき・向むけ）の漫まん 画がには、とても子こ 供ども（向むき・向むけ）とは思おもえないものがある。

最近所谓“给孩子看的”漫画里，有一些无论如何都不能让人觉得是给孩子看的作品。

始はじまる・出だす

意义：开始做某事

接续：动词的连用形 + 始はじまる・出だす

日に 本ほん 語ごを 習ならい始はじめた のは 半はん 年とし 前まえです。

开始学习日语是半年前的事。

桜さくらの花はなが 咲さき始はじめました ね。

樱花开始开了呢。

出だす 具有突发性；【急きゅうに】

雨あめが 急きゅうに 降ふり出だした。

突然下起雨来了。

やすい・よい

よい只用于积极的

住すみやすい ＝ 住すみよい

容易居住的、适合居住的

实际上，やすい 是 よい 的现代口语替代

にくい・がたい・づらい

  • にくい 困难是又对象本身引起的

    • わかりにくい 难懂、不容易理解；不容易分辨 / 看清。

    • この説せつ 明めいはわかりにくい。这个说明很难懂。

  • がたい 困难是由客观环境引起的

    • 得えがたい 难以得到；难能可贵；不容易取得的。

    • 彼かれの協きょう 力りょくは得えがたい。他的合作是难能可贵的。

  • づらい 困难是难以忍受引起的

    • 聞ききづらい 难以问、难以开口问；不好意思问；问起来尴尬/不方便。

    • こんなことは聞ききづらい。 这种事情不好开口问。

  • 言いいにくい 因为语言本身难，说不流利

  • 言いいがたい 因为周围的情景说不出口

  • 言いいづらい 因为喉咙痛等因素无法忍受

する

をする

1、意义：做~

接续：名词 + をする

サッカーをする

踢足球；进行足球运动

教きょう 師しをしている

当老师；在做教师这份工作

怪がい 我がをする

忍耐；忍着不发作；压抑情绪

2、意义：表示外部特征

接续：名词 + がする

大きな目めをしている

有一双大眼睛；长着大眼睛

「〜をしている」在这里不是“正在做”，而是表示外貌特征、身体特征的呈现状态。

翻译成中文比较自然的是“拥有XXX的特征”。

いい色いろをしている

肤色、气色很好；显得有精神

硬かたい表ひょう 情じょうをしている

表情僵硬；一脸严肃

がする

意义：表示感觉

接续：名词 + がする

  • 声こえ
    • 声音（人的声音）多指人类发出的声音、嗓音。
  • 音おと
    • 声音（一般声响）更泛指声音、噪声、物体产生的声响。
  • 匂におい
    • 气味、味道；可指好闻或不好闻的味道，不自动区分褒贬。
  • 味あじ
    • 味道（味觉）食品的味道、口味。
  • 感かんじ
    • 感觉、感受、印象；比较抽象，可指情绪或直觉。
  • 寒さむ 気け
    • 寒意、发冷、打冷颤；有时指发烧前的寒战症状。
  • 吐はき気け
    • 恶心、想吐的感觉；身体不适的典型症状之一。
  • 目眩めまい
    • 头晕、眩晕；身体状态不稳、视觉旋转感。
  • 香かおり
    • 香味、香气；通常是愉悦的味道，偏褒义。

にする

意义：表示选择；决定

接续：名词 + にする

飲のみものは 何なににしますか。

饮料要点什么？ / 想喝点什么？

じゃ、 これにします。

那就要这个吧。 / 那我选这个。

とする

意义：作为

变形：

として　＝としては （中顿）

とした （结句）

としての （接名词）

としたら （假定）

としても（让步）

これは 通つう 訳やく としての 責せき 任にんだ。

这是作为翻译的责任。

かれは 留りゅう 学がく 生せい として 日に 本ほんへ きました。

他作为留学生来到日本。

山やま 田ださんは 詩し 人じん としても 有ゆう 名めいだ。

山田先生即使作为诗人也很有名。

もし 宝たか くじに 当あたった としたら、 家いえを かおう。

如果中了彩票的话，就买房吧。

宝たか くじに 当あたった としても、 家いえが かえない。

即使中了彩票，也买不起房。

此时“作为”的意思弱化了，只保留了たら和ても的意思
