单词

1、题干：先せん 週しゅう　デパートに　かいものに　いきました。

选项：

  1. せんしゅ　
  2. せんしゅう　
  3. ぜんしゅ　
  4. ぜんしゅう

答案：2) せんしゅう（先せん 週しゅう）

中文释义

  • 句子：上周去百货商店买东西了。
  • 选项释义：本题为读音选择。2是“先周/上周”的正确读音，其余为误读。

句子成分

  • 时间：先せん 週しゅう
  • 目的地：デパートに
  • 目的：かいもの（＝购物，常写作「買かい 物もの」）に
  • 谓语：いきました

语法解析

  • 名词＋に（地点）＋いく：表示“去某处”。
  • 连用形/名词＋に＋いく：表示“去做～”（目的）。这里「かいものにいく」＝去购物。
  • 敬体过去：いきました（去过）。

接续规则

  • 名词（地点）＋に＋いく／くる
  • 动词连用形（ます去掉ます）或名词＋に＋いく（目的）
  • 五段动词「いく」→连用形いき＋ました（丁宁过去）

形态变化

  • いく（原形）→ いきます（丁宁现在）→ いきました（丁宁过去）

2、题干：ごはんの　後あとで　さんぽします。

选项：

  1. つぎ（下一次/下一个）　
  2. うしろ（后面/身后）　
  3. まえ（前面/之前）　
  4. あと（之后）

答案：4) あと（读出「後あとで」）

中文释义

  • 句子：饭后散步。
  • 选项释义：见括号。

句子成分

  • 时间：ごはんの 後あとで
  • 谓语：さんぽします

语法解析

  • Nの＋後あとで：表示“在……之后”。
  • 「さんぽする」的丁宁形「さんぽします」表示习惯或将要进行。

接续规则

  • 名词＋の＋後あとで
  • 动词た形＋後あとで（若前项是动作：例如 たべた 後あとで）

形态变化

  • する → します（丁宁现在）

3、题干：もう　いちど　言いって　ください。

选项：

  1. いって　
  2. きって　
  3. まって　
  4. たって

答案：1) いって

中文释义

  • 句子：请再说一遍。
  • 选项释义：
    • いって＝言いって（说）
    • きって＝切って（切）
    • まって＝待って（等）
    • たって＝立って（站）

句子成分

  • 程度/次数：もう（再）＋ いちど（一次）
  • 谓语请求：言いって ください

语法解析

  • 动词て形＋ください：请求/请做……
  • 五段动词「言いう」变て形时う→って（连音促音化）。

接续规则

  • 动词て形＋ください
  • 副词（もう）＋数量词（いちど）作状语

形态变化

  • 言いう（原形）→ 言いって（て形）
    • 五段う类：…う／つ／る→って；…む／ぶ／ぬ→んで；…く→いて（例外：いく→いって）；…ぐ→いで；…す→して

4、题干：ちかくに　山やまが　あります。

选项：

  1. かわ（河/川）　
  2. やま（山）　
  3. いけ（池/池塘）　
  4. うみ（海）

答案：2) やま

中文释义

  • 句子：附近有山。
  • 选项释义：见括号。

句子成分

  • 地点：ちかくに
  • 主语：山やまが
  • 谓语：あります

语法解析

  • 地点＋に＋Nが＋あります：表示“在某处有（无生命事物）”。
  • 有生命用「います」，无生命用「あります」。

接续规则

  • 地点名词＋に＋あります／います
  • 主语名词＋が＋あります／います

形态变化

  • ある（原形）→ あります（丁宁现在）

5、题干：この　ホテルは　へやが　多おおいです。

选项：

  1. すくない（少的）　
  2. おおい（多的）　
  3. せまい（狭的）　
  4. ひろい（宽的）

答案：2) おおい（对应形容词「多おおい」）

中文释义

  • 句子：这家酒店房间多。
  • 选项释义：见括号。

句子成分

  • 话题：このホテルは
  • 主语：へやが
  • 述语：多おおいです

语法解析

  • Nは Nが 多おおい／少すくない：表示该话题在某方面“多/少”。
  • い形容词＋です：礼貌表达断定。

接续规则

  • 名词＋は／が（话题与主语的基本格助词）
  • い形容词（基本形）＋です（丁宁）

形态变化

  • い形容词「多おおい」保持基本形＋です → 多おおいです

6、题干：ともだちと　いっしょに　学がく 校こうに　いきます。

选项：

  1. がこう　
  2. がこお　
  3. がっこう　
  4. がっこお

答案：3) がっこう（学がく 校こう）

中文释义

  • 句子：和朋友一起去学校。
  • 选项释义：读音辨析题；3为正确的促音＋长音写法。

句子成分

  • 共同参与：ともだちと
  • 副词短语：いっしょに（一起）
  • 方向：学がく 校こうに
  • 谓语：いきます

语法解析

  • 名词＋と：与……一起/和……
  • いっしょに：副词“一起地”。
  • 名词（地点）＋に＋いく：去往……
  • 读音要点：促音「っ」＋长音「こう（おう）」→ がっこう。

接续规则

  • 名词＋と＋动词
  • 副词（いっしょに）＋动词
  • 名词（地点）＋に＋いく

形态变化

  • いく（原形）→ いきます（丁宁现在）

7、题干：えんぴつが　六ろく 本ほん　あります。

选项：

  1. ろくぼん　

  2. ろくぼん　

  3. ろっぼん　

  4. ろっぽん

答案**：**4) ろっぽん

中文释义

  • 句子：有六支铅笔。
  • 选项释义：此题考“本(ほん)”量词的音便。正确读音应为ろっぽん（六“本”）。其他都是错误读法。

句子成分

  • 主语：えんぴつが（“铅笔”＋主格が）
  • 数量词：六ろく 本ほん（“六支”——长条状物体的量词）
  • 述语：あります（存在）

语法解析

  • 名词＋が＋あります：表示“（无生命物）存在/有”。
  • 数量＋助数词（六＋本）放在被计数名词后或前都可：えんぴつが六本あります ／ 六本のえんぴつがあります。
  • “本(ほん)”遇 1、6、8、10 等发生促音便：いっぽん／ろっぽん／はっぽん／じゅっぽん。

接续规则

  • 名词（无生命）＋が＋あります
  • 【数量表达】数词＋助数词（N5常见：本、枚、人、歳、回、冊 等）

形态变化（非原形说明）

  • ある（原形）→ あります（丁宁现在形）

8、题干：この　新しん 聞ぶんは　いくらですか。

选项：

  1. しんむん　
  2. しんぶん　
  3. しむん　
  4. しぶん

答案：2) しんぶん

中文释义

  • 句子：这份报纸多少钱？
  • 选项释义：均为“新聞”的读音选择；2为正确读音，其余为错误读法。

句子成分

  • 指示＋名词：この 新しん 聞ぶん は（话题）
  • 述语：いくらですか（多少钱？）

语法解析

  • いくらですか：询价固定句型，“多少钱？”
  • 名词＋は…ですか：就某话题进行礼貌疑问。

接续规则

  • 指示词この＋名词
  • 名词＋は＋名词／数量＋ですか（敬体疑问）

形态变化

  • だ（简体断定）→ です（丁宁断定）；加 か 构成疑问。

9、题干：この　カメラは　安やすいです。

选项：

  1. たかい（贵/高）
  2. やすい（便宜）
  3. おもい（重）
  4. かるい（轻）

答案：2) やすい

中文释义

  • 句子：这台相机很便宜。
  • 选项释义：均为形容词含义，如上。

句子成分

  • 话题：このカメラは
  • 述语：安やすいです（形容词述语）

语法解析

  • い形容词＋です：礼貌断定。“安い”表示价格低廉。
  • 与之相对：高い（贵），おもい／かるい 描述重量，不合题意。

接续规则

  • い形容词（基本形）＋です（敬体陈述）

形态变化

  • い形容词用作述语时一般保持基本形＋です；否定/过去等再作派生（安くない／安かった 等），本句无变化。

10、题干：かさは　外そとに　あります。

选项：

  1. いえ（家）
  2. なか（里面）
  3. そと（外面）
  4. にわ（庭院）

答案：3) そと

中文释义

  • 句子：伞在外面。
  • 选项释义：地点名词的意思如上。

句子成分

  • 话题：かさは
  • 地点：外そとに
  • 述语：あります

语法解析

  • 地点＋に＋あります／います：存在句。无生命体（伞）用あります；有生命体用います。
  • “は”提示话题，“に”标地点。

接续规则

  • 名词（地点）＋に＋あります（无生命）／います（有生命）
  • 名词＋は（话题）放句首强调定位

形态变化

  • ある（原形）→ あります（丁宁现在形）

11、题干：けさ　しゃわーを　あびました。

选项：

  1. シャワー　
  2. シャゥー　
  3. ツャワー　
  4. ツャゥー

答案：1) シャワー

中文释义

  • 句子：今天早上洗了淋浴。
  • 选项：考片假名拼写；只有 シャワー（shower）正确，其余是错误拼写。

句子成分

  • 时间副词：けさ
  • 宾语：シャワーを
  • 谓语：あびました

语法解析

  • N＋を＋浴びる：洗/淋（淋浴、日光等）。常用搭配「シャワーを浴びる」。
  • 丁宁过去形表示既往事实。

接续规则

  • 名词＋を＋动词
  • 时间副词（けさ）放句首作状语

形态变化

  • 浴あびる（原形，上一段/一类：いちだん）→ 浴あびます（丁宁）→ 浴あびました（丁宁过去）

12、题干：コーヒーを　のみました。

选项：

  1. 飯めしみました　
  2. 飲のみました　
  3. 飲のみました　
  4. 飲のみました

答案：2) 飲のみました

中文释义

  • 句子：喝了咖啡。
  • 选项释义：
    • 1）把“喝”的汉字写成了“饭”，错误。

句子成分

  • 宾语：コーヒーを
  • 谓语：飲のみました

语法解析

  • N＋を＋飲む：喝～。
  • 丁宁过去形表示“已经喝了”。

接续规则

  • 名词＋を＋动词

形态变化

  • 五段动词：飲のむ → 连用形 飲のみ ＋ ました → 飲のみました

选项：

  1. 卓　
  2. 卓
  3. 車くるま　

（误写变体） 答案：3) 車くるま

中文释义

  • 选项释义：选择“车”的正确汉字。

句子成分

  • 谓语：かいました

语法解析

  • 名词＋を＋动词

14、题干：この　ぼうしは　1000えん　です。

选项：

  1. 1000内　
  2. 1000用　
  3. 1000冊　
  4. 1000円えん

答案：4) 1000円えん

中文释义

  • 句子：这顶帽子1000日元。
  • 选项释义：只有 “円（日元）” 正确；其余是假字或其他量词（冊=册数）。

句子成分

  • 话题：この ぼうし は
  • 述语：1000円えん です

语法解析

  • 名词（价格）＋です：表判断/说明价格。
  • 指示词この＋名词：这～。

接续规则

  • 指示词この＋名词
  • 名词＋は＋名词＋です（判断句）

形态变化

  • （断定）だ → です（丁宁）

15、题干：しゅくだいが　はんぶん　おわりました。

选项：

  1. 羊分（误）　
  2. 半はん 分ぶん　
  3. 半分（字体变体）　
  4. 半分（字体变体）

答案：2) 半はん 分ぶん

中文释义

  • 句子：作业完成了一半。
  • 选项释义：选“半分”的正确写法；1把“半”写成“羊”。

句子成分

  • 主语：しゅくだいが（宿しゅく 題だい，此处题干用假名）
  • 程度：半はん 分ぶん

      • N を 人 に 渡す：把N交给某人。 动词变化
      • コピーする → コピーして（て形）→ …ください
      • 渡す（五段）→ 渡して → …ください 接续
      • 目的语 N を＋数量（助数词）＋サ变动词
      • Vて＋ください／N を 人 に 渡す。

  • 本 は わたし（＝山やま 口ぐち）が あした 返かえします から、わたし の 机 の 上 に 置いて ください。

    • 书明天由我（山口）归还，所以请放在我的桌子上。
    • 语法
      • Vます＋から：原因→结果（礼貌体也可接「から」）。
      • 場所 の 位置名词（上／下／中…） に 置く：把物放到某处。 动词变化
      • 返す（五段）→ 返します（丁宁现在/将来）
      • 置く（五段）→ 置いて（て形）→ …ください 接续
      • （主语）が＋Vます＋から，命令/请求句。
      • 名词 の 位置名词 に＋V（置く）。

29、题干：森もり さん は コピー を した後あと で、本ほん を どう します か。

森同学复印之后，要把书怎么处理？

选项

  1. クラス で つかいます。
  2. 南 さん に わたします。
  3. 中 川 先生 に かえします。
  4. 山 口 先生 の 机 の 上 に おきます。

答案：4

中文释义：复印后，把书放到山口老师的桌子上。

语法与接续：

  • Vて＋ください（请做…）；N に V（渡す／返す）（给／归还给…）；場所＋に＋置く（放在…）。 形态变化：借りる→借りました；渡す→渡して；返す→返します；置く→置いてください。

① 昨きの 日う の 夜よる は おそく まで 仕し 事ごと を しました。

释义：昨晚一直工作到很晚。 语法：

  • おそく まで：直到很晚（まで＝“直到”）。
  • N を する：把名词动词化（仕事をする）。 接续：副词/名词短语＋まで；名词＋を＋する。 变化：する → しました（丁宁过去）。

② とても つかれました。

释义：非常累。 语法：副词「とても」修饰述语。 接续：副词＋动词/形容词。 变化：つかれる → つかれました（过去）。

③ 仕し 事ごと の 後あと で、 電でん 車しゃ で 帰かえりました。

释义：工作之后，坐电车回家。 语法：

  • N の 後 で：在……之后。
  • 交通手段＋で：用……（电车で）。 接续：名词＋の＋後で；手段＋で＋移动动词。 变化：帰る → 帰りました。

④ 家いえ の ちかく の 駅えき で 電でん 車しゃ を おりました。

释义：在家附近的车站下了电车。 语法：

  • N の ちかく の N：……附近的……
  • 場所＋で＋おりる：在某处下（车/电梯）。 接续：名词＋の＋ちかく＋の＋名词；場所＋で＋おりる。 变化：おりる → おりました。

⑤ 外そと は 雨あめ でした が、 わたし は かさ が ありません でした。

释义：外面在下雨，但我没有伞。 语法：

  • N は … でした：名词的礼貌过去断定。
  • が：转折。
  • N が ありません（でした）：无生命“没有”。 接续：句①＋が＋句②。 变化：だ → でした；ある → ありませんでした。

⑥ とても こまりました。

释义：非常为难。 语法：副词＋动词。 接续：とても＋动词。 变化：こまる → こまりました。

⑦ 駅えき の 人ひと が わたし を 見みて、「あの 箱はこ の 中なか の かさ を 使つかって ください。」 と 言いいました。

释义：站务员看到了我，说：“请用那个箱子里的伞。” 语法：

  • Vて＋ください：请做……（使ってください）。
  • 「…」と 言う：引用别人的话。
  • N の 中：……里面。 接续：动词て形＋ください；引语＋と＋言う。 变化：見る→見て；使う→使って；言う→言いました。

⑧ 箱はこ の 中なか に は かさ が 3 本ほん ありました。

释义：箱子里有三把伞。 语法：

  • 場所＋に＋N が あります：无生命存在。
  • 数量＋助数詞：3本（长条物的量词）。 接续：場所＋に＋N＋が＋あります。 变化：ある → ありました。

⑨ わたし は「えっ、 いいん です か。」 と 聞ききました。

释义：我问：“咦，可以吗？” 语法：

  • ～んですか：解释说明＋确认语气（礼貌提问）。
  • 引用同⑦。 接续：引语＋と＋聞く。 变化：聞く → 聞きました。

⑩ 駅えき の 人ひと は「 あれ は 『みんな の かさ』 です。 お金かね は いりません。 あした、 あの 箱はこ に 返かえして ください。」 と 言いいました。

释义：站务员说：“那是大家的伞，不需要付钱。明天请还回那个箱子里。” 语法：

  • N は いりません：不需要N（要る 的否定）。
  • 場所＋に＋返す／入れる：把物品放回／归还到某处。
  • 仍是 Vて＋ください、と 言う。 接续：名词＋は＋いりません；場所＋に＋返す；引语＋と＋言う。 变化：要る→いりません；返す→返して＋ください；言う→言いました。

⑪ わたし は「わかりました。 ありがとう ございます。」 と 言いって、 かさ を 借かりて 帰かえりました。

释义：我说“明白了，非常感谢”，借了伞就回家了。 语法：

  • Vて、V：先后动作连接（言って、借りて、帰りました）。
  • N を 借りる：借入。 接续：动词て形（并列顺接）＋动词。 变化：言う→言って；借りる→借りて；帰る→帰りました。
