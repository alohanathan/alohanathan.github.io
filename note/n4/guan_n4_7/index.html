副助词しか

意义：常常和否定连用，表示“除了……就没有” → “只有”

接续：名词 + しか

肉にく しか 食たべない。＝ 肉にく だけ 食たべる。

只吃肉。

鈴木すずきさん は 日本語にほんご しか 話はなせません。

铃木先生（小姐）只会说日语。

財布さいふ には 100円えん しか ない。

钱包里只有100日元。

语法说明：「しか～ない」

结构：名词 + しか + 动词否定形

意思：“只有……”、“除了……以外什么都不……”

副助词とか

表示列举、相当于や；但比や 更口语

接续：简体 + とか

林檎りんご とか バナナ とか、果物くだもの なら 何なんでも 好すきだ。

像苹果啊香蕉啊，只要是水果我都喜欢。

運動うんどう するとか、食事しょくじ を 減へらすとか で、痩やせたい。

想通过运动啦、减少饮食啦之类的方法来减肥。

好すきだ とか、嫌きらいだ とか、言いわないで ちゃんと 食たべなさい。

别老说喜欢啊、不喜欢啊，好好吃饭！

副助词まで

意义：

  • 终点
  • 程度（最高），翻译成“就连”

接续：

各种词 + まで

授業じゅぎょう は 9時じ から 5時じ まで です。

课程从9点到5点。

東京とうきょう から、上海シャンハイ まで 3時間じかん かかります。

从东京到上海要花3个小时。

世界せかい が 終おわる まで 待まって いる。

一直等到世界的尽头。／等到世界毁灭为止。

表示程度

そんな こと を すると、子供こども に まで 笑わらわれます よ。

那样的事要是做了，连孩子都会笑话你哦。

お年寄としより まで 一生懸命いっしょうけんめい 働はたらいて いる。

甚至连老人都在拼命工作。

君きみ まで 私わたし を 裏切うらぎる ん です か。

连你也要背叛我吗？

相关句型 までに

表示：“在……之前（的任意一个时间点）”

金曜日きんようび までに 論文ろんぶん を 提出ていしゅつ して ください。

请在星期五之前提交论文。

午後ごご 5時じ までに もう 一度いちど 来きて ください。

请在下午5点之前再来一次。

副助词ばかり

意义：

  • 大概 = ぐらい
  • 限定 = だけ

接续：各种词 + ばかり

私わたし は 毎月まいつき 500元げん ばかり 使つかって いる。（大概たいがい）

我每个月大约花500元。（ばかり 表示“大概”）

妹いもうと は 甘あまい もの ばかり 食たべて います。（限定げんてい）

我妹妹只吃甜的东西。（ばかり 表示“只”或“光……”）

田中たなかさん は 寝ねて ばかり いて、何なに も しない。（限定げんてい）

田中先生光睡觉，什么都不做。（ばかり 表示“总是……”）

た　ばかり　だ

意义：刚刚做……

接续：た形 + ばかり

この 靴くつ は 先週せんしゅう 買かった ばかり です。

这双鞋是上周才买的。

私わたし は 今年ことし の 九月がつ に この クラス に 入はいった ばかり だ。

我是今年九月才刚进这个班的。

ばかり　でなく

意义：不仅仅

接续：各种词 + ばかり

山田やまださん は 英語えいご ばかり で なく、中国語ちゅうごくご も 話はなせる。

山田先生不仅会英语，还会中文。

あの アパート は 狭せまい ばかり で なく、音おと も うるさい。

那间公寓不仅狭窄，而且还很吵。

接续助词：し

表示并列，追加；翻译成“加上，并且”

今日きょう は 頭あたま も 痛いたいし、熱ねつ も あります。

今天头也疼，还有发烧。

ここ は 空気くうき が 綺麗きれい ですし、交通こうつう が 便利べんり ですし、住すむ の に いい ところ です。

这里空气又好，交通也方便，是个适合居住的地方。

接续助词：ので

意义：表示原因

接续：

  • 名词、形容动词 + ので
  • 形容词直接 + ので
  • 动词简体 + ので
今日きょう は 日曜日にちようび なので、動物園どうぶつえん へ 行いきます。

因为今天是星期天，所以去动物园。

私わたし の 家いえ は 駅えき から 遠とおい ので、不便ふべん だ。

我的家离车站远，所以不方便。

ここ は 静しずか なので、よく ここ で 本ほん を 読よんで いる。
因为这里很安静，所以我经常在这里看书。
椅子いす が 壊こわれた ので、新あたらしい の を 買かった。

椅子坏了，所以买了新的。

表示原因的说法

  • ので 多表示客观原因，から 多表示主观原因
  • 总结表示原因的说法：から、ので、んです(回答)、て(轻微)
頭あたま が 痛いたい から、学校がっこう へ 行いきません。

因为头疼，所以不去学校。

頭あたま が 痛いたい ので、学校がっこう へ 行いきません。

因为头疼，所以不去学校。（语气更委婉）

どうして 学校がっこう へ 行いきません？　頭あたま が 痛いたい ん です。

为什么不去学校？因为我头疼。

頭あたま が 痛いたくて 学校がっこう へ 行いきません。

因为头疼，没去学校。

接续助词：のに

意义：

  • 虽然……但是
  • 省略后句；表示遗憾or责难；翻译成“明明……就好了”

接续：

  • 名词、形容动词 + のに
  • 形容词直接 + のに
  • 动词简体 + のに
弟おとうと は 明日あした 試験しけん なのに、遊あそんで ばかり いる。

明天弟弟要考试，可他还一直在玩。

冬ふゆ の 山やま は 危険きけん なのに、先週せんしゅう 登のぼった。

冬天的山很危险，可他上周却去登山了。

暑あつい のに、彼かれ は エアコン を つけないで 勉強べんきょう して いる。

明明很热，他却不开空调学习。

そんな こと を 早はやく 言いえば よかった のに。（②）

要是早点说那种事就好了啊。（带有后悔、惋惜的语气）

表示转折的说法

が只是客观陈述

のに则含有不满，责难等负面情绪

日本語にほんご を 毎日まいにち 練習れんしゅう して いる が、なかなか 上手じょうず に ならない。

每天都练习日语，但始终说不好。

日本語にほんご を 毎日まいにち 練習れんしゅう して いる のに、なかなか 上手じょうず に ならない。

明明每天都练习日语，却还是说不好。
