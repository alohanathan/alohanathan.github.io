☆动词意向形

☆意义

  • 表示提议（两个人） =　ましょう的简体
    • ましょう的简体 就是动词意向形
  • 表示决心（一个人）

☆变形规则

  • 一类：う段 → お段 + う

  • 二类：去る + よう

  • 三类：

    • する → しよう
    • 来る → こよう

五段动词

原形意向形中文释义
言いう言いおう说、讲
書かく書かこう写
話はなす話はなそう说话、谈话
待まつ待まとう等待
死しぬ死しのう死亡
飛とぶ飛とぼう飞、飞行
飲のむ飲のもう喝
送おくる送おくろう送、寄送
泳およぐ泳およごう游泳

一段动词

原形意向形中文释义
見みる見みよう看、观看
食たべる食たべよう吃
起おきる起おきよう起床、起来

サ变动词

原形意向形中文释义
勉強べんきょうする勉強べんきょうしよう学习、用功

カ变动词

原形意向形中文释义
来くる来こよう来
行いく行いこう去
帰かえる帰かえろう回去、回家

☆相关句型 1：と思おもう・と思おもっている

意义：表示决心；“要”

接续：意向形 + と思おもう

明日あした 海うみに行いこうと思おもいます。

我打算明天去海边。

ずっと日本にほんへ留学りゅうがくしようと思おもっている。

我一直打算去日本留学。（长期的）

☆相关句型 2：つもり

意义：打算~ = ようと思おもう

接续：原形、ない形 + つもり

明日あした 海うみに行いくつもりです。

→ 打算明天去海边。（= 行こうと思っている）

明日あした 海うみに行いかないつもりです。

→ 打算明天不去海边。（否定形式）

明日あした 海うみに行いくつもりはない。

→ 没有打算明天去海边。（更强烈的否定）

表达语气程度中文感
〜ようと思う轻微、当前打算临时决定“打算要……”
〜つもりです坚定、自觉计划已有明确意图“我计划要……”

命令形

☆意义

  • 表示命令

☆变形规则

  • 一类：う段 → え

  • 二类：去る + ろ

  • 三类：

    • する → しろ
    • 来る → こい

五段動詞

原形命令形中文释义
言いう言いえ说！讲！
書かく書かけ写！
話はなす話はなせ说话！讲！
待まつ待まて等！
死しぬ死しね去死！（非常强烈，多用于漫画或愤怒时）
飛とぶ飛とべ飞！
飲のむ飲のめ喝！
送おくる送おくれ送！寄！
泳およぐ泳およげ游泳！

一段動詞

原形命令形中文释义
見みる見みろ看！
食たべる食たべろ吃！
起おきる起おきろ起床！起来！

サ変動詞

原形命令形中文释义
勉強べんきょうする勉強べんきょうしろ学习吧！去用功！

カ変動詞

原形命令形中文释义
来くる来こい来！
行いく行いけ去！
帰かえる帰かえれ回去！回家！

☆使用场景

  1. 较为粗鲁，多为男性使用
  2. 遇到危险的时候
  3. 加油打气的时候
手てを挙あげろ！

→ 举起手！（命令语，警察或抢劫场面常见）

お金かねを出だせ！

→ 把钱拿出来！（命令语，非常粗鲁）

行いけ！ 急いそげ！ 頑張がんばれ！

→ 去吧！快点！加油！

☆相关句型2——“禁止形”

意义：否定版命令形

接续：原形 + な

動うごくな！

不许动！别动！

触さわるな！

不许碰！别碰！

撮とるな！

不许拍照！别拍！

☆相关句型3 女性版——なさい

さっさと食たべなさい！

→ 赶快吃！

自分じぶんで部屋へやを片付かたづけなさい！

→ 自己收拾房间！

☆相关句型4 柔和版——命令形+よ

明日あした、うちに来こいよ！＝来きて

明天来我家吧！（＝来て）

この写真しゃしんを見みろよ！＝見みて

看这张照片！（＝見て）

そう思おもうなよ！＝思おもわないで

别那样想！（＝思わないで）

☆ば形

☆意义：

  • 表示假定
  • 一……就……

☆变形规则

  • 一类：う段 → え段 + ば

  • 二类：去る + れば

  • 三类：

    • する → すれば
    • 来る → くれば

☆形容词：い → ければ

高い → 高ければ

安い → 安ければ

いい → よければ

ない → なければ

☆形容动词：であれば

綺麗であれば

☆相关句型1——も~ば~も

表示并列，相当于 も~し~も

接续：ば形

人生じんせいには いい時ときも あれば、悪い時ときも ある。

人生中有好的时候，也有坏的时候。

人生じんせいには いい時ときも あるし、悪い時ときも ある。

人生既有好时光，也有坏时光。

☆相关句型2——~ば~~ほど

意义：越~就越

接续：ば形+ば，原形 + ほど

この本ほんは 読よめば 読よむほど、面白おもしろく なる。

这本书越读越有趣。

日本語にほんごは 勉強べんきょうすれば するほど 難むずかしく なる。

日语越学越难。

☆相关句型3——~さえ~~ば

意义：只要~就

接续：名词 + さえ， ば形 + ば

天気てんきさえ よければ、山やまに 行いきます。

只要天气好，就去爬山。

努力どりょくさえ すれば、夢ゆめが 叶かなえる。

只要努力，梦想就能实现。

薬くすりを 飲のみさえ すれば、病気びょうきが よく なりますよ。

只要吃药，病就会好。

☆と、ば、たら、なら

表示假定

接续：

  • と：小句简体
  • ば：ば形
  • たら：た形
  • なら：小句简体

看限制：

  • 最万能的是たら（如果，之后，结果）
  • 唯一能继续话题，提出建议的是：なら
  • 意义就接近的两个就是と和ば，同时都有：“如果”，“一……就……”；其中と更偏向“一……就……”；ば更偏向“如果”。
  • と、ば、なら不能后续接命令，意志，要求，愿望等表达
ハルビンへ（　）、バスがいい。
  • A 行ったら　
  • B 行くなら　
  • C 行けば　
  • D 行くと

要是去哈尔滨的话，坐巴士比较好。

✅ 正确答案：B 行くなら
