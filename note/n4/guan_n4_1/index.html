比况助动词：ようだ

様よう

☆意义1：表示比喻，翻译成“好像”

⭐️🌟表示比喻的ようだ常常和まるで一起连用

李りさんは 日本語にほんごが 上手じょうずで、まるで 日本人にほんじんの ようだ。

小李的日语很好，简直像日本人一样。

昨日きのうの 出会であいは まるで 夢ゆめの ようで、今でも 信しんじられない。

昨天的相遇就像梦一样，到现在都难以置信。

在中间表示中盾，把ようだ→ようで

君きみの 心こころは 氷こおりの ように 冷つめたい。

你的心像冰一样冷。

☆意义2：表示例示，翻译成“像”

この 授業じゅぎょうの 流ながれは 次つぎの ようです。

这节课的流程如下。

いつか 先生せんせいの ように うまく 日本語にほんごが 話はなせると 思おもう。

我想有一天能像老师那样流利地说日语。

メモをしない ような ことをしては いけない。

不可以做不记笔记这样的事。

这些句子展示了「〜ようだ／〜ように／〜ような」三种不同用法：

  • 「〜ようです」：表示样态、说明。
  • 「〜ように」：表示比喻或目标。
  • 「〜ような」：修饰名词，表示“像……一样的（事情／样子）”。

☆意义3：表示（主观）推测，翻译成“好像”

約束やくそくの 時間じかんになっても 誰だれも 来こなかった。どうも 場所ばしょ 時間じかんを 間違まちがえた ようだ。

到了约定的时间也没人来，好像是把地点或时间弄错了。

なんだか 風邪かぜを 引ひいた ようで、登校とうこうはしない。

好像有点感冒了，所以不去上学。

⭐️表推测的ようだ常常和どうも/　なんだか一起连用

☆意义4：表示目的；翻译成“为了”

子供こどもにも 分わかる ように、簡単かんたんな 言葉ことばで 説明せつめいします。

为了让孩子也能明白，用简单的语言说明。

みんなが 聞きこえる ように、大おおきな 声こえで 話はなしてください。

请用大家都能听到的声音说话。

☆意义5：表示表示内容；相当于“と”

合格ごうかくします ように。

希望能合格。（表达愿望的内容）

田中たなかさんに お酒さけを 飲のまない ように 言いってください。

请告诉田中先生不要喝酒。（表达请求的内容）

世界平和せかいへいわが 続つづく ように お祈いのり 申もうし 上あげます。

祈愿世界和平能够持续。（表达祈祷的内容）

⭐️〜ように，在正式的场合中用よう，例如商务文书，正式演讲等。

这里的「〜ように」用法分三种：

  • 表示愿望（「〜ますように」）；
  • 表示请求或转达（「〜ように言う／頼む」）；
  • 表示祈祷或郑重的祝愿（「〜ようにお祈りします」）。

☆相关句型1：ようにする

意思：表示努力做，“尽量做……”

接续：参考ようだ

体からだを 丈夫じょうぶにしたいので、今日きょうから 毎日まいにち 運動うんどうする ようにします。

想让身体更健康，所以从今天开始每天运动。

環境かんきょうを 守まもるため、割わり箸ばしを 使つかわない ようにしたほうがいい。

为了保护环境，最好不要使用一次性筷子。

☆相关句型2：ようにすている

意义：表示（长期）努力做；“一直尽量做到……”

接续：参考ようだ

日本語にほんごの 授業じゅぎょうでは、できるだけ 日本語にほんごで 話はなす ようにしています。

在日语课上，尽量用日语说话。

☆相关句型3：ようになる

意义：表示变化“变得”

复习：

  • 名词、形容动词 + なる
    • 名词 + に + なる
    • 形容动词 + に + なる
    • 形容词い变く + なる
  • 动词 + ようになる
日本にほんへ 来きてから、 お酒さけを 飲のむ ように なりました。　（习惯的变化）

来日本以后，开始有喝酒的习惯了。

日本にほんへ 来きてから、 お酒さけを 飲のめる ように なりました。　（能力的变化）

来日本以后，变得能喝酒了。（指原本不能，现在能了）

毎日まいにち 練習れんしゅうしますから、やっと 泳およげる ように なりました。　（能力的变化）

因为每天练习，终于能游泳了。

☆相关句型4：ようになっている

意义：说明事物的结构，功能；“装备有……”；“构造是”

何なにか故障こしょうがあった場合ばあいに、その機械きかいは止とまるようになっている。

一旦发生故障时，那台机器就会自动停止。

前まえに立たつと、ドアが自動的じどうてきに開ひらくようになっている。

站到前面，门就会自动打开。

☆相关句型5：ように見みえる

表示看上去好像……但是实际并非如此

接续：参考ようだ

便利べんりな ように 見みえたが、 使つかってみると、 それほどのものじゃない。

看起来很方便，用起来却也不过如此。

夏休なつやすみの間あいだに、 子供こどもたちは 急きゅうに 成長せいちょうした ように 見みえた。

暑假期间，孩子们看起来一下子长大了。

☆相关句型6：ように思おもう

意义：认为；

ように表内容

区别：

  • と思おもう
  • ように思おもう
私わたしはこの本ほんが面白おもしろいと思おもう。

我觉得这本书很有趣。

私わたしはこの本ほんが面白おもしろいように思おもう。（更委婉，更主观）

我好像觉得这本书挺有趣的。

语法说明： 「〜と思おもう」表示直接的主观看法。 「〜ように思おもう」则更委婉、含蓄、柔和，带有“不太确定，但似乎是那样”的语气。

☆比况助动词：みたいだ

接续：

  • 名词だ + みたいだ
  • Aい + みたいだ
  • AN だ　+ みたいだ
  • 动词简体 + みたいだ

形容动词活用：

  • みたいに + 动词
  • みたいで + 中顿
  • みたいな + 名词

☆意义1：表示比喻，翻译成“好像”

李りさんは日本語にほんごが上手じょうずで、まるで日本人にほんじんみたいだ。

小李的日语很好，简直就像日本人一样。

昨日きのうの出会であいはまるで夢ゆめみたいで、いまでも信しんじられない。

昨天的相遇简直像做梦一样，到现在都难以置信。

君きみの心こころは氷こおりみたいに冷つめたい。

你的心像冰一样冷。

☆意义2：表示例示，翻译成“像”

いつか先生せんせいみたいに うまく 日本語にほんごが 話はなせると 思おもう。

我想有一天能像老师那样流利地说日语。

☆意义3：表示推测，翻译成“好像”

この部屋へやは静しずかみたいだね。

这个房间好像挺安静的呢。

今年ことしの夏なつは暑あついみたいですね。

今年的夏天好像挺热的啊。

なんだか風邪かぜを引ひいたみたいで、登校とうこうはしない。

好像有点感冒了，不去上学。

☆比较：ようだ、みたいだ

  • 接续不同：ようだ接续更复杂一点
  • 用法不同：ようだ用法更多，みたいだ没有目的，内容的用法
  • 使用场景不同：ようだ多用于书面语；みたいだ则用于口语

☆传闻、样态助动词：そうだ

意义：表示传闻“听说”

接续：简体句 + そうだ

变形：そうだ 。そうで、

来月らいげつ 学校がっこうの近ちかくに新あたらしいレストランができるそうだ。

听说明个月学校附近要开一家新餐厅。

お手紙てがみ によると、今年ことしも暑あついそうだ。

根据信上说，今年好像也会很热。

天気予報てんきよほう によると、北海道ほっかいどうのほうが寒さむくなったそうだ。

根据天气预报，北海道那边变冷了。

☆意义：表示样态，1“看起来” 2“好像要”

接续：

  • 动词连用形 + そうだ
  • 形容词去い + そうだ
  • 形容动词 + そうだ
    • いい → よさそうだ
    • ない → なさそうだ
そうだ形容词　安やすい形容动词 有ゆう 名めい动词 降ふる
肯定安やすそうだ有ゆう 名めいそうだ降ふりそうだ
否定安やすくなさそうだ有ゆう 名めいではなさそうだ降ふりそうにない
安やすそうではない有ゆう 名めいそうではない降ふりそうもない
降ふりそうにもない
美味おいしそうな料理りょうりですね。美味おいしそうに食たべている。

看起来很好吃的料理呢；看起来吃得很香。

この店みせは綺麗きれいそうで、高たかいでしょう。

这家店看起来挺干净的，应该挺贵吧。

パソコンの修理しゅうりには時間じかんがかかりそうです。

修电脑好像要花不少时间。

⭐️そうに + 动词、 そうな+ 名词、そうで+ 中顿

強つよい風かぜが吹ふいている、木きが倒たおれそうだ。

风刮得很大，树好像要倒了。

もうすぐ雨あめが降ふりそうですね。

看样子马上要下雨了呢。

☆如何区分：看接续法

雨あめが降ふりそうだ。→ 看起来要下雨了。
雨あめが降ふるそうだ。→ 听说明天要下雨。
おいしそうだ。→ 看起来很好吃。
おいしいそうだ→ 听说很好吃。
元気げんきそうだ。→ 看起来很有精神。
元気げんきだそうだ。→ 听说他身体很好。

上面：样态、动词ます形去ます／形容词去い + ～そうだ

下面：传闻、普通形＋そうだ

☆相关句型1：そうになる

意义：“险些，差点”

接续：动词的连用形 + そうになる

電池でんちが切きれそうになった。

电池快没电了。

びっくりして、持もっていたガラスを落おとしそうになった。

吓了一跳，差点把手里的玻璃掉了。

倒たおれそうになった（差点倒下）
死しにそうになった（差点死掉）
泣なきそうになった（差点哭出来）

☆らしい

  • 结尾词らしい

  • 推量助词らしい

☆结尾词らしい

只能接名词

意义：表示典型特征；“很……”

接续：名词 + らしい

日本語にほんごらしい日本語にほんごを勉強べんきょうしたい。

想学习正宗的日语。

これは谷安たにやすらしい話はなし方かたです。

这就是很像谷安先生风格的说话方式。

この絵えは子供こども らしくない 絵えです。

这幅画不像是小孩子画的。（否定）

☆推量助动词らしい

表示客观推测 “好像”

接续：

  • 名词，形容词 + らしい
  • 动词简体 + らしい

复习：主观推测-ようだ

このレストランの料理りょうりはおいしいらしいです。

听说这家餐厅的菜很好吃。

あそこに立たっている人ひとは鈴木すずきさんらしいです。

听说那边站着的人好像是铃木先生。

最近さいきんは風邪かぜが流行はやっているらしい。

听说最近感冒在流行。
