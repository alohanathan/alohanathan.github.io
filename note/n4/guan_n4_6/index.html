可能态

意义：

  • 能力
  • 可能性 = ことができる

☆变形规则

  • 一类：う段 → え段 + る

  • 二类：去る + られる

  • 三类：

    • する → できる
    • 来る → こられる

句型变化：を → が

日に 本ほん 語ごを 話はなす → 日に 本ほん 語ごが 話はなせる

说日语 → 会说日语、能说日语

ミルクを 飲のむ → ミルクが 飲のめる

喝牛奶 → 能喝牛奶、可以喝牛奶。

知识补充1-ら抜ぬき言こと 葉ば

られる的ら在口语中常常省略

食べられる　→　食べれる
起きられる　→　起きれる

知识补充2-見える/見られる

見える：客观上能看见（与意识无关）

見られる：主观上能看见（有意识地去看）

パンダが 見みえる　　

能看到熊猫（自然地看到，视野里出现）

パンダが 見みられる

能看到熊猫（有能力、条件看到，例如“在动物园能看到”）

知识补充3

意义：表示“不可能……”的时候，主语可用（は・には）

僕ぼく（は・には）理り 解かいできない。

我（无法理解 / 理解不了）。

被动态

意义：被动

☆变形规则

  • 一类：う段 → あ段 + れる

  • 二类：去る + られる

  • 三类：

    • する → される
    • 来る → こられる

書くかく書かれるかかれる写 → 被写

句型变化1-二项动词被动句

⚠️する代表主动，される代表被动

  • 主动句： AはBを する

    • 猫ねこ は 魚さかな を 食たべる

  • 被动句：BはAに される

    • 魚さかな は 猫ねこ に 食たべられる

句型变化2-三项动词被动句

  • 主动句：AはBにCを する　

    • 彼かれ は 私わたし に 道みち を 聞きく 他向我问路。

  • 被动句：BはAにCを される

    • 私わたし は 彼かれ に 道みち を 聞きかれる 我被他问路。

句型变化3-所属关系被动句

  • 主动：AはBのCを する

    • 彼かれ は 私わたし の 年とし を 聞きく

  • 被动：BはA にCを される

    • 私わたし は 彼かれ に 年とし を 聞きかれる

句型变化4-非特定关系被动句

  • 主动：AはBを する
  • 被动：Bは される
人ひと は 鐘かね を 鳴ならす

人敲钟

鐘かね が 鳴ならされる

钟被敲响

人ひと は 展覧会てんらんかい を 開ひらく

人举办展览会

展覧会てんらんかい が 開ひらかれる

展览会被举办

句型变化5-受害者被动句

主动句：Aが する

被动句：BはAに される

雨あめ が 降ふる

下雨。

私わたし は 雨あめ に 降ふられる

犬いぬ が 死しぬ

私わたし は 犬いぬ に 死しなれる

妻つま が 怒おこる

私わたし は 妻つま に 怒おこられる

友達ともだち が 来くる

私わたし は 友達ともだち に 来こられる
お祭まつり の 日ひ に は、④ 町まち の あちこち が きれい に 飾かざられ、多おおく の 人ひと たち が 集あつまって きます。 お祭まつり の 中心地ちゅうしんち は 大変たいへん な 人出ひとで で、③人々ひとびと は 周まわり の 人ひと に 押おされたり、足あし を 踏ふまれたり します。 ③子供達こどもたち は 綺麗きれい な 着物きもの や 服ふく を 着きせてもらって、お父とうさん や お母かあさん に 手て を 引ひかれて 見物けんぶつ に 行いきます。 道路どうろ に は 小ちいさな 店みせ が たくさん 並ならんで います。 ④店みせ に は 玩具おもちゃ や お菓子おかし など が いろいろ 売うられて います。 ②お父とうさん や お母かあさん は 子供こども に 玩具おもちゃ を 強請ねだられ、⑤大おおきな 声こえ で 泣なかれて 困こまる こと が あります。

在祭典的那天，镇上的各个地方都被装饰得十分漂亮，聚集了很多人。 祭典的中心地区人山人海，人们被周围的人推来推去，或者脚被踩到。 孩子们穿上漂亮的和服或衣服，由爸爸妈妈牵着手去看热闹。 街道上排列着许多小摊。摊子上卖着各种玩具和点心。 爸爸妈妈被孩子央求买玩具，有时孩子还会大声哭闹，让他们感到为难。

这段文字中大量使用了受身形（被动态），例如：

  • 「飾られ」——被装饰；
  • 「押されたり」「踏まれたり」——被推、被踩；
  • 「着せてもらって」「引かれて」——被帮忙穿上、被牵着；
  • 「売られています」——被售卖；
  • 「強請られ」「泣かれて」——被央求、被哭闹。

使役态

意义：

  • 强制

    • 父ちち は 私わたし に バイオリン を 練習れんしゅう させた。

    • 父亲让（我）练小提琴。

  • 放任

    • 先生せんせい は 子供こども を 遊あそばせた。

    • 老师让孩子们玩。

  • 诱发

    • 梅雨つゆ が カビ を 生はえさせた。

    • 梅雨季让霉菌滋生。

☆变形规则

  • 一类：う段 → あ段 + せる

  • 二类：去る + させる

  • 三类：

    • する → させる
    • 来る → こさせる

句型变化-自动词使役形

  • 一般句：Aが する

    • 私わたし が 行いく

    • 我去。

  • 使役句：BはAを させる

    • 部長ぶちょう は 私わたし を 行いかせる

    • 部长让我去。

句型变化-他动词使役形

  • 一般句：Aは~ を する

    • 私わたし は コピー を する

    • 我复印。

  • 使役句：BはAに~を させる

    • 社長しゃちょう は 私わたし に コピー を させる

    • 社长让我去复印。

⚠️自动词使役对象用 を 提示， 他动词使役对象用 に 提示
