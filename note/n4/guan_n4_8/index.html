敬他语（尊敬語そんけいご） 别人的动作

敬他动词

吃喝来去在、看穿说做给

原动词尊他动词中文释义
食たべる・飲のむ召めし上あがる吃、喝（尊敬）
来くる・行いく・いるいらっしゃる来、去、在（尊敬）
来くる・行いく・いるおいで に なる来、去、在（尊敬）
見みるご覧らん に なる看（尊敬）
着きるお召めし に なる穿（尊敬）
言いうおっしゃる说（尊敬）
原动词尊他动词中文释义
するなさる做（尊敬）
くれるくださる给（尊敬）
来くる見みえる来（尊敬，用于“先生来了”）
知しるご存ぞんじ だ知道（尊敬）

敬他句型

1、被动表尊他

先生せんせい も この 店みせ を よく 利用りよう される ん です か。

老师也经常光顾这家店吗？

部長ぶちょう は 来週らいしゅう 中国ちゅうごく へ 行いかれます。

部长下周要去中国。　（行かれる＝“行く”的尊敬语）

2、お+ます形+になる・なさる＝する

部長ぶちょう は もう お帰かえり に なりました。

部长已经回去了。

旅行りょこう の 日程にってい は もう お決きめ に なりました か。

旅行的日程您已经决定了吗？

先生せんせい が みんな の 前まえ で ご挨拶あいさつ なさいました。

老师在大家面前致辞了。

3、お+ます形+だ＝ている

いらっしゃいません、何なに か お探さがし です か。

欢迎光临，您在找什么吗？

パスポート は お持もち です か。

您带护照了吗？

4、お+ます形+くださる＝てくれる

どうぞ お掛かけ ください。

请坐。／请就座。

ご安心あんしん ください。

请放心。

ご協力きょうりょく くださる よう お願ねがい します。

恳请您给予协助。

敬他语词头

お・ご：訓読くんよみ、音読おんよみ

一组（お）

・お名前なまえ ・お仕事しごと ・お部屋へや ・お気持きもち

二组（ご）

・ご住所じゅうしょ ・ご両親りょうしん ・ご兄弟きょうだい ・ご家族かぞく ・ご注文ちゅうもん

※ 特殊（固定用法）

・お時間じかん ・お電話でんわ ・お茶ちゃ ・お宅たく

自谦语（謙譲語けんじょうご）自己的动作

自谦动词

吃喝来去在，看听说做给

原动词自谦动词中文释义
食たべる・飲のむいただく吃／喝（自谦）
来くる・行いく参まいる去／来（自谦）
いるおる在（自谦）
見みる拝見はいけん する看（自谦）
聞きく・尋たずねるうかがう问；请教（自谦）
言いう申す说（自谦）
原动词尊他动词中文释义
するいたす做（尊敬语的谦卑版，常用于自谦场景）
上あげるさしあげる给（我给您）
もらういただく得到（自谦，礼貌化）
※知しる存ぞんじる认识／知道（尊敬）

自谦句型

１・お+ます形+する・いたす=する

この 番組ばんぐみ は ご覧らん の スポンサー の 提供ていきょう で お送おくり します。

本节目由正在收看的各位赞助商提供播放。

明日あした の 午後ごご、 また ご連絡れんらく します。

明天下午再联系您。

２・お+ます形+申し上げる=てあげる

社長しゃちょう に 代かわって ご挨拶あいさつ 申し上あげます。

我代替社长向您致意。

ご健康けんこう を お祈いのり 申し上あげます。

祝您身体健康。

３・お+ます形+いただく=でもらう

こちら に お名前なまえ を ご記入きにゅう いただけます か。

请您在这里填写姓名可以吗？

本日ほんじつ は ご来店らいてん いただき、 ありがとうございます。

非常感谢您今天光临本店。

４・お+ます形+願う=てほしい

念ねん の ため、 もう 一度いちど お調しらべ 願ねがいます。

为了保险起见，请再调查一次。

お暇ひま が あれば、ご案内あんない 願ねがえません か。

如果您有空的话，可以请您带我参观一下吗？

郑重语（丁寧語ていねいご）不分谁的动作，表示对听话人的礼貌

  • です・ます
  • 郑重措辞

🔵 ① 普通措辞 → 郑重措辞（动词类）

● 有ある → 御座ござる

中文： 有；存在（郑重说法）

● である → で御座ござる

中文： 是（郑重表达“是”）

● 座すわる → かける

中文： 坐下（郑重地请对方入座）

● 頼たのむ → 依頼いらいする

中文： 拜托；请求（更正式的“请求”）

● 分わかる → 了解りょうかいする

中文： 明白；了解（郑重用语）

● 送おくる → 発送はっそうする

中文： 发送（正式场合，如快递、邮件）

● 添そえる → 添付てんぷする

中文： 附上；附件（商务场景常用）

🔵 ② 普通措辞 → 郑重措辞（名词・表达类）

● 今日きょう → ほんじつ

中文： 今天（郑重）

● 昨日きのう → さくじつ

中文： 昨天（郑重）

● 明日あした → みょうにち

中文： 明天（郑重）

● この間あいだ → 最近さいきん

中文： 最近、前阵子（郑重）

● 今度こんど → このたび

中文： 此次（郑重祝辞、公告常用）

● 去きょ年 → さくねん

中文： 去年（郑重）

● 値段ねだん → 価格かかく

中文： 价格、价钱（正式商务用语）

🔵 ① 普通措辞 → 郑重措辞（副词类 1）

● たぶん → おそらく

中文： 大概、可能（更正式）

● だんだん → 徐々じょじょに

中文： 逐渐地，慢慢地（郑重）

● どんどん → 益々ますます

中文： 越来越……；更加（正式场合常用）

● そんなに → それほど

中文： 那么、那样程度（郑重）

● ちょっと → 少々しょうしょう

中文： 稍微、一点点（礼貌表达）

● もっと → より・更さらにに

中文： 更、更加（郑重、正式）

● たくさん → 数多あまたく

中文： 很多、大量（郑重）

🔵 ② 普通措辞 → 郑重措辞（副词类 2）

● 今いま → ただいま

中文： 现在；马上（商务或广播中常用）

● さっき → 先さきほど

中文： 刚才、稍早（郑重）

● すごく → 大変たいへん・非常ひじょうに

中文： 非常、特别（更正式）

● 後あとで → 後のちほど

中文： 之后、待会儿（郑重）

● やっぱり → やはり

中文： 果然、还是（更正式书面）
