补助动词

2025年11月10日 · 1533 字 · 4 分钟 · 日语 初级下

  • 常用的补助动词
  • ⭐️ている
    • ⭐️ている的否定 ていない
    • ⭐️ている的过去 ていた
  • ⭐️てある
  • ⭐️ておく
    • 口语缩略形
  • ⭐️てしまう
    • 音变
  • ⭐️てみる
  • ⭐️てみせる
  • ⭐️ていく、てくる

在句子里失去其本来的意思和独立性，用来补充说明中心动词的意义或者状态

⚠️补助动词一般不写汉字

常用的补助动词

实义动词补助动词
いるている
あるてある
おくておく
しまうてしまう
行くていく
来るてくる
みるてみる
みせるてみせる

⭐️ている

3个用法、2中变化

意义：

  • 正在进行
  • 结果状态
  • 反复习惯

接续：て形 + ている

今いま、 本ほんを 読よんでいる。（正在进行）
彼かれは もう 結婚けっこんしている。（结果状态）
私わたしは 毎日まいにち 電車でんしゃで 学校がっこうに 通かよっている。（反复习惯）

⭐️ている的否定 ていない

用来表示说话时，动作还没实现

まだまだ 食たべていない。

「まだ〜ていない」= 还没有做某事（动作截至现在尚未发生）。

⭐️ている的过去 ていた

用来表示过去某个时间段，做了某事

さっきまで ここで 小説しょうせつを 読よんでいた。

刚才一直在这里看小说。

昨日きのうの 夜よる、映画えいがを 見みていた。

昨晚在看电影。

彼かれは 学生がくせいの 時とき、東京とうきょうに 住すんでいた。

他学生时代住在东京。

⭐️てある

意义：表示人为动作造成的结果状态

接续：他动词 + てある

壁かべに 絵えが 掛かけてある。
ドアが 開あけてある。

辨析：

  • 自动词 + ている = 自然而然的结果状态

  • 他动词 + てある = 人为动作造成的结果状态

窓まどが 開あいている。→ 窗户开着。（状态：窗户是开着的）
窓まどが 開あけてある。→ 窗户被打开着。（人为行为的结果：有人打开后保持开着）

⭐️ておく

置おく：“放着 / 保持”

意义：1.准备 2.放任

接续：て形 + ておく

訪問ほうもんする 前まえに、相手あいてに 電話でんわしておく。

访问之前，先给对方打个电话。（准备）

電気でんきは 消けさないで、朝あさまで つけておきましょう。

电灯不要关，一直亮到早上吧。（放任、保持）

口语缩略形

  • てお く=と く

  • てお いた=と いた

  • てお いて=と いて

  • てお こう=と こう

⭐️てしまう

意义：表示动作结束了or 完了；伴有遗憾，惋惜的语气

接续：て形 + てしまう

昨日きのう、うっかりして 財布さいふを 落おとしてしまった。

昨天一不小心，把钱包掉了。

習ならったばかりなのに、忘わすれてしまった。

明明刚学的，却已经忘了。

音变

  • てしまう = ちゅう
  • でしまう = じゃう

⭐️てみる

見みる

意义：试着做某事

接续：て形 + てみる

やってみます。

我试试看。

やってみてください。

请试试看。

食たべてみてください。

请尝一尝。

⭐️てみせる

見みせる

意义：做给~看

继续：て形 + てみせる

今度こんどこそ 絶対ぜったい 勝かってみせる。

这一次我一定要赢给你看！

⭐️ていく、てくる

实义动词的用法：

V1 → V2 ：先……再去；先……再来

V1 + V2 ：……着去；……着来

补助动词（无实义）：

  • 时间：

    • 从现在到以后
    • 从以前到现在

  • 空间

    • 从远到近
    • 从近到远

  • 存在

    • てくる 从无到有
    • ていく 从有到无
    • 无———有———无

动作的先后

疲つかれたら、ここで 休やすんでいきます。

如果累了，就在这里休息一会儿再走。

あそこで 水みずを 買かってきます。

我去那边买瓶水再回来。

动作的状态附加

学校がっこうまで 走はしっていきます。

我要跑到学校去。

先週せんしゅうの パーティーに 息子むすこを 連つれてきた。

上周我带儿子来参加了聚会。

时间：从现在到以后，从以前到现在

日本にほんの少子化しょうしかは深刻しんこくになっていくでしょう。

日本少子化问题今后会越来越严重吧。

私は一人で子供こどもを育そだててきた。

我一直都是一个人把孩子养大的。

空间：从近到远，从远到近

小鳥ことりが 飛とんでいく。

小鸟飞走了。

小鳥ことりが 飛とんでくる。

小鸟飞过来了。

存在：从有到无，从无到有

あの人ひとの姿すがたは だんだん 消きえていく。

那个人的身影渐渐消失了。

ずっと 本ほんを 読よんでいたので、 目めが 疲つかれてきた。

因为一直在看书，眼睛累了起来。
