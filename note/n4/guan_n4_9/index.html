名词修饰句节-句子边长的关键

小句子+名词（小句子用来修饰名词）

  • 接续
    • 名词 + の + 名词
    • 形容词 + 名词
    • 形容动词 + な + 名词
    • 动词（简体）+ 名词
これは明日あしたパーティーで食たべるケーキです。

中文： 这是明天在派对上要吃的蛋糕。

これは李りさんが食たべないケーキです。

中文： 这是李先生不吃的蛋糕。

これは神様かみさまが食たべたケーキです。

中文： 这是神吃过的蛋糕。

これは果物くだものが多おおいケーキです。

中文： 这是水果很多的蛋糕。

これは李りさんが嫌きらいなケーキです。

中文： 这是李先生讨厌的蛋糕。

これは桜さくら 会社かいしゃのケーキです。

中文： 这是樱花公司的蛋糕。

重点名词：時とき

意义：表示……的时候

接续：名词接续 + とき

子供こどもの時とき、大おおきな地震じしんがありました。

中文： 我小时候发生过大地震。

忙いそがしい時とき、家族かぞくみんなで家事かじをします。

中文： 忙的时候，全家一起做家务。

暇ひまな時とき、街まちで買かい物ものをします。

中文： 有空的时候，会在街上购物。

日本にほんへ行いく時とき、たくさんお土産みやげを買かいました。

中文： 去日本的时候，买了很多土特产。 （动作尚未发生前的时刻 → “行く時”）

日本にほんへ行いった時とき、たくさんお土産みやげを買かいました。

中文： 去了日本的时候，买了很多土特产。 （动作发生后的时刻 → “行った時”）

✔ 日本へ行いく時とき、たくさんお土産みやげを買かいました。

中文： 去日本的时候，买了很多土特产。 语法： 行く時 ＝ 动作 发生前的时点（还没到日本那一刻）。

例：出发前想到“要买纪念品”。

✔ 日本へ行いった時とき、たくさんお土産みやげを買かいました。

中文： 到了日本之后，买了很多土特产。 语法： 行った時 ＝ 动作 完成后的时点（已经到日本了）。

✔ する時 ＝ する前に

中文： “要做…的时刻＝做之前”

例： 出でかける時とき、ドアを閉しめてください。 “要出门的时候，请关门。”

这里的“出かける時”＝出门前一瞬间

✔ した時 ＝ した後で

中文： “做了…的时候＝做之后的时刻”

例： ご飯はんを食たべた時とき、すぐ寝ねました。 “吃完饭后就睡了。”

表达时刻位置中文对应
V（辞书形）＋時动作前……之前
V（た形）＋時动作后……之后

形式体言

形式（走个形式） + 体言（名词）

没有实际意义，只有语法作用的名词。

语法作用：使用形式体言将其他词变成名词（名词化）

有哪些形式体言？

📌 最常用（N5–N3核心）

这些你天天会遇到：

  • の 用于名词化：食べるのが好き
  • ため 原因/目的：病気のため休みます
  • まま 保持某状态：そのままにする
  • ところ 正要…/刚…：今から行くところ
  • こと 抽象名词化：日本語を勉強すること

📌 N3–N2 频繁出现的形式体言

  • もの 表理由/感叹：子供なんだもの
  • わけ 表理由、道理：そういうわけで
  • はず 表应该：来るはずだ
  • つもり 打算：行くつもり
  • とおり（通り） 按照：言うとおりにする
  • うえ（上） 在此基础上：勉強した上で
  • もと（下／基） 在…下：法律のもとに
  • かぎり（限り） 限制：“只要/在…范围内”：知る限り
  • かわり（代わり） 代替：僕の代わりに行く
  • しだい（次第） 一…就…：着き次第連絡します
  • くせ（癖） 抱怨：子供のくせに

📌 格式化：为什么这些叫“形式体言”？

因为它们：

  1. 都是名词（体言）
  2. 意义很弱（不像“猫、桌子”这种有具体含义）
  3. 主要作用是
    • 接续动词、形容词、句子
    • 构成文法表达

例如：

  • 行く＋ため（因为）
  • 日本へ行く＋つもり（打算）
  • 来た＋ところ（刚…）
  • 雨が降った＋わけ（理由）
  • 若い＋くせ（明明…却…）

这些不是实质性的名词，而是“语法载体”。

形式体言の

作用：将其他词或者句子变成名词（名词化）

意义：

  • 表示人，事，物（比较万能）
  • のだ表示解释说明（N5语法）
今笑わらったのは、鈴木すずきさんだろう。

说明： 这里的 の＝人，指“那个笑的人”。 相当于中文的 “那个……的人”。

中文： 刚才笑的那个人大概是铃木先生吧。

日本語にほんごを話はなすのは難むずかしいです。

说明： 这里的 の＝事，把“说日语”整件 动作名词化。 相当于中文 “说日语这件事”。

中文： 说日语这件事很难。

手紙てがみを出だすのを忘わすれました。

说明： 这里的 の＝事，指“寄信这件事”。

中文： 忘记了寄信这件事。

今まで持もっていたのはカバンでしょう。

说明： 这里的 の＝物，代指前文中“你一直拿着的那个东西”。 相当于 “那个东西”。

中文： 你刚才一直拿着的那个应该是包吧。

形式体言ため

為ため

意义

  • 原因
  • 目的

接续：名词接续 + ため(に)

口语：ために

その仕事しごとは危険きけんなために、する人ひとがいない。

〜ために（原因）＝因为……

因为那份工作很危险，所以没有人愿意做。

ずっと働はたらいていたために、2キロも痩やせた。

〜ために（原因）＝因为……导致……（多是负面结果）

因为一直在工作，所以瘦了 2 公斤。

漫画まんがを買かうために、貯金ちょきんしている。

为了买漫画而在存钱。

将来しょうらいのために、頑張がんばって勉強べんきょうしている。

为了将来正在努力学习。

形式体言 まま

意义：以某种本该改变的状态去做……

接续：名词接续 + まま

テレビをつけたまま、出でかけてしまった。

电视开着就出门了。

服ふくを着きたまま寝ねました。

衣服没换就睡着了。

彼かれは行いき先さきを言いわないまま、家いえを出でました。

他没说要去哪儿，就出门了。

形式体言 ところ

意义：

  • 地点（实义名词）
  • 时间点（形式名词）
  • 其他

接续：

  • するところ 正要
  • ているところ 正在
  • したところ 刚做
ご飯はんを食たべるところです。

正要去吃饭。

ご飯はんを食たべているところです。

正在吃饭。

ご飯はんを食たべたところです。

刚吃完。

区别 したところ客观 vs たばかり主观

今いま 帰かえったところだ。 ＝ 今いま 帰かえったばかりだ。

刚刚回来。（两个都自然）

先せん 月げつ 車くるまを買かったばかりだ。

上个月才买的车。（“ばかり”表示主观“觉得很近”）

買かったところだ。

（不自然）“刚买完”从客观角度来说已经过去一段时间，不适合用 たところ。

  • たところ：客观“刚刚发生”。真实时间必须很短，例如几秒～几分钟。

  • たばかり：主观“刚发生”，说话人觉得“还很近”，实际上可能已经过去几天、几周甚至几个月。
