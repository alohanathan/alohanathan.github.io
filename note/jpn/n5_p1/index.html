四大句型（简体和敬体）

名词句

肯定否定
现在、将来N + ですN + ではありません
N + だN + ではない
过去N + でしたN + ではありませんでした
N + だったN + ではなかった

口语里 では 常缩成 じゃ；书面或正式场合优先用 では。

例句

上周并不空闲。

肯定否定
现在、将来私わたしは学がく生せいです。今きょうは休やすみではありません。
私わたしは学がく生せいだ。今きょうは休やすみではない。
过去昨きのうは雨あめでした。先せん週しゅうは暇ひまではありませんでした。
昨きのうは雨あめだった。先せん週しゅうは暇ひまではなかった。

形容词

ナ形容词（形容动词）

肯定否定
现在、将来ナ + ですナ + ではありません
ナ + だナ + ではない
过去ナ + でしたナ + ではありませんでした
ナ + だったナ + ではなかった

例句

这个房间很安静。

他当时身体不好。

肯定否定
现在、将来この部へ屋やは静しずかです。この部へ屋やは静しずかではありません。
この部へ屋やは静しずかだ。この部へ屋やは静しずかではない。
过去彼かれは元げん気きでした。彼かれは元げん気きではありませんでした。
彼かれは元げん気きだった。彼かれは元げん気きではなかった。

イ形容词

イ形容词本质自带断定，不需要（也不能）加「だ」。

肯定否定
现在、将来い + ですくないです / くありません
いくない
过去かったですくなかったです / くありませんでした
かったくなかった

特殊：いい → よい（词形变化基于「よい」）

否定：よくない／礼貌：よくないです／更正式：よくありません。

例句

这本书很有意思。

肯定否定
现在、将来この本ほんはおもしろいです。この本ほんはおもしろくないです / この本ほんはおもしろくありません
この本ほんはおもしろい。この本ほんはおもしろくない
过去この本ほんはおもしろかったです。この本ほんはおもしろくなかったです / この本ほんはおもしろくありませんでした
この本ほんはおもしろかった。この本ほんはおもしろくなかった。

动词

肯定否定
现在、将来ますません
基本形ない形
过去ましたませんでした
た／だなかった
