    • 标准答案：① 父と母ときょうだい

原文

  • わたし には 父ちち と 母はは と 兄あに が います。

    • 中文：我有爸爸、妈妈和哥哥。

  • 兄 は 東とう 京きょう で 一ひと 人り で 暮くららして います。

    • 中文：哥哥在东京一个人生活。

  • 兄 は 犬いぬ を 三さん 匹びき 飼かって います。

    • 中文：哥哥养了三只狗。

  • 母 には 姉あね が 二ふた 人り います。

    • 中文：妈妈有两个姐姐（对我来说是两位姨/姑妈，统称おば）。

  • 父 には 兄きょう 弟だい や 姉し 妹まい は いません。

    • 中文：爸爸没有兄弟姐妹。

设问

  • わたし には だれ が います か。
    • 中文：对“我”来说，我有谁？

选项

  1. 父 と 母 と きょうだい
  2. 父 と 母 と おじ
  3. 父 と 母 と きょうだい と いぬ
  4. 父 と 母 と きょうだい と おば

标准答案：① 父と母ときょうだい

  • 文首明确写了“父・母・兄（=きょうだい）”。
  • ② 有“おじ”（叔叔/舅舅），但父亲没有兄弟姐妹，排除。
  • ③ 加了“いぬ”，而且题目问“だれ”（人），狗再可爱也不算“谁”。
  • ④ “おば”来自“母には姉が二人います”，但题问的是“わたしにはだれがいますか”这道题一般只取文首直述的直系成员，不把亲属的亲属往里堆。

词与语法抓重点

  • 「A には B が います」表示“对A来说，有B”（存在/家人关系）。
  • 「Vて います」= 状态/习惯：暮らしている、飼っている。
  • 量词：「三匹」读さんびき。
  • 「や」= 不完全并列；「兄弟や姉妹」はいません→“兄弟姐妹都没有”。
