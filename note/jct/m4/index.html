原文

  • 先生「今きょ 日う は 教きょう 室しつ の 掃そう 除じ を します。」

    • 中文：老师：今天打扫教室。

  • 学生「いつも じゅぎょう が おわったら しています。」

    • 中文：学生：平时下课后就会打扫。

  • 先生「そう です ね。 でも、きょう は 午ご 後ご の 時じ 間かん を つかって、 いつも は あまり 掃 除 を して いない 所ところ も 掃 除 を します。」

    • 中文：老师：对。不过今天用下午的时间，把平时不常打扫的地方也打扫一下。

  • 学生「どうして です か。」

    • 中文：学生：为什么呢？

  • 先生「明あ 日した から 学がっ 校こう が お休やすみ に なる から です。 つぎ に 学 校 に 来きた とき に 教 室 が きれい だ と うれしい です よ ね。」

    • 中文：老师：因为明天开始放假。下次来学校时教室干净的话，大家都会开心吧。

  • 学生「はい。 教 室 は きれい な ほう が いい です。」

    • 中文：学生：是的，教室干净些更好。

  • 先生「それでは、さいしょ に 窓まど を あけましょう。 そして 机つくえ と 椅い 子す を ぜんぶ 廊ろう 下か に 出だして ください。」

    • 中文：老师：那么，首先把窗户打开。然后把桌子和椅子全部搬到走廊上。

  • 学生「わかりました。」

    • 中文：学生：明白了。

  • 先生「では、先生 は ぞうきん を もって きます ね。」

    • 中文：老师：那我去拿抹布过来。

设问

  • 学がく 生せい は まず 何なに を します か。
    • 中文：学生首先要做什么？

选项

  • 1）じゅぎょう を うけます。
  • 2）窓まど を あけます。
  • 3）机つくえ を 廊ろう 下か に 出だします。
  • 4）ぞうきん を もって きます。

答案：② 窓 を あけます。

依据：老师下达顺序指示「さいしょ に 窓 を あけましょう。 そして 机 と 椅 子 を …」且抹布是老师去拿「先生は ぞうきんを もってきますね」。

重音给常见东京式；固定表达与助词不必死磕。

句式与语法要点

  • 理由说明：「…に なる から です」 例：明 日 から 学 校 が お休み に なる から です。
  • 建议：「V ましょう」 例：さいしょ に 窓 を あけましょう。
  • 请求：「V て ください」 例：机 と 椅 子 を ぜんぶ 廊 下 に 出して ください。
  • 顺序标记：「さいしょ に／まず」「そして」清楚给出先后。
  • 担当分工：「先生 は ぞうきん を もって きます ね」明确是老师去拿抹布，排除⑷。

一句话记忆：看到「さいしょに 窓 を あけましょう」，就知道问“まず何をしますか”时选开窗，别被后面搬桌子和拿抹布带偏。
