日本の りょかんに とまってみよう！

公告全文

标题 日に 本ほん の りょかん に とまって みよう！ 中文：来住住日本的旅馆吧！

开场 りょかん の イベント の お知しらせ です！ 中文：这是旅馆活动的通知！

みんな で たのしく あそびましょう。 中文：大家一起开心地玩吧。

じぶん の 国くに の ホテル と 日 本 の りょかん は どんな ちがい が ある かな？ 中文：你自己的国家的酒店和日本旅馆，有哪些不同呢？

イベント の おしらせ（活动信息） 日ひ にち：5 月がつ 1 日にち（土ど）、2 日（日にち） 中文：日期：5月1日（周六）、2日（周日）

ばしょ：まつ の 木き りょかん 中文：地点：松之木旅馆

さんか できる 人ひと：7 さい いじょう の お子こさん と その かぞく 中文：可参加者：7岁及以上的孩子及其家人

ルール（规则） ・3 日にち 前まえ まで に メール で 予よ 約やく を した 人ひと だけ が さんか できます。 中文：须在活动前3天以内通过邮件完成预约，只有预约过的人才能参加。

・りょかん に 食たべもの や 飲のみもの を もって きて は いけません。 中文：不得把食物或饮料带进旅馆。

・りょかん では おひる と よる に 日 本 りょうり を 食たべる こと が できます。 中文：在旅馆里，中午和晚上可以吃到日本料理。

・子こども の みなさん は お父とうさん と お母かあさん と 来きて ください。 中文：各位小朋友请与爸爸妈妈一同前来。

语法解析

  • りょかん の イベント の お知らせ です
    • 「N1 の N2」名词连结：旅馆的活动的通知。
    • 省略主语「これは」。句末「です」断定。
  • みんな で たのしく あそびましょう
    • 「N＋で」表示一起的参与单位（大家一起）。
    • 「Vましょう」提议；副词「たのしく」修饰方式。
  • …ホテル と …りょかん は どんな ちがい が ある かな？
    • 「A と B は」对比话题。
    • 「どんな＋名」询问性质；「〜かな？」自问自语式疑问。
  • 日 にち：…／ばしょ：…
    • 题头名词作项目标签；无谓语省略。
  • さんか できる 人：7 さい いじょう の お子さん と その かぞく
    • 「V可能形」名词化：能参加的人。
    • 「いじょう」表示以上界；「A と その B」= A及其B。
  • 3 日 前 まで に メール で 予 約 を した 人 だけ が さんか できます
    • 「までに」截止时间；「で」手段（通过邮件）。
    • 体言化从句：「Vた＋人」= 完成预约的人。
    • 「だけ」限定；主格「が」突出限定主语。
    • 「さんか できます」可能表达＝“能参加”。
  • …を もって きて は いけません
    • 「Vて は いけません」禁止表达：不可以……
    • 「もって くる」携带而来。
  • …を 食べる こと が できます
    • 「V辞＋こと が できます」能力许可表达：可以吃到。
    • 「おひる と よる」名词并列。
  • …は お父さん と お母さん と 来て ください
    • 「Vて ください」礼貌请求。
    • 「A と B と」并列同行者；句首话题是「子ども」。

题目所问与判断

问题：イベント の ルール は 何 です か。

选项：

1 食べものを もって いくことが できます。→ 与“もってきては いけません”相反，错。

2 日本の りょうりを みんなで つくります。→ 公告写的是“食べることが できます”，不是做，错。

3 りょかん に 行いく 前まえ に 予 約 を します。→ 符合“3日前までに…予約した人だけ”，对。

4 10 さい の 子ども は 一 人 で とまれます。→ 必须和父母一起来，错。

答案：③。

一句话记住规则

「3日前までにメールで予約。持ち込み禁止。館内で日本料理が食べられる。子どもは保護者同伴。」 翻译成人话：三天前邮件预约；禁止自带吃喝；馆内提供午晚日餐；孩子必须跟着爸妈。 要是这都能选错，真不怪题难，是你把“吃饭”和“做饭”混成了同一个动词。
