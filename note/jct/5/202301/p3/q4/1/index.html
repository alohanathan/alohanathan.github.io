問題4

说明

  • 問もん 題だい4 は 絵え など が ありません。 文ぶん を 聞きいて 1 から 3 の 中なか から 一いち 番ばん いい もの を ひとつ 選えらんで ください。
    • 中文：第4大题没有图片。听句子后，从1到3里选最合适的一项。

問2

刺激句（听到的那句）

誕たん 生じょう 日び おめでとう ございます。 中文：生日快乐。

正确应答（选项与译）

1）どうも ありがとう ございます。（正确） 中文：非常感谢。

2）明あけまして おめでとう ございます。 中文：新年快乐。（季节串台了朋友）

3）お待またせ しました。 中文：让您久等了。（场合错了）

语法与用法摘记

  • 祝贺 → 感谢回礼：おめでとう → ありがとうございます。
  • 2）是新年用语；3）是服务场景道歉。

問3

刺激句

この 映えい 画が は 見みました か。 中文：这部电影你看过吗？

正确应答

2）はい、見みました。（正确） 中文：是的，看过。

1）はい、映えい 画が でした。→ “是电影。”答非所问。 3）いいえ、これ です。→ 语义崩坏。

语法与用法摘记

  • 过去经验确认用「Vました／Vて いません」。
  • 疑问里的主题「は」无问题，答句照动词时态就行。

問4

刺激句

日に 本ほん 語ご は できます か。 中文：会日语吗？

正确应答

3）すこし できます。（正确） 中文：会一点。

1）一ひと 人り で 来きました。→ “一个人来的。”无关。 2）日に 本ほん です。→ “是日本。”更离谱。

语法与用法摘记

  • 能力表达：「N が できる」/「N は できる」。此题用主题提示。
  • 程度副词可换：少すこし／ちょっと／べんきょう 中ちゅう です。

問5

刺激句

ここ で 煙たば 草こ を 吸すわないで ください。 中文：请不要在这里抽烟。

正确应答

2）すみません。（正确） 中文：抱歉。（接受提醒并致歉）

1）いい です よ。→ 你无权给许可，语用错位。 3）こちら です。→ “这边请。”牛头不对马嘴。

语法与用法摘记

  • 禁止请求：V ないで ください。
  • 合适回应是致歉/应允，不是反授权。

逐句语法与句式解析

  • A）誕 生 日 おめでとう ございます。
    • → 祝贺套语；对答用感谢而非再次祝贺。
  • B）この 映 画 は 見ました か。
    • → 过去完成询问；应答「はい、見ました／いいえ、見て いません」。
  • C）日 本 語 は できます か。
    • → 主题提示「は」；能力表达「できます」。恰当回答「すこし できます」。
  • D）ここ で 煙 草 を 吸わないで ください。
    • → 地点格「で」＋禁止请求「V ないで ください」。合适回应是「すみません」。
