听力对话整理（問3）

场景提示

  • 女おんな の 人ひと が 話はなしています。 女 の 人 は レストラン で どのよう に 働はたらきます か。
    • 中文：一位女生在说话。她在餐厅是怎么工作的？

1) 对话逐句

  • 私わたし は 土ど 曜よう 日び に レストラン で アルバイト を しています。

    • 中文：我在星期六在餐厅打工。

  • 17 時じ ごろ お店みせ に 行いって お店 の 中なか を 掃そう 除じ します。

    • 中文：大约17点去店里，把店内打扫干净。

  • それから テーブル の 上うえ を きれい に します。

    • 中文：然后把桌面收拾干净。

  • 18 時 に お店 が 開あいて から は お皿さら を たくさん 洗あらいます。

    • 中文：18点店开门之后，我会洗很多盘子。

  • 本ほん 当とう に 大たい 変へん です。

    • 中文：真的很辛苦。

设问（重复）

  • 女 の 人 は レストラン で どのよう に 働きます か。
    • 中文：这位女生在餐厅是如何工作的？

3) 逐句语法与句式解析

A）私わたし は 土 曜 日 に レストラン で アルバイト を しています。

  • 时间格：「〜に」标具体时间点「土 曜 日」。
  • 场所格：「で」标动作发生地点“在餐厅”。
  • 进行/习惯：「〜て いる」→「しています」表示习惯性打工。

B）17 時 ごろ お店 に 行って お店 の 中 を 掃 除 します。

  • 时间近似：「ごろ」表示大约。
  • 顺接并列：「Vて、V」先去店里，再打扫。
  • 宾语标记：「を」对「中」施加「掃 除」。

C）それから テーブル の 上 を きれい に します。

  • 连接词：「それから」= “然后”。
  • 使役性结果：「形动词词干＋に＋する」→「きれい に する」使之变干净。

D）18 時 に お店 が 開いて から は お皿 を たくさん 洗います。

  • 起点之后：「Vて から」= 在…之后。
  • 主格「が」：无意志主体“店”打开，用自动词「開く」，主语用「が」。
  • 数量副词：「たくさん」修饰动词“洗”。
  • 对比提示：「から は」给“开店以后”设定一个阶段性话题。

E）本 当 に 大 変 です。

  • 评述：副词「本 当 に」加强语气，“真的很辛苦”。
  • 名词述语：「大 変 です」评价性质状态。

4) 动词形态与用法小结

  • 习惯/正在：Vて いる → しています
    • 例：アルバイト を しています（长期/习惯性状态）。
  • 顺序动作：Vて、V／それから
    • 先「行いって」后「掃 除 します」。
  • 起点之后：Vて から（は）
    • 例：お店 が 開いて から は…
  • 自他动对立：開あく（自） vs 開あける（他）。本句用自动词，主语「お店」。
  • 结果使役：ナ形容词＋に＋する → きれい に する。

5) 设问与答案

设问：女おんな の 人ひと は レストラン で どのよう に 働はたらきます か。 答案要点：

  • 开店前：到店后打扫店里，并把桌面整理干净。
  • 开店后：大量洗盘子。 依据：对应句子「…お店 の 中 を 掃 除 します」「テーブル の 上 を きれい に します」「…お皿 を たくさん 洗います」。
