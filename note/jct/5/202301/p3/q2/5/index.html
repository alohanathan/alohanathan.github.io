听力对话整理（問5）

场景提示

  • 学がっ 校こう で 男おとこ の 人ひと と 女おんな の 学がく 生せい が 話はなしています。
    • 中文：在学校，男士和女学生在交谈。

1) 对话逐句

设问

  • 名な 前まえ と 番ばん 号ごう は どこ に 書かきます か。
    • 中文：名字和编号写在哪里？

提示

  • 女おんな の 学がく 生せい が 話はなしています。

    • 中文：由女学生来说明。

  • この 紙かみ に 名 前 と 番 号 を 書いて ください。

    • 中文：请在这张纸上写上名字和编号。

  • はい。 えーっと。

    • 中文：好的，嗯……

  • まず この 写しゃ 真しん の 下した に 名 前 を 書いて ください。

    • 中文：首先，请在这张照片的下面写上名字。

  • はい。

    • 中文：好。

  • そして 名 前 の 右みぎ 側がわ に 出しゅっ 席せき 番ばん 号ごう を 書いて ください。

    • 中文：然后，请在名字的右侧写上出席编号。

  • はい。 わかりました。

    • 中文：好的，明白了。

设问（重复）

  • 名 前 と 番 号 は どこ に 書きます か。
    • 中文：名字和编号写在哪里？

3) 逐句语法与句式解析

A）名な 前まえ と 番ばん 号ごう は どこ に 書かきます か。

  • 话题提示：「〜は」把“名字和编号”作为话题。
  • 地点格：「に」标落笔位置/放置目标，与「書く」等配置。
  • 礼貌疑问：ます形＋「か」。

B）この 紙かみ に … を 書かいて ください。

  • 请求格式：「〜て ください」礼貌请求“请……”。
  • 载体位置：「この 紙 に」说明写在“这张纸上”。
  • 宾语：「を」标被写之物「名 前 と 番 号」。

C）まず この 写しゃ 真しん の 下した に 名 前 を 書いて ください。

  • 顺序副词：「まず」先做第一步。
  • 名词连结：「名＋の＋名」→「写 真 の 下」表示“照片的下方”。
  • 位置格「に」：作为写入位置“在下面”。

D）そして 名 前 の 右みぎ 側がわ に 出しゅっ 席せき 番ばん 号ごう を 書いて ください。

  • 承接：「そして」进入第二步。
  • 参照基准：「名 前 の 右 側」= 以“名字”为基准的右侧位置。
  • 指示对象：「出 席 番 号」具体需要填写的内容。

E）はい。／わかりました。

  • 应答：确认理解与执行意图的固定搭配。

4) 动词与表达用法小结

  • 请求：Vて＋ください
    • 例：書かいて ください（请写）。
  • 顺序提示：まず／そして
    • 明确“先后步骤”。
  • 位置表达：N の 下した／N の 右みぎ 側がわ／N に V
    • “在照片下写名字”“在名字右侧写编号”。

5) 设问与答案

设问：名な 前まえ と 番ばん 号ごう は どこ に 書かきます か。

答案：名な 前まえ は 写しゃ 真しん の 下した、番ばん 号ごう は 名 前 の 右みぎ 側がわ に 書かきます。

依据：原文两步指示「…写 真 の 下 に 名 前 を」「…名 前 の 右 側 に 出 席 番 号 を」，直译即答案。
