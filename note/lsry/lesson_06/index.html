语法

こ、そ、あ、ど系词汇指分别以「こ」「そ」「あ」「ど」开头的四类指示代

词，可用于指代事物、场所或方位。

指代事物指代场所指代方位
近称これこここちら
中称それそこそちら
远称あれあそこあちら
疑问词どれでこどちら

それは何ですか。

「こちら」「そちら」「あちら」「どちら」除用于指代方位外，也可以用于

指代地点、人或事物。

  • Ｎも ＜也＞
  • N と ＜和＞

存在句

  • N 1（场所）に N 2 があります／います。 ＜存在句-Ⅰ＞

1階にコンビニがあります。

  • N 1（场所）には N 2 があります／います。＜存在句-Ⅱ＞

  • N 1 は N 2（场所）にあります／います。＜存在句-Ⅲ＞

    • 由加さんの部屋は、３階にあります。

    • 病院はどこですか。／医院在哪里？
