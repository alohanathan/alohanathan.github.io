《红莲华》TV 版 OP

☆強つよく なれる 理り 由ゆう を 知しった　僕ぼく を 連つれて 進すすめ
中文：找到了变强的理由——带着我的分身/当下的我，一路向前。

句子结构

  • 主干：〈命令〉進め（前进）。
  • 前置说明：強くなれる理由を知った（已知“能变强的理由”）作为说话者的状态。
  • 附加：僕を連れて（带着“我”）修饰“進め”。

语法要点

  • 連用形＋を＋連れて：带着…一起。
  • 动词命令形（五段：進む→進め）：歌词常用的直接、鼓动语气。

核心词汇

单词音调词性释义
強つよい②形容词强；坚韧。
理り 由ゆう⓪名理由；缘由。
知しる⓪他动·五段知道。
僕ぼく①代我（男性常用）。
進すすむ⓪自动·五段前进；推进。
☆泥どろだらけ の 走そう 馬ま 灯とう に 酔よう　こわばる 心こころ
中文：在满是泥污的走马灯回忆里沉醉，心也紧绷着。

句子结构

  • 主干：酔う（沉醉）。
  • 状语：泥だらけの走馬灯に（在沾满泥的“走马灯般回忆”里）。
  • 并列名词：こわばる心（僵硬的心），与前句意象并置。

语法要点

  • Aだらけ：满是A（负面多）。
  • 和风名词并列：无格助词连接形成画面并置。

核心词汇

单词音调词性释义
泥どろだらけ③形动满是泥。
走そう 馬ま 灯とう⓪名走马灯；比喻生涯回闪。
酔よう①自动沉醉；喝醉；晕。
心こころ②名心；内心。
☆震ふるえる 手て は 掴つかみたい もの が ある　それだけさ

中文：颤抖的手，是因为有想抓住的东西——就这么简单。

句子结构

  • 主题：震える手は（至于颤抖的手）。
  • 述部：掴みたいものがある（有想抓住的东西）。
  • 结语：それだけさ（仅此而已，强调）。

语法要点

  • V-たい + もの が ある：确有内在欲求/目标。
  • ～だけだ/さ：排除多余解释，强调本质。

核心词汇

单词音调词性释义
震ふるえる⓪自动颤抖。
掴つかむ②他动抓住；领会。
もの①名东西；事物（抽象/具体）。
☆夜よる の 匂におい に　（I’ll spend all thirty nights）

中文：在夜的气息里。（括注：我要度过这三十个夜晚。）

句子结构

  • 体言止め/省略：夜の匂いに 提示时空/氛围。
  • 括号英文：内心旁白。

语法要点

  • 名词 + に：在…中/受…所染。
  • 歌词常用省略主述营造氛围。

核心词汇

单词音调词性释义

匂におい②名气味；气息。
☆空そら 睨にらんでも　（Staring into the sky）

中文：即便怒瞪着天空。（括注：凝视天空。）

句子结构

  • Vて＋も：即使…也…（让步）。主句省略（承上文语境）。

语法要点

  • 睨む带情绪色彩；这里表不满/抗争。

核心词汇

单词音调词性释义

☆変かわって いける の は 自じ 分ぶん 自じ 身しん だけ　それだけさ

中文：能不断改变的只有自己——就这么回事。

句子结构

  • 焦点结构：変わっていける の は … だけ（能改变的“只有”……）。
  • 结语：それだけさ 强调限定。

语法要点

  • Vていける：今后也能继续…（带进展意味）。
  • だけ：限定；排他。

核心词汇

单词音调词性释义
変かわる⓪自动改变。
自じ 分ぶん⓪代/名自己；自我。
自じ 身しん①名自身；本身。
☆強つよく なれる 理り 由ゆう を 知しった　僕ぼく を 連つれて 進すすめ

中文：同第 1 句。

提示：结构/语法同 ①，不重复。

☆どうしたって！

中文：无论如何！怎么都要上！

句子结构

  • 感叹副词句，承上启下，强化意志。

语法要点

  • どうしたって：口语强调“不管怎样也…”。

核心词汇

单词音调词性释义

☆消けせない 夢ゆめ も　止とまれない 今いま も

中文：无法抹去的梦也好，停不下来的此刻也好——

句子结构

  • 并列主题：Nも…Nも…（两者皆然），下句承接主述。

语法要点

  • V-れない：可能/允许的否定，引申“无法…”。
  • も并列强调。

核心词汇

单词音调词性释义
消けす⓪他动擦掉；熄灭。
夢ゆめ②名梦想；梦。
今いま①名现在；此刻。
☆誰だれか の ため に 強つよく なれる なら

中文：如果是为了某个人而能变强的话——

句子结构

  • 条件：～なら（若是…）。
  • 目的：N の ために（为了…）。

语法要点

  • ために表示目的/缘由；此处是“目的”。
  • 可能/自发义的なれる带成就感。

核心词汇

单词音调词性释义
誰だれか①代某人；别人。
強つよく なる——组块变强。
☆何なん 度ど でも 立たち 上あがれ

中文：就一次又一次站起来吧！

句子结构

  • 命令句：立ち上がれ。
  • 程度/反复：何度でも（无论多少次都…）。

语法要点

  • Xでも（无论X也…）与不定词搭配，强调次数不限。
  • 立ち上がる：从低处起身→引申“重新振作”。

核心词汇

单词音调词性释义

立たち 上あがる⓪自动站起；振作。
☆世せ 界かい に 打うち のめされて 負まける 意い 味み を 知しった

中文：被世界击倒后，我明白了“失败”的意义。

句子结构

  • 状况：世界に打ちのめされて（被…击垮）。
  • 主干：負ける意味を知った（知道了…的意义）。

语法要点

  • 受身 + て（连用）连接原因/经历。
  • N の 意味 を 知る：认识/体悟…的意义。

核心词汇

单词音调词性释义
世せ 界かい①名世界。
打うち のめす④*他动痛击；击垮（*常标为后高型）。
負まける⓪自动失败；输。
意い 味み①名意义；意思。
☆紅ぐ 蓮れん の 華はな よ 咲さき 誇ほこれ！　運うん 命めい を 照てらして

中文：红莲之花啊，尽情盛放吧！去照亮命运！

句子结构

  • 呼格：N よ（“啊”）。
  • 命令：咲き誇れ、照らして（并列/连用祈使）。

语法要点

  • 咲き誇る：盛放、尽情绽放。
  • ～して并列祈使，表达连续动作/并列愿望。

核心词汇

单词音调词性释义

咲さく⓪自动开花。
誇ほこる②自他夸耀；（花）盛放。

☆イナビカリ の 雑ざつ 音おん が 耳みみ を 刺さす　戸と 惑まどう 心こころ

中文释义：闪电般的杂音刺进耳中，内心一阵迷惘。 句子结构：

  • 主干①：雑音が耳を刺す（主语“杂音”+ 他动“刺”）。
  • 并置名词：戸惑う心 与前句并列呈现心境。 语法要点：
  • 名词 + を + 他动词（刺す）：动作作用于宾语“耳”。
  • 并置（无助词连接）营造画面感。 核心词汇：

☆優やさしい だけ じゃ 守まもれない もの が ある？　わかってる けど

    单词标准音调词性释义
    優やさしい—形容词温柔、亲切
    守まもる—他动守护、保卫
    もの—名东西、事物（抽象亦可）
