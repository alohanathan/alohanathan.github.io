☆強つよく なれる 理り 由ゆう を 知しった　僕ぼく を 連つれれて 進すすめ

中文释义

我已经明白“能变强的理由”了——带上我，向前进。

单词表
单词词性原形声调中文释义形态说明
理り 由ゆう名同左0理由、缘由—
僕ぼく名/代同左1我（多为男性自称）—
なる动同左1变成、成为与「強つよい」连用成「強くなる」= 变强
知しる动同左0知道、得知句中为「知しった」（过去）
連つれる动同左0带、领句中为「連つれれて」（て形）
進すすむ动同左0前进句中为命令形「進すすめ」
语法解析

1）句法分裂： A. 「強つよく なれる 理り 由ゆう を 知しった」 B. 「僕ぼく を 連つれれて 進すすめ」

2）A 部分结构

  • 骨架：［连体修饰句］＋ 名词「理り 由ゆう」＋ を ＋ 「知しった」。
  • 「強つよく」= 形容词「強つよい」的连用形，作副词修饰后续动词性短语。
  • 「なれる」= 「なる」的可能形，“能变、能够成为”。构成：五段「なる」→ 语干变为「なれ」＋「る」。
  • 「理り 由ゆう を 知しる」固定搭配：“知道理由”。
  • 「知しった」= 「知しる」过去形（连用形＋た），表示已得知/领悟。

成分标注（A）：

  • 主语（省略，语境=“我”）。
  • 宾语：名词短语「強つよく なれる 理り 由ゆう」。
  • 谓语：「知しった」。

3）B 部分结构

  • 骨架：「X を 連つれれて」＋ 命令「進すすめ」。
  • 「X を 連つれれて」= “带着 X 并…”，这里 X＝「僕ぼく」。
  • 「進すすめ」= 「進すすむ」的命令形（五段 -e 命令），用于自我打气/对听者发令的摇滚式口吻。
    • 注：若是他动词「進すすめる」（推进），其常见命令形是「～ろ／～よ」，本句用的是不及物「進すすむ」的命令形，因此读作「進すすめ」。

成分标注（B）：

  • 受事/携带对象：「僕ぼく を」由「連つれれて」支配。
  • 谓语（祈使/命令）：「進すすめ」。

4）接续与形态小结

  • 形容词→副用：「強つよい」→「強つよく」。
  • 可能形：五段「なる」→「なれる」。
  • 过去形：「知しる」→「知しった」。
  • て形伴随：「連つれる」→「連つれて」。
  • 命令形（五段 -e）： 「進すすむ」→「進すすめ」。

☆泥どろだらけ の 走そう 馬ま 灯とう に 酔よう　こわばる 心こころ

中文释义

沉醉在满是泥泞的走马灯（回忆）里——僵硬的心。

单词表
单词词性原形声调中文释义形态说明
泥どろ名同左1泥、泥泞「〜だらけ」接尾后作连体修饰
走そう 馬ま 灯とう名同左0走马灯；比喻回忆翻涌合成词
酔よう动（五段）同左1沉醉、陶醉；（酒）上头句中为基本形终止
こわばる（強こわ張る）动（五段）同左3僵硬、发紧以连体形修饰名词
心こころ名同左0心、内心被连体动词修饰
语法解析

1）句法分裂 A. 「泥どろだらけ の 走そう 馬ま 灯とう に 酔よう」 B. 「こわばる 心こころ」

2）A 部分结构与接续

  • 「N＋だらけ」：接在名词后作连体修饰，意为“满是……/沾满……”，常带负面语感。这里是「泥どろだらけ」。
  • 「…の 走そう 馬ま 灯とう」：格助词「の」把连体成分连到名词「走馬灯」，整体作主语/处所短语。
  • 「N に 酔よう」：格助词「に」标示沉醉/沉浸的对象或情境；动词为基本形终止，诗歌体直接断句。
  • 语义：沉醉于“满是泥泞的走马灯（般回忆）”。

成分标注（A）

  • 连体修饰： 「泥どろだらけ」→「走そう 馬ま 灯とう」。
  • 处所/对象： 「…に」。
  • 谓语： 「酔よう」（基本形）。

3）B 部分结构与接续

  • 「こわばる」：五段动词的连体形（＝基本形在现代日语可直接连体），修饰后项名词。
  • 「こわばる 心こころ」= “僵硬的心”。
  • 与前句以顿连并列，形成意象叠加（沉醉的回忆 ↔ 僵硬的心）。

4）形态与变化小结

  • 接尾辞：「N＋だらけ」→ 连体修饰语。
  • 名词连接：「X の Y」。
  • 格与支配：「N に 酔よう」。
  • 连体用法：五段动词基本形可直接作连体（「こわばる 心こころ」）。

语法解析
  • 「N で」＝状态/场所，“在水面下”。

  • 「Vて＋見える」＝“看上去/显得…”。
  • 「N に N」：对 N（伪善）施加 N（天罚）。
